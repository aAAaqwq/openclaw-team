## 一、核心概念

回测（Backtesting）是用历史数据验证交易策略表现的过程。选择合适的回测平台是量化交易的第一步。

### 回测系统核心要素

| 要素 | 说明 |
|------|------|
| **数据质量** | 复权方式、停牌处理、退市股票 |
| **撮合引擎** | 限价单/市价单、涨跌停判断 |
| **成本模型** | 佣金、印花税、滑点、冲击成本 |
| **风险模块** | 保证金、杠杆、止损 |
| **评估指标** | 夏普比率、最大回撤、Alpha、Beta |

## 三、各平台详解

### 3.1 聚宽 (JoinQuant)

**优势：**
- A股数据最全：分钟级数据、财务数据、行业分类
- 社区活跃，策略分享多
- 支持实盘对接（部分券商）
- 内置Pipeline框架（源自Quantopian）

**劣势：**
- 在线编辑器限制多
- 免费版有并发和频率限制
- 调试不便

```python
# 聚宽示例：简单的均线策略
def initialize(context):
    """初始化"""
    set_benchmark('000300.XSHG')  # 沪深300为基准
    g.stock = '600519.XSHG'       # 贵州茅台
    g.ma_short = 5
    g.ma_long = 20

def handle_data(context, data):
    """每日执行"""
    stock = g.stock

    # 获取历史价格
    prices = attribute_history(stock, g.ma_long, '1d', ['close'])

    ma_short = prices['close'][-g.ma_short:].mean()
    ma_long = prices['close'].mean()

    # 获取当前持仓
    position = context.portfolio.positions.get(stock)

    if ma_short > ma_long and not position:
        # 金叉买入
        order_value(stock, context.portfolio.available_cash * 0.95)
        log.info(f"买入 {stock}")

    elif ma_short < ma_long and position:
        # 死叉卖出
        order_target(stock, 0)
        log.info(f"卖出 {stock}")
```

### 3.3 Backtrader（推荐本地框架）

**优势：**
- 完全开源免费
- 高度灵活，可自定义一切
- 支持多数据源、多策略
- 丰富的内置指标库
- 支持实盘（需自写Broker）

**劣势：**
- 学习曲线较陡
- 不自带A股数据
- 文档偏少

```python
# pip install backtrader
import backtrader as bt
import akshare as ak
import pandas as pd
from datetime import datetime

class MAStrategy(bt.Strategy):
    """均线交叉策略"""
    params = (
        ('ma_short', 5),
        ('ma_long', 20),
    )

    def __init__(self):
        self.ma_short = bt.indicators.SMA(
            self.data.close, period=self.params.ma_short)
        self.ma_long = bt.indicators.SMA(
            self.data.close, period=self.params.ma_long)
        self.crossover = bt.indicators.CrossOver(
            self.ma_short, self.ma_long)

    def next(self):
        if not self.position:
            if self.crossover > 0:  # 金叉
                self.buy()
        else:
            if self.crossover < 0:  # 死叉
                self.close()

# 获取数据
df = ak.stock_zh_a_hist(
    symbol="600519", period="daily",
    start_date="20200101", end_date="20241231",
    adjust="qfq"
)
df.columns = ['date', 'open', 'close', 'high', 'low', 'volume', 'amount', 'amplitude', 'pct', 'change', 'turnover']
df['date'] = pd.to_datetime(df['date'])
df.set_index('date', inplace=True)
df = df[['open', 'high', 'low', 'close', 'volume']]
df['openinterest'] = 0

# 运行回测
cerebro = bt.Cerebro()
data = bt.feeds.PandasData(dataname=df)
cerebro.adddata(data)
cerebro.addstrategy(MAStrategy)
cerebro.broker.setcash(1000000)
cerebro.broker.setcommission(commission=0.0003)  # 佣金万3
cerebro.addsizer(bt.sizers.PercentSizer, percents=95)

print(f'初始资金: {cerebro.broker.getvalue():,.2f}')
results = cerebro.run()
print(f'最终资金: {cerebro.broker.getvalue():,.2f}')
print(f'收益率: {(cerebro.broker.getvalue()/1000000-1)*100:.2f}%')
```

## 四、回测陷阱与防范

### 4.1 常见回测陷阱

| 陷阱 | 说明 | 防范方法 |
|------|------|----------|
| **未来函数** | 使用了未来数据 | 严格检查每步数据可用性 |
| **幸存者偏差** | 只用当前在市的股票 | 包含已退市股票 |
| **过拟合** | 参数过度调优 | 样本外测试、Walk-forward |
| **忽略成本** | 不计算佣金滑点 | 加入实际交易成本 |
| **忽略冲击** | 假设无限流动性 | 按成交量限制下单量 |
| **复权错误** | 除权除息处理不当 | 统一使用前复权或后复权 |
| **偷价** | 用收盘价买卖但信号在收盘前不知 | 用次日开盘价执行 |

### 4.2 正确的回测流程

```
1. 获取数据（含退市股）→ 前复权
2. 定义信号（严格无未来函数）
3. 次日开盘价/收盘价执行
4. 加入佣金(万2.5) + 印花税(卖千1) + 滑点(0.1-0.2%)
5. 限制单只股票成交量 < 当日成交量的5%
6. Walk-forward验证
7. 样本外测试
```

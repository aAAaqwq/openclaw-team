## 一、核心概念

### 1.1 因子构建流程

```
原始数据 → 数据清洗 → 因子计算 → 去极值 → 标准化 → 缺失值处理 → 因子检验
```

**每一步的核心操作：**

| 步骤 | 方法 | 目的 |
|------|------|------|
| 数据清洗 | 剔除ST/*ST/停牌/上市<60天 | 去除异常样本 |
| 去极值 | MAD法/3σ法/百分位法 | 消除极端值影响 |
| 标准化 | Z-score / Rank | 消除量纲差异 |
| 缺失值处理 | 行业均值填充/剔除 | 保证数据完整性 |
| 中性化 | 行业中性化+市值中性化 | 剔除行业/规模暴露 |

## 二、IC/IR — 因子预测能力

### 2.1 IC (Information Coefficient)

**定义**：因子值与未来收益率之间的相关系数

| IC类型 | 计算方式 | 说明 |
|--------|----------|------|
| **Normal IC** | Pearson相关系数 | 线性关系 |
| **Rank IC (Spearman)** | 秩相关系数 | 更稳健，推荐使用 |
| **Rank IC (Kendall)** | Kendall秩相关 | 适合小样本 |

```python
import pandas as pd
import numpy as np
from scipy import stats

def calc_ic(factor_df, return_df, method='spearman'):
    """
    计算每期IC
    factor_df: DataFrame (index=date, columns=stock_code), 因子值
    return_df: DataFrame (同上), 未来N日收益率
    method: 'spearman' 或 'pearson'
    """
    ic_list = []
    dates = factor_df.index.intersection(return_df.index)

    for date in dates:
        f = factor_df.loc[date].dropna()
        r = return_df.loc[date].dropna()
        common = f.index.intersection(r.index)

        if len(common) < 30:  # 样本太少跳过
            continue

        if method == 'spearman':
            ic, _ = stats.spearmanr(f[common], r[common])
        else:
            ic, _ = stats.pearsonr(f[common], r[common])

        ic_list.append({'date': date, 'IC': ic})

    return pd.DataFrame(ic_list).set_index('date')

def summarize_ic(ic_df):
    """IC汇总统计"""
    return {
        'IC_mean': ic_df['IC'].mean(),
        'IC_std': ic_df['IC'].std(),
        'IR': ic_df['IC'].mean() / ic_df['IC'].std(),  # Information Ratio
        'IC_positive_ratio': (ic_df['IC'] > 0).mean(),
        'IC_tstat': ic_df['IC'].mean() / (ic_df['IC'].std() / len(ic_df)**0.5),
        'IC_abs_mean': ic_df['IC'].abs().mean(),
    }
```

### 2.2 IR (Information Ratio)

**IR = IC_mean / IC_std**

- IR衡量因子预测能力的**稳定性**
- IR > 0.5 → 优秀因子
- IR > 1.0 → 顶级因子（极少）
- 年化IR = 月度IR × √12

## 三、分层回测（Quantile Analysis）

### 3.1 原理

将股票按因子值分为N组（通常5或10组），观察每组的平均收益率。有效因子应表现出：
- **单调性**：各组收益率单调递增/递减
- **多空收益**：最高组 - 最低组显著为正

```python
def quantile_backtest(factor_df, return_df, n_quantiles=5):
    """
    分层回测
    返回每组平均收益率和多空组合
    """
    results = []
    dates = factor_df.index.intersection(return_df.index)

    for date in dates:
        f = factor_df.loc[date].dropna()
        r = return_df.loc[date].dropna()
        common = f.index.intersection(r.index)

        if len(common) < n_quantiles * 10:
            continue

        # 按因子值分组
        ranks = f[common].rank(pct=True)

        for q in range(1, n_quantiles + 1):
            lower = (q - 1) / n_quantiles
            upper = q / n_quantiles
            mask = (ranks > lower) & (ranks <= upper)
            group_return = r[common][mask].mean()

            results.append({
                'date': date,
                'quantile': q,
                'return': group_return
            })

    result_df = pd.DataFrame(results)

    # 汇总
    summary = result_df.groupby('quantile')['return'].agg(['mean', 'std'])
    summary['sharpe'] = summary['mean'] / summary['std'] * (252**0.5)

    # 多空收益
    long_ret = result_df[result_df['quantile'] == n_quantiles].set_index('date')['return']
    short_ret = result_df[result_df['quantile'] == 1].set_index('date')['return']
    ls_ret = long_ret - short_ret

    return summary, ls_ret
```

### 3.2 分层回测判断标准

| 现象 | 说明 | 因子判断 |
|------|------|----------|
| 单调递增 | 高因子值组收益更高 | ✅ 正向因子 |
| 单调递减 | 低因子值组收益更高 | ✅ 反向因子 |
| U型 | 两端高、中间低 | ⚠️ 可能有非线性关系 |
| 无规律 | 各组收益接近 | ❌ 无效因子 |

## 四、因子衰减（Factor Decay）

### 4.1 定义

因子衰减是指因子预测能力随时间窗口变化的特征。

```python
def factor_decay_analysis(factor_df, return_df, max_lag=60):
    """
    分析因子在不同持有期的IC衰减
    """
    decay_results = []

    for lag in [1, 3, 5, 10, 20, 40, 60]:
        # 计算lag日后的收益率
        shifted_return = return_df.shift(-lag)
        ic_series = calc_ic(factor_df, shifted_return)
        mean_ic = ic_series['IC'].mean()
        decay_results.append({'lag': lag, 'IC': mean_ic})

    return pd.DataFrame(decay_results)
```

### 4.2 衰减模式解读

| 衰减模式 | IC曲线 | 含义 | 策略建议 |
|----------|--------|------|----------|
| 快速衰减 | IC在5日内降至0 | 短期信号 | 日内或短线策略 |
| 中等衰减 | IC在20日内降至0 | 中期信号 | 月度调仓 |
| 缓慢衰减 | IC在60日仍>0 | 长期信号 | 季度调仓 |
| 先升后降 | IC先升后降 | 滞后效应 | 延迟建仓 |
| 负向衰减 | IC转负 | 反转效应 | 反向操作 |

**A股常见因子衰减特征：**
- 动量因子：快速衰减（3-5日反转）
- 价值因子：缓慢衰减（60日仍有IC）
- 质量因子：中等衰减（20日IC显著）
- 规模因子：缓慢衰减

## 五、因子中性化（Factor Neutralization）

### 5.1 为什么需要中性化

原始因子可能暴露于行业或规模因子，导致：
- 因子收益实际来自行业Beta而非Alpha
- 回测看起来好但实盘不好用

### 5.2 行业中性化

```python
import statsmodels.api as sm

def industry_neutralize(factor_df, industry_df):
    """
    行业中性化
    factor_df: 因子值 (Series, index=stock_code)
    industry_df: 行业哑变量 (DataFrame, index=stock_code, columns=行业)
    """
    common = factor_df.dropna().index.intersection(industry_df.index)
    y = factor_df.loc[common]
    X = industry_df.loc[common]

    # OLS回归取残差
    model = sm.OLS(y, X).fit()
    neutralized = pd.Series(model.resid, index=common)

    return neutralized
```

### 5.3 市值中性化

```python
def size_neutralize(factor_df, size_df):
    """
    市值中性化
    factor_df: 因子值
    size_df: 对数市值
    """
    common = factor_df.dropna().index.intersection(size_df.dropna().index)
    y = factor_df.loc[common]
    X = sm.add_constant(np.log(size_df.loc[common]))

    model = sm.OLS(y, X).fit()
    neutralized = pd.Series(model.resid, index=common)

    return neutralized
```

### 5.4 完整中性化流程

```python
def full_neutralize(factor_df, industry_df, size_df):
    """行业 + 市值双重中性化"""
    common = (factor_df.dropna().index
              .intersection(industry_df.index)
              .intersection(size_df.dropna().index))

    y = factor_df.loc[common]
    log_size = np.log(size_df.loc[common])
    X = pd.concat([industry_df.loc[common], log_size.rename('log_size')], axis=1)

    model = sm.OLS(y, X).fit()
    neutralized = pd.Series(model.resid, index=common)

    return neutralized
```

## 六、横截面回归（Cross-Sectional Regression）

### 6.1 Fama-MacBeth 回归

经典的横截面回归方法，用于检验因子定价能力：

```python
def fama_macbeth_regression(return_df, factor_df_list):
    """
    Fama-MacBeth两步回归
    return_df: 收益率 (index=date, columns=stock)
    factor_df_list: 因子DataFrame列表
    """
    # Step 1: 每期做横截面回归
    coef_list = []
    dates = return_df.index

    for date in dates:
        y = return_df.loc[date].dropna()
        X_data = {f'factor_{i}': f.loc[date] for i, f in enumerate(factor_df_list)}
        X = pd.DataFrame(X_data).dropna()

        common = y.index.intersection(X.index)
        if len(common) < 30:
            continue

        model = sm.OLS(y.loc[common], sm.add_constant(X.loc[common])).fit()
        coef_list.append({'date': date, **model.params.to_dict()})

    # Step 2: 对系数做时间序列平均
    coef_df = pd.DataFrame(coef_list).set_index('date')

    results = {}
    for col in coef_df.columns:
        mean = coef_df[col].mean()
        stderr = coef_df[col].std() / len(coef_df)**0.5
        tstat = mean / stderr
        results[col] = {'mean': mean, 'tstat': tstat, 'significant': abs(tstat) > 2}

    return results
```

## 八、实战建议

1. **先Rank IC后Normal IC**：Rank IC对异常值更稳健
2. **必须做中性化**：不做中性化的因子检验结果不可信
3. **关注衰减曲线**：决定调仓频率
4. **多维度验证**：IC + 分层回测 + Fama-MacBeth，三个都通过才算有效
5. **注意 survivorship bias**：用包含退市股票的数据

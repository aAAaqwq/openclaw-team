## 一、核心概念

### 1.1 什么是量化交易

量化交易（Quantitative Trading）是指利用数学模型、统计方法和计算机算法来指导投资决策的交易方式。核心要素：

- **数据驱动**：基于历史数据和实时数据做决策，而非主观判断
- **模型化**：将投资逻辑形式化为可计算的数学模型
- **自动化**：通过程序自动执行交易，减少人为情绪干扰
- **系统性**：从信号生成到风控到执行，形成完整系统

### 1.2 A股市场结构

| 层级 | 组成 | 说明 |
|------|------|------|
| **主板** | 上交所主板 + 深交所主板 | 大盘蓝筹，流动性最好 |
| **科创板** | 上交所科创板 (688xxx) | 注册制，涨跌幅20%，科技企业 |
| **创业板** | 深交所创业板 (300xxx) | 成长型企业，涨跌幅20% |
| **北交所** | 北交所 (8xxxxx) | 精选层升级，中小企业 |
| **新三板** | 基础层 + 创新层 | 非上市公众公司 |

**交易制度核心参数：**

| 参数 | A股 |
|------|-----|
| 交易时间 | 9:30-11:30, 13:00-15:00 |
| T+N | T+1（当日买入次日可卖） |
| 涨跌幅限制 | 主板±10%，科创/创业±20% |
| 最小交易单位 | 100股（1手） |
| 交易费用 | 佣金(万2.5左右)+印花税(卖出千1)+过户费 |
| 集合竞价 | 9:15-9:25（开盘）, 14:57-15:00（收盘） |

**量化基金代表（公募）：**
- 华夏量化精选、景顺长城量化精选、嘉实量化阿尔法
- 管理规模：单只通常 5-50 亿

**量化私募代表（头部）：**
- 幻方量化、九坤投资、明汯投资、灵均投资、锐天投资
- 管理规模：头部 100-600 亿+
- 策略：高频CTA、统计套利、指数增强、市场中性

### 1.4 发展历程

| 阶段 | 时间 | 特征 |
|------|------|------|
| 萌芽期 | 2010前 | ETF套利为主，少数机构参与 |
| 成长期 | 2010-2015 | 股指期货推出(2010)，量化对冲兴起 |
| 爆发期 | 2015-2017 | 程序化交易普及，量化私募井喷 |
| 监管期 | 2017-2019 | 股指期货限制，监管趋严，行业洗牌 |
| 成熟期 | 2019-2023 | 注册制改革，科创板推出，量化占比持续提升 |
| 深化期 | 2023-至今 | AI+量化融合，北交所扩容，监管精细化 |

**关键里程碑：**
- 2010年4月：沪深300股指期货上市（IF）→ 量化对冲工具诞生
- 2015年：股灾后股指期货限仓 → 量化策略大调整
- 2019年：科创板开市 → 新的量化标的
- 2021年：量化私募规模突破万亿
- 2023年：DMA、场外衍生品监管收紧

### 1.5 监管框架

**核心监管机构：**
- **证监会**：顶层规则制定
- **沪深交易所**：交易行为监控、异常交易管理
- **中基协**：私募基金备案与管理

**量化相关监管要点：**

| 监管领域 | 核心要求 | 影响 |
|----------|----------|------|
| 程序化交易报备 | 向券商报备程序化交易策略 | 增加合规成本 |
| 异常交易监控 | 禁止频繁撤单、幌骗 | 限制高频策略 |
| 股指期货限仓 | 非套保持仓限制 | 限制对冲规模 |
| DMA监管 | 场外衍生品规模控制 | 影响量化杠杆 |
| 报告制度 | 大额持仓/交易报告 | 增加透明度 |

### Python快速获取A股行情数据

```python
# 安装依赖
# pip install akshare -U

import akshare as ak
import pandas as pd

# 获取沪深300成分股
hs300 = ak.index_stock_cons_csindex(symbol="000300")
print(f"沪深300成分股数量: {len(hs300)}")
print(hs300.head())

# 获取个股行情（以贵州茅台为例）
df = ak.stock_zh_a_hist(
    symbol="600519",
    period="daily",
    start_date="20240101",
    end_date="20241231",
    adjust="qfq"  # 前复权
)
print(f"数据条数: {len(df)}")
print(df.tail())

# 计算简单收益率
df['return'] = df['收盘'].pct_change()
df['ma20'] = df['收盘'].rolling(20).mean()
df['volatility'] = df['return'].rolling(20).std() * (252**0.5)  # 年化波动率

print(f"年化收益率: {df['return'].mean() * 252:.2%}")
print(f"年化波动率: {df['volatility'].iloc[-1]:.2%}")
```

## 五、实战建议

### 入门路线
1. **第1-2周**：学习A股市场基础知识（交易规则、市场结构）
2. **第3-4周**：安装Python环境，用AKShare获取数据
3. **第5-6周**：学习Backtrader框架，编写第一个回测策略
4. **第7-8周**：在聚宽平台上练习策略开发
5. **第9-12周**：学习因子分析（R2内容），构建简单多因子策略

### 避坑指南
- ❌ **不要用未来函数**：回测中最常见的错误，用未来数据做决策
- ❌ **不要忽视交易成本**：A股T+1 + 印花税，频繁交易成本极高
- ❌ **不要过度拟合**：回测收益好 ≠ 实盘能赚钱
- ✅ **先用模拟盘**：至少模拟3个月再考虑实盘
- ✅ **小资金开始**：用可承受亏损的资金试水

### 关键认知
> A股量化最大的挑战不是技术，而是制度限制（T+1、涨跌停、做空限制）。很多海外经典策略在A股需要大幅改造才能适用。

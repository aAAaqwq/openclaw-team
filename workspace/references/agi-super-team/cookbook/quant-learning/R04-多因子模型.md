## 一、核心概念

多因子模型（Multi-Factor Model）是量化投资的基石，通过多个因子的组合来解释和预测资产收益率。

### 通用表达式

$$r_i = \alpha + \sum_{k=1}^{K} \beta_{ik} \cdot f_k + \epsilon_i$$

其中：
- $r_i$：资产i的收益率
- $\alpha$：超额收益（Alpha）
- $\beta_{ik}$：资产i对因子k的暴露度
- $f_k$：因子k的收益率
- $\epsilon_i$：特质风险

## 二、Fama-French 经典模型

### 2.1 三因子模型（1993）

$$r_i - r_f = \alpha + \beta_{MKT}(r_m - r_f) + \beta_{SMB} \cdot SMB + \beta_{HML} \cdot HML + \epsilon$$

| 因子 | 含义 | 构建方法 |
|------|------|----------|
| **MKT** | 市场超额收益 | 全市场收益率 - 无风险利率 |
| **SMB** | 规模因子 (Small Minus Big) | 小盘股组合收益 - 大盘股组合收益 |
| **HML** | 价值因子 (High Minus Low) | 高B/M组合 - 低B/M组合 |

### 2.2 五因子模型（2015）

在三因子基础上增加：

| 因子 | 含义 | 构建方法 |
|------|------|----------|
| **RMW** | 盈利因子 (Robust Minus Weak) | 高盈利组合 - 低盈利组合 |
| **CMA** | 投资因子 (Conservative Minus Aggressive) | 低投资组合 - 高投资组合 |

## 三、BARRA 风险模型

### 3.1 模型结构

BARRA模型是业界最广泛使用的风险模型，MSCI提供多市场版本。

**BARRA中国模型（CNE5）因子结构：**

```
BARRA CNE5 风险因子体系:

├── 风格因子 (Style Factors)
│   ├── Beta (市场Beta)
│   ├── Book-to-Price (价值)
│   ├── Earnings Yield (盈利收益率)
│   ├── Growth (成长性)
│   ├── Leverage (杠杆)
│   ├── Liquidity (流动性)
│   ├── Momentum (动量)
│   ├── Non-linear Size (非线性规模)
│   ├── Residual Volatility (残差波动率)
│   └── Size (规模)
│
├── 行业因子 (Industry Factors)
│   ├── 申万一级行业 (31个)
│   └── 基于GICS分类
│
└── 特质风险 (Specific Risk)
    └── 因子无法解释的个股风险
```

### 3.2 风险矩阵分解

$$\Omega = X \cdot F \cdot X' + \Delta$$

其中：
- $\Omega$：N×N 协方差矩阵
- $X$：N×K 因子暴露矩阵
- $F$：K×K 因子收益协方差矩阵
- $\Delta$：N×N 特质风险对角矩阵

### 3.3 Python实现：简化BARRA模型

```python
import numpy as np
import pandas as pd
from sklearn.linear_model import LinearRegression

class SimplifiedBARRA:
    """简化版BARRA风险模型"""

    def __init__(self, style_factors, industry_dummies):
        """
        style_factors: DataFrame (stock × factors) 风格因子暴露
        industry_dummies: DataFrame (stock × industries) 行业哑变量
        """
        self.style = style_factors
        self.industry = industry_dummies
        self.X = pd.concat([self.style, self.industry], axis=1)

    def fit(self, returns):
        """
        拟合模型，计算因子收益率
        returns: DataFrame (date × stock) 日收益率
        """
        # 每期横截面回归
        self.factor_returns = []
        for date in returns.index:
            r = returns.loc[date].dropna()
            common = r.index.intersection(self.X.index)
            if len(common) < self.X.shape[1] + 10:
                continue

            model = LinearRegression().fit(
                self.X.loc[common], r[common]
            )
            self.factor_returns.append({
                'date': date,
                **dict(zip(self.X.columns, model.coef_))
            })

        self.factor_returns = pd.DataFrame(self.factor_returns).set_index('date')

        # 因子收益协方差矩阵
        self.F = self.factor_returns.cov()

        # 特质风险（残差波动率）
        self.specific_risk = {}
        for date in returns.index[-60:]:  # 最近60日
            r = returns.loc[date].dropna()
            common = r.index.intersection(self.X.index)
            pred = LinearRegression().fit(
                self.X.loc[common], r[common]
            ).predict(self.X.loc[common])
            resid = r[common] - pred
            for stock in resid.index:
                if stock not in self.specific_risk:
                    self.specific_risk[stock] = []
                self.specific_risk[stock].append(resid[stock])

        # 特质方差
        self.delta = pd.Series({
            k: np.var(v) for k, v in self.specific_risk.items()
        })

        return self

    def portfolio_risk(self, weights):
        """
        计算组合风险
        weights: Series (stock → weight)
        """
        w = weights.reindex(self.X.index).fillna(0)

        # 因子风险贡献
        factor_exposure = self.X.T @ w  # K×1
        factor_var = factor_exposure @ self.F.values @ factor_exposure

        # 特质风险贡献
        specific_var = (w**2 * self.delta.reindex(w.index).fillna(0)).sum()

        total_var = factor_var + specific_var
        return np.sqrt(total_var * 252)  # 年化波动率
```

## 四、自定义因子模型

### 4.1 因子合成方法

| 方法 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| **等权** | 各因子权重相同 | 简单 | 忽略因子有效性差异 |
| **IC加权** | 按历史IC加权 | 数据驱动 | IC不稳定 |
| **IR加权** | 按IR加权 | 稳定 | 需要足够历史数据 |
| **最大化IR** | 均值-方差优化 | 理论最优 | 对参数敏感 |
| **机器学习** | XGBoost/LSTM | 非线性 | 过拟合风险 |

### 4.2 IC加权因子合成

```python
def ic_weighted_combination(factor_dict, return_df, lookback=252):
    """
    IC加权因子合成
    factor_dict: {factor_name: factor_df}
    return_df: 收益率DataFrame
    lookback: 回看期
    """
    combined = pd.DataFrame(index=factor_dict[list(factor_dict.keys())[0]].index)

    # 计算每个因子的滚动IC
    ic_weights = {}
    for name, factor_df in factor_dict.items():
        rolling_ic = []
        for i in range(lookback, len(factor_df)):
            date = factor_df.index[i]
            f = factor_df.iloc[i].dropna()
            r = return_df.iloc[i].dropna()
            common = f.index.intersection(r.index)
            if len(common) > 30:
                from scipy.stats import spearmanr
                ic, _ = spearmanr(f[common], r[common])
                rolling_ic.append(ic)
            else:
                rolling_ic.append(0)

        # IC均值作为权重
        ic_weights[name] = np.mean(rolling_ic[-lookback:])

    # 归一化权重
    total = sum(abs(v) for v in ic_weights.values())
    ic_weights = {k: v/total for k, v in ic_weights.items()}

    # 加权合成
    combined_score = pd.DataFrame(0, index=factor_df.index,
                                   columns=factor_df.columns)
    for name, factor_df in factor_dict.items():
        combined_score += ic_weights[name] * factor_df.rank(pct=True)

    return combined_score, ic_weights
```

### 4.3 因子择时

```python
def factor_timing(combined_score, factor_ic_history, method='momentum'):
    """
    因子择时：动态调整因子权重
    method: 'momentum'(动量), 'reversal'(反转), 'vol_target'(波动率目标)
    """
    if method == 'momentum':
        # IC动量：近期IC好的因子给更高权重
        recent_ic = factor_ic_history.tail(60).mean()
        weights = recent_ic / recent_ic.abs().sum()

    elif method == 'reversal':
        # IC反转：近期IC差的因子可能反弹
        recent_ic = factor_ic_history.tail(20).mean()
        long_term_ic = factor_ic_history.tail(252).mean()
        weights = (long_term_ic - recent_ic)
        weights = weights / weights.abs().sum()

    elif method == 'vol_target':
        # 波动率目标：降低高波动因子的权重
        ic_vol = factor_ic_history.tail(252).std()
        ic_mean = factor_ic_history.tail(252).mean()
        weights = ic_mean / ic_vol
        weights = weights / weights.abs().sum()

    return weights
```

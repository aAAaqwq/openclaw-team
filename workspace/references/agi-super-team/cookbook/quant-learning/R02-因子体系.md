## 一、核心概念

### 1.1 什么是因子（Factor）

因子（Factor/Alpha）是能够解释股票收益率差异的变量。一个有效因子应满足：
- **预测性**：能预测未来收益率（正或负）
- **单调性**：因子值与收益率呈单调关系
- **稳定性**：在不同时间段有效
- **经济学逻辑**：有合理的解释

## 二、六大类因子详解

### 2.1 价值因子（Value）

**核心逻辑**：买入低估值的股票，做空高估值的股票

| 因子名称 | 计算公式 | 说明 |
|----------|----------|------|
| **PE（市盈率）** | Price / EPS | 越低越"便宜" |
| **PB（市净率）** | Price / BPS | 适合金融、地产 |
| **PS（市销率）** | Price / Revenue | 适合高增长公司 |
| **PCF（市现率）** | Price / Cash Flow | 现金流更难操纵 |
| **EV/EBITDA** | 企业价值/息税折旧前利润 | 跨行业可比 |
| **股息率** | DPS / Price | 高股息 = 价值 |

```python
# 价值因子计算示例
import pandas as pd
import numpy as np

def calc_value_factors(df):
    """计算价值因子
    df: 包含 price, eps, bps, revenue, cashflow, shares 列的 DataFrame
    """
    df['PE'] = df['price'] / df['eps']
    df['PB'] = df['price'] / df['bps']
    df['PS'] = df['price'] / (df['revenue'] / df['shares'])
    df['PCF'] = df['price'] / (df['cashflow'] / df['shares'])
    df['EV'] = df['price'] * df['shares'] + df['debt'] - df['cash']
    df['EBITDA'] = df['revenue'] - df['cost'] - df['sga'] + df['depreciation']
    df['EV_EBITDA'] = df['EV'] / df['EBITDA']

    # 价值得分 (低估值 = 高分)
    for col in ['PE', 'PB', 'PS']:
        df[f'{col}_rank'] = df[col].rank(ascending=True, pct=True)

    df['value_score'] = (
        df['PE_rank'] * 0.3 +
        df['PB_rank'] * 0.3 +
        df['PS_rank'] * 0.2 +
        df['PCF_rank'] * 0.2
    )
    return df
```

**A股价值因子特点**：
- 在A股长期有效，但2019-2021年成长风格压制价值
- 银行、地产、煤炭等低估值板块是价值因子的主要来源
- 周期性明显：经济上行期价值占优，下行期成长占优

### 2.2 动量因子（Momentum）

**核心逻辑**：过去表现好的股票未来继续表现好（趋势延续）

| 因子名称 | 计算方式 | 说明 |
|----------|----------|------|
| **过去N日收益率** | (P_t / P_{t-N}) - 1 | 经典动量 |
| **均线偏离度** | P_t / MA_N - 1 | 价格vs均线 |
| **动量加速度** | MOM_20 - MOM_60 | 短期vs长期动量变化 |
| **成交量动量** | VOL_t / VOL_{t-N} | 量能配合 |
| **跳空缺口** | Open_t - Close_{t-1} | 缺口动量 |

```python
def calc_momentum_factors(df, price_col='close'):
    """计算动量因子"""
    # 过去N日收益率
    for n in [5, 10, 20, 60, 120]:
        df[f'ret_{n}d'] = df[price_col].pct_change(n)

    # 均线偏离度
    for n in [5, 10, 20, 60]:
        df[f'ma{n}_bias'] = df[price_col] / df[price_col].rolling(n).mean() - 1

    # 动量加速度
    df['momentum_accel'] = df['ret_20d'] - df['ret_60d']

    # 成交量动量
    df['vol_mom_20'] = df['volume'] / df['volume'].rolling(20).mean()

    return df
```

**A股动量因子特点**：
- ⚠️ A股存在**反转效应**，与美股不同！
- 短期（1-5日）反转效应显著：涨多了会跌
- 中期（3-12月）动量效应存在但不稳定
- **涨跌停限制**使动量策略在A股表现特殊

### 2.3 质量因子（Quality）

**核心逻辑**：买入盈利能力强、财务健康的公司

| 因子名称 | 计算公式 | 说明 |
|----------|----------|------|
| **ROE** | Net Income / Equity | 核心盈利指标 |
| **ROA** | Net Income / Assets | 资产利用率 |
| **毛利率** | Gross Profit / Revenue | 竞争力 |
| **净利润增长率** | (NI_t - NI_{t-1}) / NI_{t-1} | 成长性 |
| **资产负债率** | Total Debt / Total Assets | 财务安全 |
| **经营现金流/净利润** | OCF / NI | 盈利质量 |

```python
def calc_quality_factors(df):
    """计算质量因子"""
    # 盈利能力
    df['ROE'] = df['net_income'] / df['equity']
    df['ROA'] = df['net_income'] / df['total_assets']
    df['gross_margin'] = (df['revenue'] - df['cost']) / df['revenue']

    # 成长性
    df['earnings_growth'] = df['net_income'].pct_change()
    df['revenue_growth'] = df['revenue'].pct_change()

    # 财务安全
    df['debt_ratio'] = df['total_debt'] / df['total_assets']
    df['current_ratio'] = df['current_assets'] / df['current_liabilities']

    # 盈利质量
    df['earnings_quality'] = df['operating_cf'] / df['net_income']

    # 综合质量得分
    df['quality_score'] = (
        df['ROE'].rank(pct=True) * 0.3 +
        df['gross_margin'].rank(pct=True) * 0.2 +
        df['earnings_growth'].rank(pct=True) * 0.2 +
        (1 - df['debt_ratio'].rank(pct=True)) * 0.15 +
        df['earnings_quality'].rank(pct=True) * 0.15
    )
    return df
```

### 2.4 规模因子（Size）

**核心逻辑**：小盘股长期跑赢大盘股（小盘股效应）

| 因子名称 | 计算方式 | 说明 |
|----------|----------|------|
| **总市值** | Price × Shares | 经典规模因子 |
| **流通市值** | Price × 流通股数 | A股更常用 |
| **对数市值** | ln(Market Cap) | 消除量纲 |

**A股规模因子特点**：
- 小盘股效应在A股非常显著（壳价值、炒作资金偏好）
- 但波动极大：2017年蓝筹行情、2021年小盘行情
- 科创板/创业板放宽涨跌幅后，规模因子表现有变化

### 2.5 波动率因子（Volatility）

**核心逻辑**：低波动率股票风险调整后收益更好（低波动率异象）

| 因子名称 | 计算方式 |
|----------|----------|
| **历史波动率** | std(daily_return) × √252 |
| **已实现波动率** | 基于5分钟数据的日内波动 |
| **特质波动率** | CAPM回归残差的标准差 |
| **波动率变化** | 当前波动率 / 过去60日波动率 |

**A股特点**：
- 低波动率异象在A股存在但不如美股显著
- 高波动率股票往往有更高的短期收益率（投机需求）

### 2.6 流动性因子（Liquidity）

**核心逻辑**：低流动性股票有流动性溢价

| 因子名称 | 计算方式 |
|----------|----------|
| **Amihud非流动性** | |return| / (price × volume) |
| **日均换手率** | Volume / Shares Outstanding |
| **买卖价差** | (Ask - Bid) / Mid |
| **零收益天数** | 日收益=0的天数占比 |

```python
def calc_liquidity_factors(df):
    """计算流动性因子"""
    # Amihud非流动性
    df['amihud'] = (df['close'].pct_change().abs() /
                    (df['close'] * df['volume'])).rolling(20).mean()

    # 换手率
    df['turnover'] = df['volume'] / df['shares_outstanding']

    # 零收益天数占比
    df['zero_ret_days'] = (df['close'].pct_change() == 0).rolling(20).mean()

    return df
```

## 三、因子有效性评估

### 快速评估框架

```python
import scipy.stats as stats

def evaluate_factor(factor_values, forward_returns):
    """
    评估因子有效性
    factor_values: 因子值 (DataFrame: stocks × dates)
    forward_returns: 未来收益率 (同维度)
    """
    results = {}

    # 1. IC (Information Coefficient) — 因子值与收益率的秩相关
    ic_series = factor_values.corrwith(forward_returns, method='spearman')
    results['IC_mean'] = ic_series.mean()
    results['IC_std'] = ic_series.std()
    results['IR'] = results['IC_mean'] / results['IC_std']  # Information Ratio

    # 2. IC > 0 的比例
    results['IC_positive_ratio'] = (ic_series > 0).mean()

    # 3. t统计量
    results['IC_tstat'] = results['IC_mean'] / (results['IC_std'] / len(ic_series)**0.5)

    # 4. 分层回测 (将股票分为5组)
    quantile_returns = {}
    for q in range(1, 6):
        mask = factor_values.rank(pct=True, axis=1).apply(
            lambda x: (q-1)/5 < x <= q/5, axis=0
        )
        quantile_returns[q] = (forward_returns * mask).mean(axis=1).mean()

    results['long_short'] = quantile_returns[5] - quantile_returns[1]
    results['monotonic'] = all(
        quantile_returns[i+1] > quantile_returns[i] for i in range(1, 5)
    )

    return results
```

### 因子有效性判断标准

| 指标 | 优秀 | 合格 | 无效 |
|------|------|------|------|
| IC均值 | > 0.05 | 0.03-0.05 | < 0.03 |
| IR | > 0.5 | 0.3-0.5 | < 0.3 |
| IC胜率 | > 55% | 50-55% | < 50% |
| t统计量 | > 3.0 | 2.0-3.0 | < 2.0 |
| 多空收益 | 显著 | 微弱 | 不显著 |

## 六、实战建议

1. **从简单开始**：先用PE/PB/ROE等基础因子练手
2. **关注A股特色**：壳价值、涨跌停、T+1等因素使很多海外因子需要改造
3. **避免过拟合**：不要通过尝试大量因子组合来拟合历史数据
4. **因子衰减**：因子有效性会随时间衰减，需持续监控
5. **因子正交化**：不同因子间可能高度相关，需要正交化处理

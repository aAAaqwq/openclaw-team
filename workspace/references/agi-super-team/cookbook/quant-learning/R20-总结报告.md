### 2.2 风险指标

| 指标 | 公式/定义 | 用途 |
|------|-----------|------|
| **VaR** | 95%分位数 | 最大可能损失 |
| **CVaR** | 超过VaR的均值 | 尾部风险 |
| **夏普比率** | (收益-无风险) / 波动率 | 风险调整收益 |
| **最大回撤** | 峰值到谷底最大跌幅 | 最坏情况 |
| **Calmar比率** | 年化收益 / 最大回撤 | 回撤补偿 |
| **IR** | IC均值 / IC标准差 | 因子预测稳定性 |
| **凯利公式** | f* = (p×b-q)/b | 最优仓位 |

## 七、关键认知总结

> **1. 因子是量化的基石。** 一切策略的本质都是因子（Alpha的来源）。理解因子体系是量化投资的第一步。

> **2. 回测≠实盘。** 过拟合是量化最大的敌人。Walk-Forward验证 + 样本外测试 + 模拟盘，缺一不可。

> **3. 风控>收益。** 活下来比赚钱重要。止损、仓位管理、分散投资，是长期生存的根本。

> **4. A股和BTC是两个完全不同的市场。** T+1 vs T+0、涨跌停 vs 无限制、监管严格 vs 相对自由。策略不能直接复制。

> **5. 从简单开始。** 不要一上来就用深度学习。先跑通均线策略，再逐步升级。

> **6. 数据质量决定一切。** Garbage in, garbage out. 数据清洗、复权、缺失值处理，比模型更重要。

> **7. 交易成本很重要。** 很多回测好看的策略，加上真实成本后就亏钱。

> **8. 持续迭代。** 市场在变化，策略会失效。持续监控、定期重训练、不断优化是常态。

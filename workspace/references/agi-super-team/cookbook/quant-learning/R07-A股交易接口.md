## 一、核心概念

A股实盘交易接口是量化交易从回测到实盘的关键桥梁。目前主要有以下几种途径：

```
A股量化实盘路径:

┌─────────────────────────────────────────┐
│            个人量化（门槛最低）             │
│    QMT (迅投) / PTrade (恒生)            │
│    → 通过券商开通，Python/minisoft        │
└─────────────┬───────────────────────────┘
              │
┌─────────────┼───────────────────────────┐
│         机构量化（需要资质）               │
│    XTP (中泰) / 恒生UFT / 飞马           │
│    → 需要私募备案/机构账户                │
└─────────────┬───────────────────────────┘
              │
┌─────────────┼───────────────────────────┐
│         期货量化（门槛中等）               │
│    CTP / 飞马 / 易盛                      │
│    → 期货公司开户即可                     │
└─────────────────────────────────────────┘
```

## 二、QMT (迅投) — 个人量化首选

### 2.1 简介

| 项目 | 说明 |
|------|------|
| 全称 | 迅投极速策略交易系统 (QMT) |
| 开发商 | 迅投科技 |
| 语言 | Python (xtquant) / miniSoft |
| 支持券商 | 国金、华鑫、中泰、中信建投等 30+ |
| 最低资金 | 通常 50-100万 |
| 延迟 | ~毫秒级（普通）/ ~微秒级（极速柜台） |

### 2.2 开通流程

```
1. 选择支持QMT的券商开户
2. 满足资金门槛（通常50-100万）
3. 申请开通QMT权限
4. 下载QMT客户端 + xtquant Python包
5. miniQMT 或 xtquant 模式接入
```

### 2.3 xtquant 交易示例

```python
# pip install xtquant  (从券商获取安装包)
from xtquant import xtdata
from xtquant.xttrader import XtQuantTrader, XtQuantTraderCallback
from xtquant.xttype import StockAccount

# ===== 行情订阅 =====

# 下载历史数据
xtdata.download_history_data('600519.SH', period='1d')

# 获取实时行情
quote = xtdata.get_full_tick(['600519.SH'])

# 订阅实时行情
def on_data(data):
    print(data)

xtdata.subscribe_quote('600519.SH', period='tick', callback=on_data)

# ===== 交易 =====

class MyCallback(XtQuantTraderCallback):
    def on_order_stock_async_response(self, response):
        print(f"下单响应: {response}")

    def on_stock_order(self, order):
        print(f"订单状态更新: {order.order_id, order.order_status}")

    def on_stock_trade(self, trade):
        print(f"成交通知: {trade.order_id, trade.traded_price, trade.traded_volume}")

# 初始化
path = r'C:\国金QMT\userdata_mini'  # miniQMT路径
session_id = 123456
trader = XtQuantTrader(path, session_id)
trader.register_callback(MyCallback())
trader.start()

# 连接账户
account = StockAccount('your_account_id', 'STOCK')
trader.connect()

# 下单
# 买入100股贵州茅台
order_id = trader.order_stock(
    account,
    stock_code='600519.SH',
    order_type=2,       # 2=市价单
    order_volume=100,   # 最小1手
    price_type=11,      # 11=市价最优五档即时成交剩余撤销
    price=0,            # 市价单价格填0
)

# 查询持仓
positions = trader.query_stock_positions(account)

# 查询订单
orders = trader.query_stock_orders(account)

# 撤单
trader.cancel_order_stock(account, order_id)
```

### 2.4 QMT优缺点

| 优点 | 缺点 |
|------|------|
| Python接口友好 | 仅支持Windows |
| 支持市价/限价多种订单 | 资金门槛较高 |
| 可自定义策略 | 极速柜台需额外费用 |
| 支持miniQMT轻量运行 | 不同券商版本有差异 |

## 三、XTP (中泰证券) — 机构级

### 3.1 简介

| 项目 | 说明 |
|------|------|
| 全称 | 中泰证券极速交易平台 (XTP) |
| 开发商 | 中泰证券 |
| 语言 | C++ / Python (xtp_api) |
| 延迟 | < 1ms（极速模式） |
| 适合 | 私募、机构 |
| 门槛 | 私募备案 + 机构账户 |

### 3.2 特点

- 业界最低延迟之一
- 支持融券、期权
- 完善的风控系统
- 提供C++和Python API

## 四、CTP (期货) — 最成熟的开源接口

### 4.1 简介

| 项目 | 说明 |
|------|------|
| 全称 | 综合交易平台 (Comprehensive Transaction Platform) |
| 开发商 | 上期技术 |
| 语言 | C++ / Python (vnpy/ctpbee) |
| 费用 | 免费 |
| 市场 | 期货（股指期货、商品期货） |
| 延迟 | 毫秒级 |

### 4.2 Python接入（通过vnpy）

```python
# pip install vnpy
# pip install vnpy-ctp

from vnpy.event import EventEngine
from vnpy.trader.engine import MainEngine
from vnpy_ctp import CtpGateway

# 创建引擎
event_engine = EventEngine()
main_engine = MainEngine(event_engine)
main_engine.add_gateway(CtpGateway)

# 连接
setting = {
    "用户名": "your_id",
    "密码": "your_password",
    "经纪商代码": "broker_id",
    "交易服务器": "tcp://180.168.146.187:10130",
    "行情服务器": "tcp://180.168.146.187:10131",
    "产品名称": "期货公司分配",
    "授权编码": "期货公司分配",
}
main_engine.connect(setting, "CTP")

# 查询合约
main_engine.subscribe(req, "CTP")

# 下单
from vnpy.trader.object import OrderRequest
from vnpy.trader.constant import Direction, OrderType

req = OrderRequest(
    symbol="IF2401",           # 沪深300股指期货
    exchange=Exchange.CFFEX,   # 中金所
    direction=Direction.LONG,  # 买入
    type=OrderType.LIMIT,      # 限价单
    volume=1,                  # 1手
    price=3500.0,
)
order = main_engine.send_order(req, "CTP")
```

## 六、实盘交易注意事项

### 6.1 风控规则

| 规则 | 说明 |
|------|------|
| **单笔限额** | 不超过总资金的一定比例 |
| **日止损** | 设定日亏损上限（如2%） |
| **总止损** | 设定总亏损上限（如10%） |
| **持仓集中度** | 单只股票不超过总仓位X% |
| **异常处理** | 网络断线、订单失败的应急方案 |

### 6.2 实盘 vs 回测差异

| 差异 | 说明 |
|------|------|
| **滑点** | 实际成交价 ≠ 回测价格 |
| **冲击成本** | 大单会影响市场价格 |
| **延迟** | 信号到执行的延迟 |
| **涨跌停** | 涨跌停时无法成交 |
| **停牌** | 策略信号无法执行 |
| **资金限制** | 回测无限资金，实盘有限 |

### 6.3 建议的实盘流程

```
回测通过 → 模拟盘(1-3个月) → 小资金实盘(3-6个月) → 逐步加仓
```

## 七、选择建议

| 阶段 | 推荐接口 | 原因 |
|------|----------|------|
| **学习** | 无需实盘 | 先做好回测 |
| **期货入门** | CTP (via vnpy) | 免费、成熟 |
| **A股个人** | QMT (xtquant) | Python友好 |
| **A股机构** | XTP/恒生UFT | 低延迟、功能全 |

# 第10章：广告投放（付费放大）

**本章核心问题**

- 什么时候该开始投广告，什么时候不该投？
- 广告账户健康度该怎么看？
- Google Ads、Amazon Ads、Meta Ads分别适合什么场景？
- 为什么很多广告越投越亏，问题不在素材，而在结构？

---

## 10.1 广告投放的底层逻辑：先证明，再放大

### 原理说明

广告不是救命药，是放大器。

如果你的内容、产品、落地页、转化路径本身不成立，投广告只会更快地把钱烧光。很多团队犯的错误是：自然流量做不起来，就想用广告“补回来”。这是最贵的幻想。

广告的正确使用时机只有一个：**你已经用自然流量或小规模测试证明，这个内容/产品/人群组合是能转化的，然后用广告去放大。**

付费投放的四个底层变量：

| 变量 | 说明 | 判断标准 |
|------|------|---------|
| Offer | 你卖什么/承诺什么价值 | 用户看到是否想点 |
| Audience | 你投给谁 | 人群是否足够精准 |
| Creative | 你怎么表达 | 3秒内能否抓住注意力 |
| Funnel | 用户点击后发生什么 | 页面/表单/购买链路是否顺畅 |

这四个变量里，只优化Creative最容易，也是最没用的短视行为。真正决定ROI的，通常是Audience和Funnel。

### 实操步骤

**第一步：投放前自检（必须全部满足）**

| 检查项 | 标准 |
|--------|------|
| 自然流量验证 | 至少有1-2个内容/产品在自然流量下已证明能转化 |
| 转化追踪 | Pixel/GA/事件埋点已正常工作 |
| 落地页 | 页面打开快、价值主张清晰、CTA明确 |
| 目标明确 | 品牌曝光/获客/带货，只能先选一个主目标 |
| 预算准备 | 有连续测试预算，不是“先试300块看看” |

**第二步：先小预算测试，再逐步放大**

测试不是为了赚钱，是为了找出：
- 哪个人群最便宜
- 哪个素材点击率高
- 哪个落地页转化最好

建议起始预算：
- Meta：单广告组日预算 ≥ 5×目标CPA
- TikTok：单广告组日预算 ≥ 50×目标CPA（更吃学习期）
- Google：按关键词竞争强度设置，但至少覆盖7天以上学习样本

**第三步：建立账户结构**

最怕的不是花钱，而是结构混乱。一个清晰的账户结构应该做到：

```
Campaign（目标层）
  ├─ Ad Set / Ad Group（人群/关键词层）
  │    ├─ Ad 1（素材A）
  │    ├─ Ad 2（素材B）
  │    └─ Ad 3（素材C）
```

结构规则：
- 不要一个广告组混太多人群
- 不要一个活动里混多个核心目标
- 不要学习期还频繁改预算、改素材

**第四步：按数据做“保、停、放”**

- **保留**：CTR、CVR、CPA稳定达标
- **暂停**：CPA > 目标CPA的3倍（3x Kill Rule）
- **放大**：已稳定3-7天且ROI为正，再逐步加预算

### 参数配置

| 参数项 | 推荐设置 | 说明 |
|--------|----------|------|
| 测试周期 | 7-14天 | 给算法学习时间 |
| Kill Rule | CPA > 目标3倍暂停 | 避免持续烧钱 |
| 扩量节奏 | 单次加预算10%-20% | 避免打断学习 |
| 核心目标数 | 1个主目标 | 不要一锅炖 |
| 素材测试量 | 每组2-4个 | 足够比较，不至于过散 |

### 常见问题

**Q：自然流量没跑通，可以直接投广告测试吗？**
可以，但前提是你明确知道自己在测试什么，而且预算是“学习成本”，不是“盈利预算”。如果什么都没验证，上来就大预算投放，基本是交学费。

**Q：广告没效果，是素材问题还是人群问题？**
先看链路：点击率低，多半是素材或Offer问题；点击率高但转化低，多半是页面或人群问题；全都低，说明产品/定位本身就有问题。

### 避坑指南

| 错误 | 后果 | 解决方案 |
|------|------|---------|
| 没验证就大投 | 快速烧钱 | 先小预算测试 |
| 学习期频繁改动 | 算法一直重置 | 至少等3-7天再判断 |
| 目标混乱 | 数据无法解释 | 一个Campaign一个主目标 |
| 只看曝光/点击 | 以为热闹就是效果 | 盯住CPA、ROAS、CVR |

## 10.2 Google Ads：高意图流量的精细化收割

### 原理说明

Google Ads最适合做**高意图流量收割**。用户已经在搜索，问题不是“要不要教育他”，而是“你能不能成为他看到的最佳答案”。

Google Ads最值钱的资产不是预算，而是**关键词结构和转化数据**。关键词选错，账户再大也白搭。

### 实操步骤

**第一步：先查配置和模式**

如果有API，可以批量分析；没有API，也可以用浏览器模式检查账户。关键检查项：
- Campaign表现
- 零转化关键词
- 转化跟踪是否正常
- 优化分数是否过低

**第二步：找出浪费花费的关键词**

重点筛选：
- Conversions < 1
- Cost > 指定阈值（如$500）
- 最近30-90天持续烧钱

这类词通常就是账户里的“漏血点”。

**第三步：看Campaign层级数据**

建议按表格看：

| Campaign | Cost | Conv | CPA | Status |
|----------|------|------|-----|--------|
| Brand | $5,000 | 50 | $100 | 保留 |
| Generic Search | $8,000 | 20 | $400 | 优化 |
| PMax | $10,000 | 5 | $2,000 | 审核/暂停 |

**第四步：遵守Google Ads硬规则**

- 不要在没有Smart Bidding时盲上Broad Match
- 3x Kill Rule：CPA超过目标3倍就处理
- 学习期别乱动
- 数据驱动归因优先于拍脑袋判断

### 参数配置

| 参数项 | 推荐设置 | 说明 |
|--------|----------|------|
| 审核周期 | 每周一次 | 及时发现漏损 |
| 零转化关键词阈值 | 30-90天 + 花费阈值 | 避免误杀新词 |
| 优化分数警戒线 | <70%关注 | 不是绝对指标，但有参考 |
| 关键词匹配 | Exact/Phrase优先 | 预算小的时候更可控 |
| 归因模型 | Data-driven优先 | 更接近真实路径 |

### 常见问题

**Q：为什么我的关键词点击很多但没转化？**
可能是三个原因：关键词意图太泛、广告文案承诺和落地页不匹配、页面本身转化差。先别急着换素材，先检查搜索词报告和落地页一致性。

**Q：PMax到底要不要投？**
可以投，但前提是你已经有足够好的素材资产和转化数据。没有数据喂给系统，PMax只会变成黑盒烧钱器。

### 示例

最近30天账户里发现3个关键词：
- “free AI tool” 花费$620，0转化
- “best AI software” 花费$540，1转化
- “AI tool for accountants” 花费$210，6转化

结论：
1. 泛词竞争大、转化差，应收缩
2. 长尾意图词虽然量小，但转化效率高，值得扩量
3. 预算应从泛词转向高意图长尾词

## 10.3 Amazon Ads：用广告推动商品，而不是幻想广告替代产品力

### 原理说明

Amazon Ads和Google Ads最大的区别在于：用户已经在购物环境里，离购买更近。但也因为更近，产品页本身不行，广告效果会立刻暴露问题。

Amazon广告投放的核心不是“让更多人看到”，而是“让已经接近购买的人更快下单”。所以广告效率高度依赖：
- Listing标题和主图
- 价格竞争力
- Review评分和数量
- 类目竞争环境

### 实操步骤

**第一步：先看广告账户和Profile**

通过Amazon Ads API或后台，先确认：
- 当前Profile是否正确
- 有哪些Sponsored Products Campaigns在跑
- 预算是否耗尽过快
- 哪些Campaign有稳定转化

**第二步：做Campaign分层**

常见做法：
- 自动投放：找词
- 手动精准：吃高转化词
- 手动广泛：拓词
- 品牌防守：守自己的品牌词

**第三步：看关键词与ASIN表现**

重点找三类对象：
1. 高花费低转化词 → 降价/暂停
2. 低花费高转化词 → 加预算/提价
3. 高曝光低点击词 → 检查主图、价格、标题竞争力

**第四步：广告与Listing联动优化**

Amazon广告不是孤立系统。CTR低时，未必是广告词不行，也可能是主图差；CVR低时，未必是流量不准，也可能是评分不够高。

### 参数配置

| 参数项 | 推荐设置 | 说明 |
|--------|----------|------|
| 广告类型 | SP优先 | 新手从Sponsored Products开始 |
| 预算分配 | 60%手动精准 + 20%自动 + 20%拓词 | 相对稳妥 |
| 审核周期 | 每周 | 电商变化快 |
| 关键词决策阈值 | 看7-14天数据 | 避免当天波动 |
| 核心指标 | ACoS、CTR、CVR | 三个一起看 |

### 常见问题

**Q：广告有点击没订单，先改广告还是改Listing？**
先改Listing。因为用户点击进来已经很接近购买，如果还不下单，问题多半出在页面说服力、价格或评价。

**Q：自动投放是不是浪费钱？**
不是。自动投放适合找词，但不能长期无脑跑。找到高转化搜索词后，要迁移到手动精准Campaign里吃利润。

### 示例

某Campaign数据：
- 自动投放：花费$300，找到2个高转化搜索词
- 手动精准：这2个词花费$180，产生12单
- 结论：自动投放负责发现机会，手动精准负责稳定放大

## 10.4 Meta / Facebook Ads：兴趣人群与创意迭代的战场

### 原理说明

Meta Ads的核心不是“精准到一个人”，而是**通过创意和信号，让系统找到最有可能转化的人**。

很多团队还在迷信“手动兴趣叠加=精准投放”。现在的Meta更看重：
- Pixel / CAPI数据是否健康
- 素材是否能快速抓住注意力
- 转化事件是否清晰
- 学习期是否被打断

### 实操步骤

**第一步：先读账户目标和历史**

在Meta体系里，优化动作不能只看今天数据。要看最近3天甚至更长时间的历史，避免刚降过预算又继续降，导致账户来回震荡。

优化前先确认：
- 目标CPL / CPA / ROAS
- 最近3天是否做过预算调整
- 当前Ad Set是否仍在学习期
- 哪些广告在吃量，哪些广告是“广告吞预算器”

**第二步：按账户健康度分组**

可按表现粗分：
- very good：继续保留，准备扩量
- good：观察，轻微优化
- neutral：等待更多样本
- bad：准备暂停或替换素材

**第三步：预算与结构管理**

预算规则：
- 单Ad Set预算至少是目标CPA的5倍
- 学习期不要频繁改预算
- 放量时每次上调10%-20%

结构规则：
- 不要一个Ad Set塞太多创意
- 不要同时测试人群和素材，避免变量混乱
- 明确每个Campaign的目标：获客、再营销、转化，分开跑

**第四步：创意分析与替换**

Meta是创意平台。CTR低、Thumb-stop低、频次升高，通常意味着创意疲劳。

创意替换优先级：
1. 开头3秒钩子
2. 主视觉/第一帧
3. 核心承诺文案
4. CTA

### 参数配置

| 参数项 | 推荐设置 | 说明 |
|--------|----------|------|
| 日预算 | ≥5×目标CPA | 否则学习不充分 |
| 学习观察期 | 48-72小时 | 新Ad Set至少给时间 |
| 扩量幅度 | 10%-20%/次 | 避免震荡 |
| 创意测试数 | 每组2-3个 | 控制变量 |
| 归因窗口 | 7-day click / 1-day view | 常见默认配置 |

### 常见问题

**Q：Meta广告突然变差，要不要立刻暂停？**
先看是不是学习期、节假日波动、受众疲劳或追踪异常。不要见红就砍。先看48小时数据，再做决定。

**Q：一个创意跑得很好，要不要复制10份扩量？**
不要。复制不是扩量，很多时候只是让自己和自己竞争。优先做相似但不完全重复的新创意迭代。

### 避坑指南

| 错误 | 后果 | 解决方案 |
|------|------|---------|
| 学习期频繁改预算 | 一直学不会 | 48-72小时内少动 |
| 人群和素材一起测 | 无法判断问题在哪 | 一次只测一个变量 |
| 忽略历史操作 | 账户来回震荡 | 先看最近3天动作记录 |
| 只追求低CPM | 便宜流量不一定转化 | 最终看CPA/ROAS |

### 示例

某教育产品账户：
- Ad Set A：日预算$50，CPA $8，稳定3天 → 可加预算10%
- Ad Set B：日预算$50，CPA $27（目标CPA $8）→ 超过3x Kill Rule，暂停
- Ad Set C：CTR高但Lead Form提交率低 → 问题不在创意，在表单或承接页

## 10.5 章节小结

广告投放不是玄学，也不是“多烧点预算就能赢”。它是一个标准工程问题：**先验证，再放大；先看结构，再看素材；先看转化，再看热闹。**

**第一，广告是放大器，不是修复器。** 产品不行、页面不行、追踪不行，投广告只会更快暴露问题。

**第二，不同平台承担不同角色。** Google Ads收割高意图搜索流量，Amazon Ads推动近购买场景转化，Meta Ads依靠创意和系统信号寻找人群。

**第三，账户管理的核心是纪律。** 学习期不乱动、3x Kill Rule及时止损、预算渐进扩量、每次只测试一个关键变量。

**行动清单：**

1. 今天：检查你的广告投放是否满足“先验证再放大”的前提
2. 本周：为当前账户建立清晰结构，删除混乱Campaign
3. 本周：找出过去30天里所有零转化高花费对象，逐一处理
4. 本月：建立固定广告复盘模板，不再只看表面点击量

---

*写作风格注记：广告不是讲概念，讲纪律。该停就停，该保就保，该放再放。数据不骗人，账户结构会说话。*

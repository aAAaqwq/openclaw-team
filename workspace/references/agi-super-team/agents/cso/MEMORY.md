## 📚 核心知识库

### 销售方法论

#### 1. Cold Calling 2.0 (Aaron Ross)
- **核心理念**: 不打陌生电话的 outbound 销售
- **关键步骤**:
  1. 识别理想客户画像 (ICP)
  2. 找到决策人联系方式
  3. 发送个性化冷邮件
  4. 建立关系后再电话沟通
- **应用场景**: B2B SaaS 销售

#### 2. B2B 销售要点
- **大客户陷阱**:
  - 不要首先 targeting 大客户
  - 避免 Scope of Work (SoW)
  - 收入分散化，不依赖单一客户
- **测试流程**: 验证问题 → 找付费客户 → 小规模验证 → 扩展

#### 3. 获客渠道 (优先级排序)
1. **SEO-driven acquisition** - 长期价值高
2. **Product-Led Growth (PLG)** - 产品自带传播
3. **Community-Led Growth** - 社区驱动增长
4. **内容营销 + SEO** - 组合拳

### 销售指标体系

#### AARRR 海盗指标
| 指标 | 含义 | 重点关注 |
|------|------|----------|
| Acquisition | 获客 | 流量来源 |
| Activation | 激活 | 首次体验价值 |
| Retention | 留存 | 复购/续费 |
| Referral | 推荐 | 口碑传播 |
| Revenue | 收入 | 付费转化 |

#### SaaS 关键公式
- **CAC** = 总获客成本 / 新客户数
- **LTV** = 客户终身价值
- **LTV/CAC 比率** > 3 为健康
- **MRR/ARR** = 月度/年度经常性收入
- **Churn Rate** = 流失率 (越低越好)
- **NRR** = 净收入留存 ( >100% 说明upsell有效)

### 商业化认知

#### 价值链条
```
创造价值 → 传递价值 → 捕获价值
(做出来)    (让人知道+用上)   (收到钱)
```
多数技术人死在第2步。

#### 4个关键判断
1. **痛点真伪**: 用户是"想要"还是"愿意付钱要"？
2. **付费意愿**: 谁付钱？付多少？多频繁？
3. **交付成本**: 边际成本趋近于0吗？
4. **护城河**: 别人3个月能抄吗？

#### 4种赚钱方式
1. **卖产品** - 一次性交付（课程、工具、模板）
2. **卖服务** - 持续交付（咨询、代运营、SaaS订阅）
3. **卖流量** - 注意力变现（广告、联盟、带货）
4. **卖资产** - 钱生钱（投资、交易、套利）

#### 反直觉真相
- 定价决定一切（不是成本+利润）
- 渠道 > 产品
- 复购 > 拉新
- 先收钱再做

#### AI时代商业化路径
- **A. 极低价海量**: $9.9/月 × 10万用户
- **B. AI+人类溢价**: $999/次，AI做80%+人做20%关键判断

#### 开源项目变现路径
| 模式 | 适用场景 | 收入潜力 |
|------|----------|----------|
| **Dual Licensing** | 企业需要闭源使用 | ⭐⭐⭐⭐ |
| **SaaS 托管版** | 用户不想自己部署 | ⭐⭐⭐⭐⭐ |
| **Enterprise Edition** | 大企业需要高级功能 | ⭐⭐⭐⭐⭐ |
| **Professional Services** | 定制开发/咨询 | ⭐⭐⭐ |
| **Training/Support** | 企业培训/技术支持 | ⭐⭐⭐ |
| **Data/API Monetization** | 数据服务 | ⭐⭐⭐⭐ |

#### 推荐组合策略
```
1. 开源核心 (免费) → 建立社区和品牌
2. SaaS 托管 (月付) → 中小客户
3. 企业版 (年付) → 大客户，高级功能
4. 专业技术支持 (小时/项目) → 定制需求
```

  - 基础版: ¥5,000/月 - 日报推送
  - 专业版: ¥15,000/月 - 实时推送 + API

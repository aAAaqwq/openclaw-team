# 歌词节拍分析指南 (Beat Analysis)

## 概述
歌词节拍分析是将歌词文本转化为可执行 MV 时间轴的关键步骤。核心目标：**精确到秒**。

## 一、歌曲结构识别

### 标准流行歌曲结构
```
Intro → Verse 1 → Pre-Chorus → Chorus → Verse 2 → Pre-Chorus → Chorus 2 → Bridge → Final Chorus → Outro
```

### 各段落特征

| 段落 | 英文 | 通常时长 | 情绪特点 | 乐器密度 |
|------|------|---------|---------|---------|
| 前奏 | Intro | 8-16s | 铺垫/悬念 | 最低 |
| 主歌 | Verse | 20-30s | 叙事/平稳 | 中低 |
| 预副歌 | Pre-Chorus | 10-15s | 铺垫/渐强 | 中等渐强 |
| 副歌 | Chorus | 20-30s | 高潮/爆发 | 最高 |
| 桥段 | Bridge | 10-15s | 转折/反差 | 变化大 |
| 尾奏 | Outro | 8-15s | 收束/余韵 | 递减 |

## 二、BPM 与场景时长映射

### BPM 基准

| 曲风 | 典型 BPM | 每拍时长 | 4小节时长 |
|------|---------|---------|----------|
| 抒情慢歌 | 60-80 | 0.75-1.0s | 12-16s |
| 中板流行 | 80-110 | 0.55-0.75s | 8.7-12s |
| 电子/舞曲 | 110-130 | 0.46-0.55s | 7.4-8.7s |
| 快节奏 | 130-160 | 0.375-0.46s | 6-7.4s |

### 《樱花落尽时》BPM 分析
- **建议 BPM**: 90-100（国风电子流行，中板偏慢）
- **每拍**: ~0.6-0.67s
- **4小节**: ~9.6-10.7s
- **8小节**: ~19.2-21.3s

### 时长计算公式
```
段落时长(秒) = 小节数 × 4拍/小节 × 60/BPM
```

## 三、卡点逻辑 (Beat Sync)

### 什么是卡点？
MV 分镜切换、运镜变化、特效触发与音乐节拍对齐的技术。

### 关键卡点时机

#### 1. 段落切换卡点（必须）
```
Verse → Pre-Chorus: 鼓点加重的第一拍
Pre-Chorus → Chorus: 贝斯/合成器爆发的第一拍
Chorus → Verse: 节奏收紧的瞬间
Bridge → Final Chorus: 静→动的转折点
```

#### 2. 歌词重点卡点（推荐）
```
关键词/核心歌词: 配合视觉爆发
歌词断句: 配合运镜切换
长音/拖音: 配合慢动作或延时报
```

#### 3. 音乐元素卡点（高级）
```
鼓点重击: 快速切镜
贝斯 Drop: 场景突变
弦乐拉高: 升降镜头
合成器琶音: 快速蒙太奇
```

### 卡点强度等级

| 等级 | 说明 | 分镜处理 |
|------|------|---------|
| Hard Cut | 强制卡点，瞬间切换 | 硬切/闪白/闪黑 |
| Soft Sync | 柔性卡点，渐变过渡 | dissolve/crossfade |
| Free Flow | 不卡点，自由流动 | 长镜头/慢转场 |

## 四、情绪曲线计算

### 情绪评分维度 (0-10)

```python
emotion_score = (
    keyword_weight * 0.3 +    # 关键词权重（如"泪"/"痛"高分）
    structure_weight * 0.25 + # 段落位置权重（副歌>主歌）
    dynamics_weight * 0.25 +  # 动态变化（渐强/渐弱）
    instrument_weight * 0.2   # 乐器密度（多=高分）
)
```

### 关键词情绪映射

| 关键词类别 | 示例 | 情绪分 |
|-----------|------|--------|
| 破碎/离别 | 落尽、散、碎、断 | 7-9 |
| 温柔/美好 | 晨曦、花瓣、柔光 | 3-5 |
| 力量/爆发 | 燃烧、怒放、咆哮 | 8-10 |
| 孤独/思念 | 独、忆、空、远 | 5-7 |
| 治愈/希望 | 亮、新、生、暖 | 4-6 |

## 五、时间轴精确对齐

### 对齐原则
1. **段落边界必须对齐** — Verse/Chorus 切换精确到秒
2. **歌词行对齐** — 每行歌词对应场景中的视觉变化
3. **呼吸点** — 歌词间留白处配合空镜或慢动作
4. **总时长守恒** — 所有段落时长之和 = 歌曲总时长

### 时间轴标注格式
```
[MM:SS] 段落标记 | 歌词内容 | 情绪分 | 卡点标记
```

# 场景映射指南 (Scene Mapping)

## 概述
基于歌词情绪和段落结构，为每个分镜分配最匹配的场景类型、光源和氛围。

## 一、情绪→场景映射表

### 核心映射规则

| 情绪区间 | 情绪标签 | 推荐场景类型 | 视觉氛围 | 色调 |
|----------|---------|-------------|---------|------|
| 0-2 | 极静/空灵 | 天空、云海、水面倒影 | 梦幻空灵 | 淡蓝/白 |
| 2-3 | 宁静/期待 | 晨雾溪畔、花间小径 | 柔和清新 | 晨曦白+浅绿 |
| 3-4 | 平静/温柔 | 樱花林、庭院、书房 | 温馨安静 | 落樱粉+白 |
| 4-5 | 微伤/怀念 | 雨巷、旧窗台、秋千 | 怀旧淡淡忧伤 | 灰蓝+淡粉 |
| 5-6 | 铺垫/渐强 | 回廊、走廊、阶梯 | 压抑蓄力 | 暖黄→橙 |
| 6-7 | 紧张/期待 | 暴风前夕、乌云、闪电前 | 张力 | 深紫+暗金 |
| 7-8 | 爆发/力量 | 暴风雨中、火焰旁、山顶 | 强烈冲击 | 红橙+金 |
| 8-9 | 极致/燃烧 | 星空下、光柱中、碎裂空间 | 磅礴宏大 | 深蓝+白光 |
| 9-10 | 巅峰/超越 | 多重空间叠化、宇宙、极光 | 超现实 | 全光谱 |

### 段落→场景优先级

| 段落 | 优先场景 | 备选场景 |
|------|---------|---------|
| Intro | 空景→远景→人物剪影 | 水面倒影、云海 |
| Verse | 小场景、近景为主 | 室内一角、窗前 |
| Pre-Chorus | 过渡场景，有纵深感 | 走廊、阶梯、回廊 |
| Chorus | 大场景、开阔 | 山顶、海边、星空下 |
| Bridge | 反差场景 | 废墟→花园、暗→光 |
| Outro | 回归开头的场景 | 呼应 Intro |

## 二、Chloe 专属场景库

### A 级场景（核心场景，至少使用2次）

#### 🌸 溪畔樱花林
- **位置**: 故事主场景
- **视觉**: 溪水潺潺，两岸樱花树，花瓣飘落
- **光源**: 早晨 golden hour，柔和侧光
- **配色**: 晨曦白 + 落樱粉 + 浅草绿
- **适合**: Intro, Verse, Outro
- **运镜**: 慢推入，樱花飘落做前景

#### 🌙 月下庭院
- **位置**: 高潮转折
- **视觉**: 日式庭院，石灯笼，月光洒落
- **光源**: 月光 + 花灯暖光
- **配色**: 月白 + 暖黄 + 墨蓝
- **适合**: Chorus, Bridge
- **运镜**: 环绕拍摄，月光做逆光

#### 🌧️ 雨巷
- **位置**: 情感转折
- **视觉**: 青石板路，油纸伞/蕾丝折伞，雨滴溅起
- **光源**: 阴天散射光 + 远处暖窗光
- **配色**: 灰蓝 + 暖橙 + 白
- **适合**: Pre-Chorus, Bridge
- **运镜**: 跟拍，雨滴做前景虚化

### B 级场景（辅助场景，使用1-2次）

#### 🌅 晨雾溪畔
- **位置**: 开场/结尾
- **视觉**: 薄雾笼罩的溪水，若隐若现
- **光源**: 黎明前柔光，光线穿透雾气
- **配色**: 灰白 + 淡金 + 浅蓝
- **适合**: Intro, Outro
- **运镜**: 缓慢航拍式前进

#### 🏔️ 山顶远眺
- **位置**: 高潮爆发
- **视觉**: 悬崖边，远景山脉，云海翻涌
- **光源**: 日出/日落 golden hour
- **配色**: 暖金 + 深蓝 + 橙红
- **适合**: Final Chorus
- **运镜**: 升降镜头，从低到高

#### 🪟 窗前光影
- **位置**: 内省段落
- **视觉**: 和室窗户，光影投射，花瓣贴在玻璃上
- **光源**: 斜射阳光，窗花投下阴影
- **配色**: 暖白 + 木色 + 影灰
- **适合**: Verse 2
- **运镜**: 固定+微摇，光斑变化

#### 🌌 星空下
- **位置**: 奇幻高潮
- **视觉**: 开阔地带，银河横跨，萤火虫
- **光源**: 星光 + 萤火虫暖光
- **配色**: 深蓝 + 银白 + 暖黄
- **适合**: Final Chorus
- **运镜**: 仰拍→平移→俯拍

### C 级场景（特殊场景，按需使用）

- **碎裂空间**: 超现实，镜面破碎，花瓣化作光点
- **水中倒影**: 半身入水，长发漂浮
- **古风回廊**: 朱红柱子，长廊延伸
- **花海**: 整片花田，Chloe 站立其中
- **枯枝冬景**: 反差场景，凋零之美


## 四、场景转换逻辑

### 转换类型

| 转换方式 | 适用情况 | 时长 |
|----------|---------|------|
| dissolve | 情绪平缓过渡 | 1-2s |
| crossfade | 空间转换 | 0.5-1s |
| 硬切 | 强卡点、节奏变化 | 0s |
| 闪白/闪黑 | 爆发点、高潮 | 0.3s |
| 匹配剪辑 | 视觉连续（花瓣→雨滴） | 0.5s |
| 遮罩转场 | 用前景物体划过 | 0.5-1s |

### 转换原则
1. **情绪平缓** → dissolve / crossfade
2. **节奏突变** → 硬切 / 闪白
3. **视觉延续** → 匹配剪辑（前一帧元素=后一帧元素）
4. **空间跳跃** → 遮罩转场 / 闪黑

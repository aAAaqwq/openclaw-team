# 即梦AI数字人界面字段说明

## 平台信息

- **地址**: https://jimeng.jianying.com/ai-tool/home?type=digitalHuman&workspace=0
- **所属**: 剪映旗下AI创作平台
- **功能**: 数字人口播视频生成

## 界面核心字段

### 角色说（必填）

数字人的口播台词文本。填入后数字人会以该文本进行语音合成并配合口型。

- 直接输入文字即可
- 支持中文、英文
- 建议每段不超过200字（过长可能影响生成质量）

### 动作描述（可选）

添加动作描述和镜头语言，指导数字人的表演风格和镜头运动。

- 示例：`镜头推进，他摘下眼镜，对着镜头笑着说`
- 可描述：表情、手势、身体动作、镜头运动
- 不填则使用默认站姿/坐姿口播

## 其他界面选项

| 选项 | 说明 |
|------|------|
| **数字人形象** | 选择不同的数字人模型，如"爱牧男2"等 |
| **快速模式** | 加速生成，可能降低部分质量 |
| **上传音频** | 用自定义音频替代TTS合成 |

## 生成与下载

- 点击发送按钮后等待生成（通常1-3分钟/段）
- 生成完成后可直接下载MP4视频文件
- 每段视频独立生成，需分别下载

## 分镜头操作流程

1. 打开即梦数字人页面
2. 选择数字人形象
3. 在"角色说"输入第1个镜头台词
4. 在"动作描述"输入对应动作描述
5. 点击生成，等待完成，下载视频
6. 重复3-5步完成所有镜头
7. 所有分段视频下载后，用 video-merge-send skill 合并

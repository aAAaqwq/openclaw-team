# Dan Koe 思维框架：Human 3.0

> "Level 1是NPC，运行预设脚本；Level 2是主角，选择自己的故事线；Level 3是程序员，创造让别人也喜欢玩的新游戏。" —— Dan Koe

## 核心理念

Dan Koe 在15年的自我探索中，融合了心理学、哲学、个人发展、科技、创业、灵性和意义的research，最终形成了 Human 3.0 框架。

**核心主张**：传统成长模型都是孤立的一个领域——健身教练不管灵性，创业者不管健康，spiritual guru不管财务。真正的全面发展必须同时覆盖四个维度。

**核心问题**：单一维度的极致发展，带不来整体人生的进化。

---

## 一、四象限模型（Four Quadrants）

Human 3.0 的底层结构借鉴了 Ken Wilber 的 AQAL 模型，但做了现代化和实用化的改造：

```
         内在（Interior）          外在（Exterior）
        ┌──────────────┐          ┌──────────────┐
  个人   │   MIND       │          │   BODY      │
        │   心智       │          │   身体       │
        │  思想/情绪   │          │  行为/外貌   │
        │  /信念系统   │          │  /习惯/营养  │
        └──────────────┘          └──────────────┘
  集体   │   SPIRIT     │          │   VOCATION  │
        │   精神       │          │   使命       │
        │  关系/社群   │          │  职业/事业   │
        │  /文化/意义   │          │  /经济系统   │
        └──────────────┘          └──────────────┘
```

### 象限详解

| 象限 | 定义 | 问题类型 | 典型发展需求 |
|------|------|----------|-------------|
| **Mind（心智）** | 你的内部世界——思想、情绪、信念 | 我如何看待世界？如何思考？ | 心理学、冥想、认知重构 |
| **Body（身体）** | 你的外部表现——行为、习惯、外貌 | 世界如何看待我？我如何呈现？ | 训练、营养、睡眠、仪表 |
| **Spirit（精神）** | 你与集体的关系——社群、文化、意义 | 我与他人如何连接？生命的意义是什么？ | 关系、community、信仰 |
| **Vocation（使命）** | 你与系统的关系——职业、经济、贡献 | 我在社会经济系统中处于什么位置？如何创造价值？ | 技能、商业、创作 |

**关键洞察**：四个象限相互解锁。身体差则心智低效（能量不足）；职业失败则精神空虚；精神混乱则身体出问题。**跨象限协同改进才是最快的进化方式。**

## 二、三级意识进化（Levels）

每个象限内部都有三个宏观的意识层次：

| 层次 | 名称 | 特征 | 比喻 |
|------|------|------|------|
| **1.0** | 顺从者（Conformist）| 依赖外部权威，黑白思维，"只有一条正确道路" | NPC（非玩家角色），运行预设脚本 |
| **2.0** | 个人主义者（Individualist）| 拒绝规范，追求个人目标，寻求地位和认可，"我的方式才是对的" | 主角，选择自己的故事线 |
| **3.0** | 综合者（Synthesist）| 能采纳多重视角，连接不同现实模式，为他人创造新游戏 | 程序员，创造新游戏让其他人玩 |

**重要规则**：
- 你不会"离开"某个层次——你**超越并包含（transcend and include）**它
- 你可以同时处于不同象限的不同层次（例如：职业3.0，但灵性1.0）
- 3.0的人仍然可以有意图地展现1.0的特征——那是主动选择，而非被困

**典型表现示例**：

| 象限 | 1.0 顺从者 | 2.0 个人主义者 | 3.0 综合者 |
|------|-----------|---------------|-----------|
| Mind | 相信所有权威观点 | 拒绝一切权威，形成独特世界观 | 理解各派观点，灵活切换 |
| Body | 照搬健身计划，不思考原理 | 疯狂Biohacking，追逐最新研究 | 将运动整合为生活方式 |
| Spirit | 盲目皈依某个宗教/意识形态 | 不断换灵性老师/哲学（Spiritual Shopper） | 建立自己的意义系统 |
| Vocation | 追求安稳的"铁饭碗" | 创业/自由职业，追求财务独立 | 创造价值系统，让他人参与 |

## 三、三相位成长（Phases）

每个层次内部都有三个必须经历的成长阶段：

```
Phase 1: Dissonance（矛盾期）
   ↓  "够了，但我不知道下一步是什么"
Phase 2: Uncertainty（不确定期）
   ↓  "我愿意走进未知，接受新知识"
Phase 3: Discovery（发现期）
   ↓  "我找到了方向，开始行动"
   →  进入下一个 Level
```

**标记法**：Vocation 2.1 = 使命象限处于 Level 2.0，Phase 1（接近必须升级的边缘）

**危险陷阱——虚假进化（False Transformation）**：
> 你觉得自己已经升级了，但其实只是在模仿那个层次的表面特征，没有真正完成 trait（知识、经验、技能）的积累。结果：卡在原地，直到真正解决当前层次的核心问题。

## 四、三大特质（Traits）

在每个相位中移动，需要积累三种特质：

| 特质 | 定义 | 获取方式 |
|------|------|----------|
| **Knowledge（知识）** | 理解原理和概念 | 阅读、研究、课程 |
| **Skill（技能）** | 实际做出来的能力 | 刻意练习、项目经验 |
| **Experience（经验）** | 在真实世界中的应用和反馈 | 实战、试错、迭代 |

> 常见悲剧模式：知识丰富但技能为零（"那个很胖的私人教练"）。知识必须转化为技能，技能必须通过经验验证。

## 五、加速通道（Channels）

当你进入某个象限的 Dissonance 期，你会获得进入一个"Channel"的能力——

**Channel 就是让你废寝忘食的"兔子洞"**。它的一端是知识，另一端是技能。通常的模式是：你开始痴迷于学习某个主题（知识），然后开始动手做（技能），时间飞速流逝但你完全不觉察。

| 象限 | Channel 典型表现 |
|------|-----------------|
| Mind | 深度冥想状态；或深夜追着一个思维线索，脑子里涌现大量想法 |
| Body | 痴迷于某种饮食/补剂/训练法，三个月内突飞猛进 |
| Spirit | 神秘体验；蜜月期的亲密关系；找到让你"豁然开朗"的哲学 |
| Vocation | 清晰的职业转折点或产品发布；为创业机会无法停止学习技能 |

**进入 Channel 的路径**：
1. 先达到当前 Level 的 Dissonance（对现状厌倦）
2. 用这种厌倦作为燃料，推入未知
3. 创建目标/愿景
4. 开始获取知识和技能
5. 足够多的实验后，Channel 会自动出现

## 六、高风险催化剂（Glitches）

Dan Koe 提到的"战术性捷径"——高回报但有代价：

| Glitch | 用途 | 风险 |
|--------|------|------|
| **AI** | 加速Vocational生产力，但无知识基础的人会被AI吞噬 | AI输出的内容缺乏核心判断力 |
| **Psychedelics** | 精神象限的快速突破 | 需要专业指导，方向错误则有长期负面后果 |
| **Strategic Pressure** | 创造商业截止日期等强制性能量 | 压力失控导致burnout |
| **极端节食/断食** | 快速身体状态改变 | 没有5年以上训练基础的人风险极高 |

**核心忠告**：先把你的"天然"潜力发展到最大，这样你才有足够的经验值，不会被这些高风险机制一击必杀。**知识和技能降低风险。**

## 七、类型系统（Archetypes & Metatypes）

- **Archetypes（原型）**：每个象限在不同层次出现的典型模式
- **Metatypes（元类型）**：你个人在四个象限的层次组合，形成独特的你

### 各象限原型简表

| 象限 | Level 1.0 | Level 2.0 | Level 3.0 |
|------|-----------|-----------|-----------|
| Mind | NPC（程序化思维）| Philosopher（哲学思考者）| Systems Thinker（系统思考者）|
| Body | Skinny Fat（瘦胖子）| Biohacker（生物黑客）| Integrated Mover（整合型运动者）|
| Spirit | True Believer（盲目信仰者）| Spiritual Shopper（灵性逛店者）| Bridge Builder（桥梁构建者）|
| Vocation | Clock Puncher（打卡族）| Hustler（拼搏者）| Value Creator（价值创造者）|

## 八、Superhuman 90（超级人类90天计划）

Dan Koe 推出的实操性项目——用90天重启你的四个象限：

### 三阶段执行框架

**Week 1-3：基础建设**
- Mind：识别限制性信念
- Body：建立运动基线
- Spirit：审查关系网络
- Vocation：明确贡献目标

**Week 4-8：技能获取**
- 每个象限掌握一个微技能
- 实施每日进度追踪
- 加入问责社群

**Week 9-12：整合阶段**
- 创建跨象限协同仪式
- 建立维护系统
- 规划下一个进化阶段

## 决策框架

### Human 3.0 决策流程

```
你面临一个挑战或决策
  │
  ▼
1. 这个挑战属于哪个象限？
   （Mind/Body/Spirit/Vocation）
   │
   ▼
2. 我在这个象限当前处于什么层次？
   （1.0 / 2.0 / 3.0）
   │
   ▼
3. 我在这个层次处于什么相位？
   （Dissonance / Uncertainty / Discovery）
   │
   ▼
4. 我需要积累哪些特质才能向上一级？
   （Knowledge / Skill / Experience）
   │
   ▼
5. 是否有 Channel 可以加速这个过程？
   │
   ▼
6. 如果要跨越多个层次——我是否在犯
   "虚假进化"的错误？（只模仿表面）
   │
   ▼
7. 做出决策并记录：下次类似情境的原则是什么？
```

## 反模式

### Dan Koe / Human 3.0 明确反对的做法

1. **❌ 单一维度极致发展** —— 职业成功但身体垮掉、灵性空虚，不是真正的进化
2. **❌ 跳过层次的进化** —— 2.0的人试图直接模仿3.0的行为，是虚假进化的典型
3. **❌ 积累知识但不实践** —— "那个很胖的私人教练"是最常见的悲剧
4. **❌ 把别人的价值观当作自己的** —— 每个象限的发展都必须真正解决你自己的问题
5. **❌ 用一个象限的逻辑套用到另一个** —— 职业成功逻辑不能解决灵性问题，健身计划不能解决意义危机
6. **❌ 把舒适误认为成长** —— 如果你不感到 dissonant，你可能只是在麻木，不是在进化
7. **❌ 用AI替代核心认知** —— AI可以加速执行，但无法替代你的意义判断（没有知识基础的人会被AI吞噬）

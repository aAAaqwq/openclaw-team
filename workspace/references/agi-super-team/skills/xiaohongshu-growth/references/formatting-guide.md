# 小红书排版规范

## 基本原则

1. **每段不超过3行** - 手机屏幕阅读优化
2. **多用空行** - 视觉呼吸感
3. **emoji点缀** - 不超过每行1个
4. **重点加粗/强调** - 关键信息突出

## 字数建议

| 内容类型 | 推荐字数 |
|----------|----------|
| 图文笔记 | 300-600字 |
| 干货教程 | 800-1500字 |
| 好物推荐 | 200-400字 |
| Vlog文案 | 100-300字 |

## 排版模板


## 禁忌事项

### ❌ 不要做
- 大段文字无空行
- 全篇emoji堆砌
- 标题党内容不符
- 过度营销感

### ✅ 要做到
- 结构清晰
- 重点突出
- 情感真实
- 价值明确

## 发布检查清单

- [ ] 标题吸引人且准确
- [ ] 开头3秒能留住用户
- [ ] 内容分段合理
- [ ] emoji使用得当
- [ ] 话题标签相关
- [ ] 封面图清晰美观
- [ ] 无敏感词/错别字

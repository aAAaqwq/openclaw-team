# 抖音发布流程与排版规范

**平台**: https://creator.douyin.com  
**调研日期**: 2026-03-17

---

## 1. 发布入口

- **网页端**: https://creator.douyin.com → 发布作品
- **支持类型**: 
  - 视频发布 (短视频)
  - 图文发布 (图文轮播)
  - 直播推流

---

## 2. 排版规则表

### 2.1 视频发布规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **视频时长** | 15秒 - 60分钟 | 超过15分钟需粉丝≥1000 |
| **视频格式** | MP4, MOV | 推荐MP4(H.264) |
| **分辨率** | ≥720×1280 | 推荐1080×1920 |
| **宽高比** | 9:16(竖屏) / 16:9(横屏) | 竖屏最优 |
| **文件大小** | ≤4GB | 单个视频 |
| **帧率** | ≥30fps | 推荐30fps |
| **码率** | 推荐2500-5000kbps | - |

### 2.2 图文发布规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **图片数量** | 2-35张 | 至少2张 |
| **图片格式** | JPG, PNG, WebP | JPG优先 |
| **图片尺寸** | 推荐1080×1440(3:4) | 竖版最优 |
| **单张大小** | ≤20MB | - |
| **总大小** | 无硬性限制 | 建议<200MB |

### 2.3 标题/描述规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **标题字数** | 建议≤30字 | 无硬性上限，但显示截断 |
| **描述总字数** | 建议≤200字 | 含话题标签 |
| **话题标签** | #话题名 | 格式同小红书 |
| **话题数量** | 建议3-5个 | 超过会被限流 |
| **@功能** | @用户名 | @相关账号增加互动 |
| **位置** | 可添加POI定位 | 本地内容推荐 |
| **emoji** | ✅ 适度使用 | 每2-3句1个 |

### 2.4 内容标识要求(2023.9起)

| 内容类型 | 标识要求 | 说明 |
|----------|----------|------|
| **AIGC内容** | "内容由AI生成" | AI生成内容必须标注 |
| **虚构情节** | "情景演绎，仅供娱乐" | 非真实场景 |
| **专业行为** | 声明专业身份 | 医疗/法律/金融 |
| **引用内容** | 标注拍摄时间/地点 | 引用他人内容标注来源 |

## 3. 发布步骤

### 3.1 视频发布流程

```
1. 打开 https://creator.douyin.com
2. 点击 [发布作品] / 上传区域
3. 上传视频文件:
   - 拖拽或点击上传
   - 支持 MP4, MOV
   - 等待上传+转码
4. 视频编辑:
   - 裁剪视频片段
   - 添加滤镜/特效(可选)
   - 选择封面帧:
     a. 从视频帧选取
     b. 自定义上传封面(1080×1920)
5. 填写描述:
   - 文字描述(≤200字建议)
   - 添加 #话题标签
   - @相关账号
6. 发布设置:
   - [可见性]: 公开/好友可见/仅自己
   - [允许合拍/下载]: 开关
   - [定时发布]: 选择时间
   - [位置]: 添加POI
7. [发布] → 立即发布
   [草稿] → 保存草稿
```

### 3.2 图文发布流程

```
1. 打开 https://creator.douyin.com
2. 选择 [发布图文] / 切换到图片上传模式
3. 上传图片(2-35张):
   - 拖拽或点击上传
   - 支持拖拽排序
4. 为每张图片添加描述(可选):
   - 每张图可单独配文
5. 填写整体标题/描述:
   - 文字描述
   - 添加 #话题标签
   - @相关账号
6. 选择封面:
   - 从上传图片中选择
   - 或自定义上传
7. 发布设置 → [发布]
```

### 3.3 话题标签使用

```
格式: #话题名
规则:
- 无空格: #AI工具推荐 ✅ / #AI 工具推荐 ❌
- 大小写不敏感
- 建议3-5个话题
- 前1-2个用精准话题，后2-3个用泛话题
- 热门话题加推荐权重
- 避免与内容无关的话题(会被判定标题党)
```

## 5. 注意事项

### 5.1 发布限制

| 限制项 | 规则 |
|--------|------|
| **日发布上限** | 无硬性限制，但过多会被降权 |
| **频率建议** | 每天1-3条 | 
| **长视频** | ≥15分钟需粉丝≥1000 |
| **修改** | 发布后可编辑描述，不可更换视频 |
| **删除** | 支持删除已发布视频 |
| **定时** | 支持定时发布，最远可设置10天后 |

### 5.2 发布最佳时间

| 时间段 | 推荐度 | 说明 |
|--------|--------|------|
| 7:00-9:00 | ⭐⭐⭐ | 早通勤 |
| 11:30-13:00 | ⭐⭐⭐ | 午休 |
| 17:30-19:00 | ⭐⭐⭐ | 下班通勤 |
| 20:00-22:00 | ⭐⭐⭐⭐⭐ | 晚间黄金档 |
| 22:00-00:00 | ⭐⭐⭐⭐ | 睡前刷手机 |

### 5.3 算法要点

- 发布后2小时内"赛马机制"
- 完播率 > 点赞率 > 评论率 > 转发率
- 前3秒决定用户是否继续观看
- 话题标签影响推荐池分配

### 5.4 内容规范

- 禁止搬运、水印、低质内容
- AIGC内容必须标注
- 医疗/法律/金融需专业声明
- 严禁违法违规、虚假宣传

## 6. 自动化要点

- **框架**: React SPA
- **登录要求**: 必须登录(扫码)
- **反爬**: 滑块验证、频率限制、行为检测
- **视频上传**: 分片上传，需签名
- **图文上传**: 图片先上传获取URL
- **API**: 无公开API，需模拟操作
- **草稿**: 支持保存草稿，但无公开API

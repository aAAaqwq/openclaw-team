# 抖音创作者内容风格指南

> 专为 AI/科技类短视频定制
> 核心原则：前3秒定生死

---

## 一、短视频标题公式

### 痛点型
`{人群}必看！{数字}秒解决{痛点}`
- "打工人必看！60秒学会AI写周报"

### 好奇型
`你知道{惊人事实}吗？`
- "你知道AI已经能自动做PPT了吗？"

### 对比型
`{A} VS {B}，差距居然这么大！`
- "人工写文案 VS AI写文案，差距居然这么大！"

### 干货型
`{数字}个{工具/技巧}，建议收藏`
- "5个AI神器，建议收藏（第3个太强了）"

---

## 二、脚本结构（通用模板）

```
【Hook - 0-3秒】生死3秒
{强情绪/强反差/强悬念的1句话}
→ 必须：制造好奇心或共鸣
→ 禁止："大家好我是xxx"式的自我介绍

【背景 - 3-5秒】快速切入
{为什么这个内容跟你有关}
→ 1句话讲清"跟你有什么关系"

【核心内容 - 5-40秒】信息密度拉满
{方法/工具/知识，拆成2-3个点}
→ 每个点1句话说明 + 1个画面展示
→ 节奏：每5秒换一个信息点

【高潮 - 40-50秒】价值峰值
{最精彩的部分放这里}
→ 最强的工具/效果/对比放最后

【CTA - 50-60秒】引导行动
{关注/收藏/评论/下期预告}
→ 具体指令："长按点赞=收藏"
```

---

## 三、视频规范

| 要素 | 短视频(≤1min) | 中视频(1-3min) |
|------|-------------|--------------|
| 时长 | 30-60秒最佳 | 1.5-2.5分钟 |
| 画面比例 | 9:16 竖屏 | 9:16 竖屏 |
| 分辨率 | 1080×1920 | 1080×1920 |
| 语速 | 220-260字/分钟 | 200-240字/分钟 |
| BGM | 选热门BGM(蹭流量) | 无/轻音乐 |
| 字幕 | 必须有，大字 | 必须有 |
| 封面 | 大字+人物/工具截图 | 同左 |

---

## 四、完播率优化策略

1. **前3秒**：直接上干货或制造悬念，不做自我介绍
2. **每5秒**：切换画面/信息点，保持视觉刺激
3. **中段**：节奏加快，信息密度拉满
4. **后段**：放"最强的一个"做高潮
5. **结尾**：预告下期 → 引导关注

---

## 五、标签策略

每条视频添加 3-5 个标签：
- 1个精准标签：`#AI工具` `#ChatGPT`
- 1个场景标签：`#效率神器` `#打工人必备`
- 1个流量标签：`#上热门` `#涨知识`
- 1个话题标签：`#AI时代` `#科技改变生活`

## 七、图文视频（图文轮播）格式

如果用图文而非视频：

| 要素 | 规范 |
|------|------|
| 图片数 | 6-9张 |
| 比例 | 3:4 (1080×1440) |
| 首图 | 大字标题，3秒可读 |
| 内容图 | 每张1个核心点，文字≤30字 |
| 末图 | 引导关注/互动 |

## 八、违禁词避雷

与小红书类似，额外注意：
- 不能有其他平台水印
- 不能引导站外交易
- AI生成内容需标注（抖音2026新规）
- 不能夸大效果（"月入10万"等）

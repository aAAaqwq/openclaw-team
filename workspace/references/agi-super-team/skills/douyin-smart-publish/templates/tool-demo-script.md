# 抖音：AI工具展示短视频脚本模板

> 适用：工具推荐、效果展示、效率提升
> 时长：30-60秒

---

## 脚本结构

| 时间段 | 内容 | 画面 |
|--------|------|------|
| 0-3s | **Hook**（生死3秒） | 工具最终效果+大字 |
| 3-8s | 痛点切入 | 对比画面(BEFORE) |
| 8-20s | 工具展示1 | 操作录屏+字幕 |
| 20-35s | 工具展示2 | 操作录屏+字幕 |
| 35-45s | 效果展示 | 结果画面(AFTER) |
| 45-55s | 总结推荐 | 工具列表截图 |
| 55-60s | CTA引导 | 点赞关注提示 |

## 脚本示例

```
[0-3s] 你还在{手动操作}? {数字}秒解决👇

[3-8s] 以前我要花{时间}做{事情}
       现在用这个AI工具，{时间}搞定

[8-20s] 第一步打开{工具名}
        输入{指令}
        看，{效果}

[20-35s] 第二步{操作}
         还有这个功能：{亮点}
         {数据/效果}

[35-45s] 最终效果就是这样的
         {效果截图/录屏}

[45-55s] 总结一下，{数字}个AI效率神器：
         1️⃣ {工具名} - {一句话}
         2️⃣ {工具名} - {一句话}
         3️⃣ {工具名} - {一句话}

[55-60s] 关注我，每天分享一个AI效率技巧
         长按点赞=收藏
```

## 制作要点
- 语速：220-260字/分钟
- 字幕：必须加，大字号，底部居中
- BGM：选热门BGM增加推荐概率
- 画面：工具录屏 + 关键步骤高亮

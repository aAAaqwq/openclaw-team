# Cron 时间表达式速查表

## Cron 表达式格式

```
分钟 小时 日 月 星期
 │    │   │  │   │
 │    │   │  │   └── 星期 (0-7, 0和7都是周日)
 │    │   │  └────── 月份 (1-12)
 │    │   └───────── 日期 (1-31)
 │    └────────────── 小时 (0-23)
 └─────────────────── 分钟 (0-59)
```

## 常用表达式

### 每天执行
| 需求 | 表达式 |
|------|--------|
| 每天 00:00 | `0 0 * * *` |
| 每天 09:00 | `0 9 * * *` |
| 每天 09:30 | `30 9 * * *` |
| 每天 18:00 | `0 18 * * *` |
| 每天 23:59 | `59 23 * * *` |

### 每天多次
| 需求 | 表达式 |
|------|--------|
| 每天 9点和18点 | `0 9,18 * * *` |
| 每天 9点、12点、18点 | `0 9,12,18 * * *` |
| 每天 9:30、11:30、14:30 | `30 9,11,14 * * *` |
| 每天 8点到18点每小时 | `0 8-18 * * *` |

### 工作日
| 需求 | 表达式 |
|------|--------|
| 工作日 09:00 | `0 9 * * 1-5` |
| 工作日 09:30 | `30 9 * * 1-5` |
| 工作日 9点和14点 | `0 9,14 * * 1-5` |

### 周末
| 需求 | 表达式 |
|------|--------|
| 周末 10:00 | `0 10 * * 0,6` |
| 周日 09:00 | `0 9 * * 0` |
| 周六 09:00 | `0 9 * * 6` |

### 高频执行
| 需求 | 表达式 |
|------|--------|
| 每小时整点 | `0 * * * *` |
| 每30分钟 | `*/30 * * * *` |
| 每15分钟 | `*/15 * * * *` |
| 每5分钟 | `*/5 * * * *` |
| 每分钟 | `* * * * *` |

### 特定时段
| 需求 | 表达式 |
|------|--------|
| 凌晨1-6点每15分钟 | `*/15 1-6 * * *` |
| 工作时间每小时 (9-18点) | `0 9-18 * * 1-5` |
| 夜间每小时 (22-6点) | `0 22-23,0-6 * * *` |

### 每周/每月
| 需求 | 表达式 |
|------|--------|
| 每周一 09:00 | `0 9 * * 1` |
| 每周五 18:00 | `0 18 * * 5` |
| 每月1号 09:00 | `0 9 1 * *` |
| 每月15号 09:00 | `0 9 15 * *` |
| 每月最后一天 | 不支持，用脚本判断 |

## 间隔调度 (everyMs)

| 需求 | 毫秒值 |
|------|--------|
| 每1分钟 | 60000 |
| 每5分钟 | 300000 |
| 每10分钟 | 600000 |
| 每15分钟 | 900000 |
| 每30分钟 | 1800000 |
| 每1小时 | 3600000 |
| 每2小时 | 7200000 |
| 每3小时 | 10800000 |
| 每6小时 | 21600000 |
| 每12小时 | 43200000 |
| 每24小时 | 86400000 |

## 一次性任务 (at)

```javascript
// 计算目标时间戳
const targetTime = new Date('2026-02-05T15:00:00+08:00');
const atMs = targetTime.getTime();

// 或者：N分钟后
const atMs = Date.now() + N * 60 * 1000;
```

## 时区

始终使用 `"tz": "Asia/Shanghai"` 确保按北京时间执行。

## 特殊字符

| 字符 | 含义 | 示例 |
|------|------|------|
| `*` | 任意值 | `* * * * *` 每分钟 |
| `,` | 列表 | `0 9,18 * * *` 9点和18点 |
| `-` | 范围 | `0 9-18 * * *` 9点到18点 |
| `/` | 步长 | `*/15 * * * *` 每15分钟 |

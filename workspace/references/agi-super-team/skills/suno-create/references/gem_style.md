# G.E.M. 邓紫棋 Style Prompt

## Description

Use this style prompt when user wants to generate a song in the style of G.E.M. (邓紫棋).

## Style Prompt

```
G.E.M.邓紫棋风格, 强大的 belting 声乐, 沙哑音色, R&B 流行, 宽广音域, 情感化中文演唱, 呼吸感副歌, 灵魂情歌, 2000s-2020s 华语天后气质, 怀旧浪漫
```

## Voice Characteristics

- **声线**: 强大的 belting 能力, 沙哑质感, 高音通透
- **风格**: R&B 流行, 灵魂乐元素
- **音域**: 宽广, 可以驾驭高低音转换
- **情感**: 深度情感化演唱, 呼吸感副歌
- **气质**: 华语天后级别, 2000s-2020s 黄金年代感

## Usage

Paste the above prompt into the Suno "Style" or "Custom" field.

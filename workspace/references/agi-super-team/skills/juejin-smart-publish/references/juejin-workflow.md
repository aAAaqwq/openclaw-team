## 2. 排版规则表

## 3. 发布步骤

### 3.1 完整发布流程

```
1. 打开 https://juejin.cn/editor/drafts/new
2. 编辑文章:
   a. 填写标题(15-35字建议)
   b. 使用Markdown编辑器编写正文
   c. 编辑器左侧Markdown输入，右侧实时预览
   d. 插入图片(拖拽/URL/剪贴板粘贴)
   e. 插入代码块(三个反引号+语言标识)
3. 文章设置(右侧面板):
   a. 选择文章分类(必选1个):
      - 后端 / 前端 / Android / iOS
      - 人工智能 / 开发工具 / 代码人生 / 阅读
   b. 添加标签(1-5个):
      - 搜索已有标签
      - 或输入新标签按Enter
   c. 上传封面图(可选):
      - 点击上传区域
      - 支持拖拽
   d. 填写摘要(可选):
      - 50-150字
      - 不填则取正文前N字
4. 发布选项:
   [发布文章] → 立即发布
   [保存草稿] → 存入草稿箱
```

### 3.2 编辑器快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + B` | **加粗** |
| `Ctrl/Cmd + H` | # 标题 |
| `Ctrl/Cmd + K` | 插入链接 |
| `Ctrl/Cmd + Shift + C` | 插入代码块 |
| `Ctrl/Cmd + Shift + K` | 行内代码 |
| `Ctrl/Cmd + Shift + I` | 插入图片 |
| `Ctrl/Cmd + Shift + L` | 无序列表 |
| `Ctrl/Cmd + Z` | 撤销 |

### 3.3 Markdown语法支持

掘金编辑器支持完整GFM(GitHub Flavored Markdown)语法:

```markdown

## 二级标题
### 三级标题

**加粗** / *斜体* / ~~删除线~~

> 引用

- 无序列表
1. 有序列表
- [x] 任务列表

[链接](https://example.com)
![图片](https://example.com/img.png)

`行内代码`

```python
# 代码块(语法高亮)
print("hello juejin")
```

| 表头1 | 表头2 |
|-------|-------|
| 内容1 | 内容2 |
| :--- | ---: |  # 左对齐/右对齐

---

# 分割线
```

- 将文章归入专栏
- 专栏可设置付费(掘金小册)

## 5. 注意事项

### 5.1 发布限制

| 限制项 | 规则 |
|--------|------|
| **发布频率** | 无硬性限制，建议每天≤3篇 |
| **修改** | 发布后可编辑 |
| **删除** | 支持删除 |
| **草稿** | 无限保存 |
| **原创** | 建议声明原创，掘金有原创检测 |
| **审核** | 新用户可能有审核机制 |

### 5.2 SEO优化

- 标题含核心关键词
- 前言段落包含关键词(前100字)
- 标签选择精准+热门
- 摘要包含关键词
- 文章内链(引用自己的其他文章)
- 外链增加权威性

### 5.3 排版最佳实践

1. **代码块**: 始终指定语言标识(```python)
2. **图片**: 每个关键步骤配图
3. **表格**: 对比类内容使用表格
4. **目录**: 长文章使用目录(掘金自动生成)
5. **引用**: 重要信息用引用块
6. **加粗**: 关键术语加粗
7. **分割线**: 区分章节

### 5.4 掘金特色功能

- **沸点**: 简短动态(类似微博)
- **小册**: 付费专栏
- **活动**: 参与官方活动增加曝光
- **稀土掘金**: 技术社区互动

## 6. 自动化要点

- **框架**: Vue.js SPA
- **编辑器**: 自研Markdown编辑器
- **登录**: 手机号/GitHub/微信
- **API**: 掘金有开放API(需申请)

  - 草稿管理API
- **图床**: 掘金自带图床，上传即托管
- **反爬**: 有频率限制
- **Markdown**: 完整GFM支持，最佳自动化平台

# 抖音描述排版模板

> 填写描述时参考此模板，保持风格一致

## 图文描述模板

```
{核心信息句/痛点句} emoji

{1-2句补充说明/价值点}

{互动引导/CTA}

#精准话题1 #精准话题2 #场景话题 #泛话题
```

## 视频描述模板

```
{Hook句（前3秒法则）}

{核心内容/方法/工具}

{效果/对比/结果}

#精准话题1 #精准话题2 #场景话题 #泛话题
```

## 话题标签策略

| 位置 | 类型 | 数量 | 示例 |
|------|------|------|------|
| 前1-2 | 精准话题 | 1-2 | `#AI工具` `#ChatGPT` |
| 中间 | 场景话题 | 1-2 | `#效率神器` `#打工人必备` |
| 最后 | 泛话题 | 0-1 | `#AI时代` `#科技改变生活` |

## 字数控制

| 项 | 上限 | 建议 |
|----|------|------|
| 标题 | 55字 | 20-35字 |
| 描述 | 200字 | 100-150字 |
| 话题数 | 5个 | 3-5个 |

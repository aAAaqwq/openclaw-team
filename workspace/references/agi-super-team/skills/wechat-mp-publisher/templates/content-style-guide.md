# 微信公众号内容风格指南

> 专为 AI/科技类长文定制
> 特点：深度、结构化、可收藏

---

## 一、标题公式

### 深度分析类
`{领域}深度分析：{核心结论}，{数字}个关键发现`
- "AI Agent深度分析：OpenClaw VS LangGraph，5个关键差异"

### 教程实战类
`从0到1：{具体成果}，{人群}也能学会的完整指南`
- "从0到1：用AI Agent自动化你的工作流，非程序员也能学会"

### 工具测评类
`{数字}款{工具类}横评：谁才是{场景}的最佳选择？`
- "6款AI写作工具横评：谁才是中文创作者的最佳选择？"

### 趋势洞察类
`{时间}{趋势}：{人群}必看的{领域}变化`
- "2026年3月AI大模型格局巨变：从业者必看的5个趋势"

### 痛点解决类
`为什么{痛点}？我用{方法}解决了，附完整方案`
- "为什么你的AI总写不出好文案？我用Prompt Engineering解决了，附模板"

## 二、文章结构（通用模板）

```markdown
# {标题}

> {副标题/一句话概括核心价值}

## 写在前面
{为什么写这篇文章，读者能获得什么，阅读时间}

## 一、{背景/痛点引入}
{2-3段描述现状、问题、为什么重要}

## 二、{核心内容/解决方案}
### 2.1 {子主题1}
{具体内容}

### 2.2 {子主题2}
{具体内容}

### 2.3 {子主题3}
{具体内容}

## 三、{实战/案例/对比}
{具体案例、操作步骤、数据对比}

## 四、{总结与展望}
{核心结论 + 未来趋势}

---
👋 **我是Daniel**，专注AI效率工具分享
觉得有用？点个「在看」让更多人看到
💬 有问题？评论区见，有问必答
🔄 转发给需要的朋友
```

## 三、排版规范

| 要素 | 规范 |
|------|------|
| 标题层级 | H1(1个) → H2(3-5个) → H3(按需) |
| 段落长度 | 每段2-4句，不超过100字 |
| 图片 | 每300-500字配一张，宽度677px |
| 代码块 | 深色背景，等宽字体，附注释 |
| 重点标注 | **加粗**核心结论，不滥用 |
| 引用块 | 用于金句、数据、重要提示 |

---

## 四、配图规范

| 类型 | 规格 | 要求 |
|------|------|------|
| 题图 | 900×383px (2.35:1) | 无文字/少文字，吸引点击 |
| 正文图 | 宽度677px | 截图需标注重点区域 |
| 二维码 | 200×200px | 文末引导关注 |
| 封底图 | 同题图 | 品牌一致 |

## 六、SEO 优化

- 标题含核心关键词（前15字最重要）
- 副标题含长尾关键词
- 文中自然出现关键词3-5次
| 
| 标签 | 设置2-3个精准标签 |
| 摘要 | 含核心关键词+吸引点击 |

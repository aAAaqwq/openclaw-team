# 微信公众号发布流程与排版规范

**平台**: https://mp.weixin.qq.com  
**调研日期**: 2026-03-17

---

## 1. 发布入口

- **网页端**: https://mp.weixin.qq.com → 内容与互动 → 群发
- **编辑器**: 富文本编辑器(自有) / Markdown导入(第三方工具)
- **发布方式**: 群发(推送给所有粉丝) / 发布(不推送，可分享)

---

## 2. 排版规则表

### 2.1 标题规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **字数上限** | 64字 | 含标点符号 |
| **建议字数** | 13-22字 | 超出13字在封面图上方换行遮挡 |
| **摘要位置** | 独立显示 | 标题下方灰色小字 |
| **emoji** | ⚠️ 少用 | 部分客户端显示异常 |

### 2.2 正文规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **字数上限** | 20,000字(2万字) | 实际建议3,000-5,000字 |
| **建议字数** | 1,500-3,000字 | 阅读体验最优 |
| **图片数量** | 无硬性限制 | 建议每300-500字配一张图 |
| **排版工具** | 官方编辑器 / 135编辑器 / 秀米 / mdnice | Markdown→微信推荐mdnice |
| **代码块** | ⚠️ 不原生支持 | 需截图或第三方工具 |

### 2.3 摘要规范

| 规则项 | 限制 | 备注 |
|--------|------|------|
| **字数上限** | 120字 | 分享到朋友圈/群聊时显示 |
| **不填** | 自动取正文前54字 | 建议手动填写 |

### 2.4 封面图规范

| 规则项 | 尺寸 | 备注 |
|--------|------|------|
| **头条封面(大图)** | 900×500px (1.8:1) | 正方形裁剪显示为900×383 |
| **次条封面(小图)** | 200×200px (1:1) | 缩略图 |
| **分享封面** | 自动裁剪为正方形 | 核心元素放居中 |
| **格式** | JPG/PNG | < 2MB |
| **重要提示** | 核心内容放正中 | 朋友圈分享自动裁两边 |

### 2.5 排版详细参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| **正文字号** | 14-16px | 15px最优 |
| **标题字号** | 18-22px | 二级标题 |
| **行间距** | 1.5-1.75倍 | 推荐1.75 |
| **段间距** | 10-15px | 段落间留白 |
| **字间距** | 1-2px | 提升阅读感 |
| **页边距** | 15-20px | 两端留白 |
| **文字颜色** | #3f3f3f | 主文字颜色(非纯黑) |
| **引用颜色** | #8c8c8c | 引用文字 |
| **强调色** | 品牌色或#07c160 | 微信绿 |

## 3. 发布步骤

### 3.1 完整发布流程

```
1. 登录 mp.weixin.qq.com
2. 进入 [内容与互动] → [图文消息] → [新建图文]
3. 编辑器操作:
   a. 填写标题(≤64字)
   b. 填写作者(可选)
   c. 上传封面图:
      - 首条大图: 900×500px
      - 次条小图: 200×200px
   d. 填写摘要(≤120字)
   e. 编辑正文:
      - 使用富文本编辑器
      - 或从第三方工具(135编辑器/mdnice)复制粘贴
   f. 插入图片/视频/音频
   g. 添加原文链接(可选)
4. 原创设置:
   - [声明原创] → 填写原创声明
   - 可设置赞赏、留言、合集
5. 预览:
   - [预览] → 发送到手机预览
6. 发布:
   - [群发] → 推送给所有粉丝(每天1次)
   - [保存并群发] → 定时群发
   - [保存] → 保存为草稿
```

### 3.2 编辑器功能

| 功能 | 操作方式 | 快捷键 |
|------|----------|--------|
| 加粗 | 工具栏 B 按钮 | Ctrl+B |
| 斜体 | 工具栏 I 按钮 | Ctrl+I |
| 引用 | 工具栏 > 按钮 | - |
| 有序/无序列表 | 工具栏列表按钮 | - |
| 插入图片 | 点击图片图标 / 粘贴 | - |
| 插入链接 | 点击链接图标 | Ctrl+K |
| 分割线 | 工具栏分割线按钮 | - |
| 字体颜色 | 工具栏颜色选择 | - |
| 字号 | 工具栏字号下拉 | - |
| 对齐 | 左/中/右对齐 | - |
| 缩进 | 增加缩进/减少缩进 | - |

### 3.3 草稿管理

- 草稿位置: [内容与互动] → [草稿箱]
- 草稿无有效期限制
- 支持编辑、复制、删除
- 可从草稿直接群发

## 5. 注意事项

### 5.1 发布限制

| 限制项 | 规则 |
|--------|------|
| **群发频率** | 订阅号每天1次，服务号每月4次 |
| **发布无限制** | "发布"功能不限次数(不推送粉丝) |
| **修改限制** | 群发后**不可修改**标题/正文 |
| **删除** | 群发后可删除，但已读用户仍可见 |
| **字数** | 标题≤64字，正文≤20,000字 |

### 5.2 排版最佳实践

1. **字色用 #3f3f3f** 而非纯黑 #000000
2. **行间距 1.75倍** 阅读最舒适
3. **段落间空一行** 增加呼吸感
4. **配图宽度 640px** 适合手机阅读
5. **引用块** 用于重要信息强调
6. **分割线** 区分章节，提升结构感
7. **小标题** 统一字号+颜色，建立视觉层级

### 5.3 常见问题

- **图片模糊**: 确保原图≥640px宽，2MB以内
- **排版错乱**: 避免从Word直接粘贴，先粘贴为纯文本
- **emoji异常**: 微信对emoji支持有限，谨慎使用
- **链接不可点**: 正文中的URL需要手动添加超链接

---

## 6. 自动化要点

- 编辑器类型: **UEditor** (百度)
- 登录要求: **必须登录** (扫码)
- 反爬: 验证码 + IP限制
- API: 公众平台API可群发，但需认证服务号
- Markdown转微信: 使用 **mdnice** (https://mdnice.com) 或 **Mdnice**
- 草稿API: POST /cgi-bin/draft/add 需access_token

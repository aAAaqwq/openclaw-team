# 平台排版模板库

这份 reference 提供可直接复用的中文封面版式骨架。

## 小红书（3:4）

### 模板 XHS-01：左上双行冲击型
适用：反常识 / 观点 / 避坑 / 批判

结构：
- 左上角小标签：1 行（可选）
- 主标题：2 行
- 关键词：1-2 个高对比强调
- 主体画面：右侧或下方

排版 spec：
- 对齐：左对齐
- 主标题字重：Heavy
- 小标签字重：Medium
- 行距：偏紧
- 安全边距：四周保留明显呼吸区

示例：
- 标签：`AI创业`
- 第1行：`90% AI Agent`
- 第2行：`是噱头`

适合原因：
- 第一眼就能看到数字 + 赛道词
- 第二眼接判断词，形成冲突

### 模板 XHS-02：中部大标题 + 角标
适用：实测 / 榜单 / 工具合集

结构：
- 主标题：居左或居中 2 行
- 角标：右上/左上 `实测` / `2026` / `避坑`
- 背景：相对干净

示例：
- 第1行：`6个AI编程工具`
- 第2行：`实测说真话`
- 角标：`亲测`

排版重点：
- “6个”可放大 1.15-1.25 倍
- `AI编程工具` 保持完整，不拆专有名词

### 模板 XHS-03：对象 + 判断短句型
适用：短标题、极简结论

结构：
- 主标题：1-2 行
- 不放副标题
- 只保留一个标签或不放

示例：
- `别再神化`
- `AI Agent`

注意：
- 这种模板适合字少但判断强的内容
- 背景必须给文字让位

## 公众号首图（16:9）

### 模板 MP-01：编辑型稳重标题
适用：文章首图 / 深度分析 / 行业观察

结构：
- 栏目标签：左上或顶部一行
- 主标题：1-2 行
- 小副标题：1 行（解释文章角度）

排版 spec：
- 对齐：左对齐
- 字重：主标题 Bold，副标题 Regular/Medium
- 行距：正常偏松
- 配色：克制，不要短视频封面感

示例：
- 标签：`AI内容观察`
- 主标题：`6个AI编程工具实测`
- 副标题：`聊聊效率、幻觉与真实边界`

### 模板 MP-02：单行标题 + 小说明
适用：专业感强、结论明确的文章

结构：
- 单行主标题
- 底部或上方一行小说明

示例：
- 主标题：`AI Agent，离落地还很远`
- 说明：`一次真实工作流复盘`

注意：
- 如果标题超过 16-18 字，优先改两行，不要硬塞一行

## 抖音图文封面（9:16）

### 模板 DY-01：超大标题 + 小副标
适用：快节奏结论型内容

结构：
- 主标题：1-2 行，字号非常大
- 小副标：一行放在上方或下方
- 重点词高对比

示例：
- 小副标：`做AI别踩这个坑`
- 第1行：`AI Agent`
- 第2行：`别再吹了`

排版重点：
- 判断要短
- 画面上要先见字，再见图

### 模板 DY-02：数字开头型
适用：教程 / 清单 / 经验

示例：
- 第1行：`3个动作`
- 第2行：`优化封面点击率`

规则：
- 数字一定放前面
- 第二行补结果，不补过程

## 知识星球 / 方图（1:1）

### 模板 ZSXQ-01：中心块状信息卡
适用：知识卡、金句卡、观点卡

结构：
- 居中或偏左标题块
- 一行标签
- 一行署名/栏目名（可选）

示例：
- 标签：`内容系统`
- 主标题：`先排主次`
- 次标题：`再谈字体`

## 掘金（16:9）

### 模板 JJ-01：技术编辑风
适用：技术文章封面、工具评测

结构：
- 主标题：1-2 行
- 副标题：一句解释范围
- 标签：技术名词 / 年份 / 场景

示例：
- 主标题：`6个AI编程工具实测`
- 副标题：`从速度、准确率到实际可用性`
- 标签：`2026 / Tooling / Review`

注意：
- 不要做成营销海报
- 技术内容封面最怕浮夸

## 快速选模板规则

如果内容是：
- **反常识/批判/争议** → XHS-01 / DY-01
- **实测/榜单/对比** → XHS-02 / JJ-01 / MP-01
- **深度分析/行业观察** → MP-01 / MP-02
- **方法论/教程** → DY-02 / MP-01
- **知识卡/观点短句** → XHS-03 / ZSXQ-01

## 输出给设计师/Agent 的最终模板

```markdown
平台：小红书 3:4
模板：XHS-01 左上双行冲击型

标题压缩：90% AI Agent是噱头
分行：
- 第1行：90% AI Agent
- 第2行：是噱头

层级：
- 最大：90%
- 次级：AI Agent
- 强调：噱头

对齐：左对齐
字重：主标题 Heavy，标签 Medium
强调方式：仅换色 + 局部放大
注意：不要描边、不要多重阴影、不要再加多余副标题
```

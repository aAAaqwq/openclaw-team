# 中文内容封面排版参考

这份 reference 给更细的执行规则，用于把一句内容标题变成可落地的封面排版 spec。

## 1. 标题压缩模板

### 模板 A：数字 + 结论
- `6个AI编程工具实测`
- `3种起号方式对比`
- `90% AI Agent是噱头`

适合：测评、复盘、批判型内容

### 模板 B：对象 + 判断
- `AI Agent，大多是包装`
- `小红书，开始封AI代发`
- `大学生做AI副业，别走弯路`

适合：观点、评论、趋势判断

### 模板 C：动作 + 结果
- `这样写封面，点击率更高`
- `把标题砍短，质感立刻上来`
- `先做分行，再谈风格`

适合：教程、方法论、干货

## 2. 分行模板

### 两行强冲击模板
适用：小红书 / 抖音

例 1：
- 第1行：`90% AI Agent`
- 第2行：`是噱头`

例 2：
- 第1行：`6个AI编程工具`
- 第2行：`实测说真话`

原则：
- 第一行放赛道词 / 数字
- 第二行放判断 / 结果

### 一长一短模板
适用：编辑型 / 公众号

例：
- 第1行：`6个AI编程工具实测`
- 第2行：`一些不太好听的大实话`

原则：
- 第一行是主题
- 第二行是语气和补充

### 主标题 + 标签模板
适用：封面信息少但要建立主题感

例：
- 标签：`AI内容创业`
- 主标题第1行：`别再神化`
- 主标题第2行：`AI Agent`

## 3. 字体使用建议

## 中文无衬线优先
优先选择：
- 思源黑体
- 阿里巴巴普惠体
- HarmonyOS Sans
- 苹方
- 优设标题黑（更有标题感）

### 选型原则
- 看“识别速度”先于“风格感”
- 标题型封面优先粗黑、几何、结构稳定的字体
- 内容型、编辑型封面优先中性专业字

## 4. 配色建议

### 深底亮字
最稳：
- 深灰 / 黑底 + 白字
- 深蓝底 + 白字
- 黑底 + 白字 + 黄重点词

适用：科技、商业、趋势、工具类

### 亮底深字
适用：教程、生活方式、知识卡片
- 米白底 + 黑字
- 浅灰底 + 深灰字
- 奶白底 + 深绿/深蓝重点词

### 重点色只留一个主色
推荐主重点色：
- 黄：利益点、结果、警示
- 红：冲突、否定、封禁、争议
- 蓝：理性、科技、工具
- 绿：增长、效率、收益

## 5. 常见错误

### 错误 1：所有词都重要
表现：
- 每行都大
- 每个词都高亮
- 结果没有焦点

修正：
- 只保留一个主判断
- 只选 1-3 个词强调

### 错误 2：标题太像正文
表现：
- 字多
- 语序拖
- 有太多解释性虚词

修正：
- 删掉“为什么 / 其实 / 之后 / 这件事 / 一下 / 几句”等废字

### 错误 3：中文拉字距
表现：
- 像 PPT 初学者作品
- 读起来断气

修正：
- 中文标题默认字距正常或略紧

### 错误 4：背景很吵，标题也很吵
表现：
- 花背景 + 多色标题 + 重阴影

修正：
- 背景和文字只能有一个唱主角

## 6. 给设计师 / 出图 agent 的排版 spec 模板

```markdown
平台：小红书封面 3:4
类型：冲击型

主标题：90% AI Agent是噱头
分行：
- 第1行：90% AI Agent
- 第2行：是噱头

层级：
- 最大：90%
- 次级：AI Agent
- 判断词：噱头（高对比强调）

字体：
- 主标题：无衬线粗黑体，Heavy
- 对齐：左对齐
- 行距：偏紧
- 字距：默认或略紧

强调：
- “90%” 放大
- “噱头” 用高对比色或底色条
- 全图只保留 1 个重点色

留白：
- 标题块距边缘保留安全边距
- 不遮挡主体脸部/核心物体
```

## 7. 适合直接复用的排版判断句

当需要快速给出建议时，可直接用这些句子：

- `这版的问题不是字体不对，是没有主次。先把一个词打出来。`
- `别急着加特效，先把标题砍短再分行。`
- `中文封面先默认左对齐，稳定很多。`
- `这一版字太平均了，没有“第一眼先看哪”。`
- `重点词别超过3个，不然等于没有重点。`
- `背景已经很满了，文字就别再花。`

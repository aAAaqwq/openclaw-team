## 产品概述

**字流** (ziliu.online) 是AI驱动的多平台内容分发工具。
- 核心理念：「让文字如流水般顺畅流向每个平台」
- 一次创作 → AI智能适配 → 一键分发到16+平台
- 减少80%重复排版，发布效率提升7倍

## 核心工作流

```
创作/导入 → 智能适配 → 一键分发 → 数据回收
```

### STEP 1: 创作 & 导入
- 在字流编辑器直接写作（Markdown WYSIWYG）
- 或从飞书文档导入
- 支持 Markdown/Word 文档导入
- AI 自动提取结构与素材

### STEP 2: 智能适配
- AI 调整标题、摘要、封面与格式
- 自动生成多平台最佳版本
- 智能识别段落、标题、引用与代码块
- 保持品牌一致性

### STEP 3: 一键分发
- 通过浏览器插件注入到各平台编辑器
- 批量定时发布
- 跨平台快速登录与推送

### STEP 4: 数据回收
- 实时查看发布结果
- 回流评论与数据
- 优化下一篇内容

## 浏览器插件

- **支持**: Chrome / Edge
- **版本**: v2.0.24
- **大小**: ~50KB
- **权限**: 只读，安全可控
- **下载**: https://www.ziliu.online/extension
- **安装**: 手动加载（开发者模式 → 加载已解压的扩展程序）

### 插件安装步骤
1. 从 https://www.ziliu.online/extension 下载 zip 包
2. 解压到任意文件夹
3. Chrome → `chrome://extensions` → 开启开发者模式
4. 点击「加载已解压的扩展程序」→ 选择解压目录
5. 回到字流页面点击「重新检测」确认

### 插件使用方式
1. 在字流编辑器完成文章创作和适配
2. 打开目标平台（如小红书创作中心）的编辑器
3. 点击字流插件 → 一键填充
4. 自动带格式填充到目标平台编辑器

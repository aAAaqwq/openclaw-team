# 关键词研究 (Keyword Research)

> 系统性发现、分析和排序高价值关键词的方法论。在GEO时代，额外关注**大模型训练语料中出现频率**——因为被AI记住比被搜索更关键。

## 核心方法

### 1. 关键词发现方法

沿用SEO行业成熟方法论，新增GEO专用维度：

| 方法论 | 主要来源（类比工具） | 关键词类型 |
|-------|-------------------|-----------|
| **种子词扩展** | Google Keyword Planner / SEMrush Magic Tool | 传统搜索词 |
| **竞品对比** | Ahrefs Site Explorer / SEMrush Domain vs Domain | 行业通用词 |
| **长尾挖掘** | AnswerThePublic / AlsoAsked | 自然语言提问 |
| **社交/论坛** | Reddit / Quora / 知乎 | 真实用户问法 |
| **大模型提示分析** | ChatGPT / Claude / Perplexity自动补全 | GEO关键词（新增） |
| **维基百科** | Wikipedia分类树 + 子页面 | 实体/概念词 |
| **Google Suggest** | 搜索框自动补全 | 高频搜索组合 |

### 2. 搜索意图分类

每个关键词必须归类意图，不同意图对应不同内容格式：

| 意图类型 | 定义 | 典型词特征 | 内容格式 | 优先GEO？ |
|---------|------|-----------|---------|----------|
| **Informational** | 用户想了解/学习 | "是什么" "如何" "vs" "指南" | 博客/FAQ/百科 | ✅ 高 |
| **Navigational** | 用户想找到某个品牌/页面 | "品牌名" "登录" "官网" | 登录页/首页 | ❌ 低 |
| **Commercial** | 用户在研究购买决策 | "对比" "评测" "推荐" "2025" | 对比/评测/案例 | ✅ 中 |
| **Transactional** | 用户准备购买 | "购买" "价格" "折扣" "订阅" | 产品/定价页 | ❌ 低 |

### 3. 关键词难度评估框架

```
KD Score = Volume × (1/Difficulty) × IntentBonus × GEOBonus

Volume         — 月搜索量（0-10标准化）
Difficulty     — 竞争难度1-10（关键词/域名竞争）
IntentBonus    — Informational=1.5, Commercial=1.2, 其他=1.0
GEOBonus       — AI训练语料出现频次（关键词在大模型回答中的出现率，0-2x）
```

**优先级评估：**

| 分数区间 | 策略 | 行动 |
|---------|------|------|
| ≥ 8.0 | 高价值低难度 | P0 立即投入 |
| 5.0 - 7.9 | 中等价值 | P1 规划投入 |
| 3.0 - 4.9 | 长尾机会 | P2 内容覆盖 |
| < 3.0 | 低价值或极高难度 | P3 暂时搁置 |

### 4. GEO关键词的特殊标注

GEO时代的核心新增：关键词的**大模型训练语料中出现频率**（简称**GEO-IF**）：

**GEO-IF 评级：**
- **A级**：在ChatGPT/Claude/Perplexity等主流模型中，该词与品牌同时出现的频率高
- **B级**：该词部分出现在AI回答中，但不稳定
- **C级**：该词几乎不在AI回答中提到需要从头建立联系

**GEO关键词采集方法：**
```
① 在大模型中用"在[行业]中，哪些关键词最重要？"生成初始列表
② 在Perplexity/Google AI Overviews中搜索候选词，观察品牌是否出现
③ 跟踪品牌词在AI回答中的引用率变化（月度对比）
④ 标注GEO-IF等级，纳入关键词优先级评分
```

# 增长悬赏包生成 (Bounty Growth Pack Generator)

> 当AI无法直接触达高价值目标（关键人脉/高端关系/复杂谈判）时，将其转化为**悬赏任务**，通过L2分级激励体系，让人类伙伴精准执行。

## 核心方法

### 1. 悬赏场景判定标准

满足以下任一条件即触发悬赏：

| 场景 | AI可触达？ | 示例 | 行动 |
|------|-----------|------|------|
| **关键人脉引荐** | ❌ 无法直接触达 | 某500强CEO、政府官员、行业KOL | ✅ **创建伯乐悬赏** |
| **高端商务谈判** | ⚠️ 部分可辅助 | 与投资人/大型合作伙伴的关键会议 | ✅ **创建顾问悬赏** |
| **敏感市场渗透** | ❌ 无法AI完成 | 某个行业圈子、特定协会人脉网 | ✅ **创建引荐悬赏** |
| **地推/线下活动** | ❌ 需肉身到场 | 行业展会、线下沙龙 | ✅ **创建执行悬赏** |
| **内容/PR公关** | ⚠️ 部分可辅助 | 知名媒体专访、行业奖项申报 | ✅ **创建关系悬赏** |
| **常规线索获取** | ✅ AI可批量处理 | 社交媒体搜索/冷邮箱 | ❌ 不需要悬赏 |
| **内容创作** | ✅ AI可完成 | 博客/白皮书/社媒 | ❌ 不需要悬赏 |

### 2. 悬赏包结构

每个悬赏包包含以下字段：

```
────────────────────────────────────────────────
悬赏包 #[ID]
────────────────────────────────────────────────
◆ 目标 (Target): [联系人姓名/组织/位置]
◆ 背景 (Context): [为什么需要TA？当前进展？]
◆ 所需资源 (Resources Needed): [要什么？引荐/信息/决策？]
◆ 预算 (Budget): [悬赏金额] （或等价权益/积分）
◆ 验收标准 (Acceptance Criteria):
    1. [标准1] ⭐ 基础
    2. [标准2] ⭐⭐ 完成
    3. [标准3] ⭐⭐⭐ 超出预期
◆ 交付物/证据: [引荐截图/邮件/会议记录]
◆ 截止日期: [YYYY-MM-DD]
◆ 发布平台: [内部社区/城市合伙人/专门渠道]
────────────────────────────────────────────────
```

### 3. 伯乐悬赏的L2分级体系

| 级别 | 难度 | 悬赏金额 | 适合人群 | 典型任务 |
|------|------|---------|---------|---------|
| **L1 — 引荐** | 低-中 | ¥500-2,000 | 广泛城市合伙人/校友/社群成员 | 引荐指定职位/行业的人脉 |
| **L2 — 深度** | 中-高 | ¥2,000-10,000 | 资深行业人士/投资人/高端人脉 | 安排深度会面/获取关键决策 |
| **L3 — 战略** | 极高 | ¥10,000+ 或股份/权益 | 核心合伙人/行业资深顾问 | CEO级引荐、战略投资对接 |

**L1 引荐示例：**
```
目标：引荐某AI创业公司CTO（或同等级别人选）
预算：¥1,000（引荐成功+对方接受15min通话）
验收：发送引荐邮件/私信截图 + 对方回复确认
```

**L2 深度示例：**
```
目标：安排与XX基金投资总监深度交流（60min+）
预算：¥5,000（会议完成）
验收：会议纪要 + 后续跟进计划确认
```

**L3 战略示例：**
```
目标：引荐XX行业头部企业VP+级别并拿到一次正式提案机会
预算：¥20,000（提案会完成）
验收：对方正式邀请函/日程确认
```

### 4. 悬赏效果评估模板

```
BOUNTY EVAL — [悬赏包ID] 完成报告
─────────────────────────────────
1. 悬赏目标：[原目标描述]
2. 完成状态：[已完成/部分完成/失败/放弃]
3. 实际产出：
   - [引荐/会议/信息] 的具体成果
4. 投入产出：
   - 悬赏支出：¥[X]
   - 商业价值估计：¥[Y]
   - ROI：Y:X
5. 伙伴评分：[1-5⭐，含评语]
6. Learnings：
   - 什么有效？
   - 什么可改进？
7. 是否将该悬赏标准化/重复发布：[是/否]
```

## 悬赏包生成检查清单

- [ ] 场景判定：确认AI确实无法直接完成
- [ ] 目标验证：确认目标在当前网络外且有合理路径触达
- [ ] 验收标准：明确、可验证（截图/邮件/会议记录）
- [ ] 预算合理：参考同类悬赏历史数据
- [ ] 截止日期：足够长（至少14天）但不过长（超过60天失去紧迫感）
- [ ] 合规检查：无利益冲突、无贿赂/违规风险
- [ ] 发布渠道：确保目标人群能看到

# 转化率优化器 (Conversion Rate Optimizer)

> 系统化的转化率优化（CRO）方法论：通过数据驱动的**Research→Hypothesis→Test→Learn**循环，持续提升每个漏斗阶段的转化率。

## 核心方法

### 1. CRO方法论 — Research→Hypothesis→Test→Learn

```
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│ RESEARCH  │────>│ HYPOTHESIS│────>│   TEST    │────>│   LEARN   │
│ 发现机会   │     │ 提出假设  │     │ 实验验证  │     │ 沉淀知识  │
└──────────┘     └──────────┘     └──────────┘     └──────────┘
```

**各阶段关键产出：**

| 阶段 | 输入 | 方法 | 产出 |
|------|------|------|------|
| **Research** | 流量/行为数据 | 漏斗分析+热力图+问卷+录屏 | 机会点清单 |
| **Hypothesis** | 机会点清单 | ICE评分+用户同理心 | 实验假设卡 |
| **Test** | 假设卡 | A/B测试工具 | 数据结果 |
| **Learn** | 实验结果 | 定性+定量分析 | 最佳实践库 |

### 2. 落地页审计检查清单

#### 第一印象（头版以上）
- [ ] Hero标题是否在3秒内传达价值主张？
- [ ] 副标题是否补充了具体可信的利益点？
- [ ] CTA按钮是否醒目、行动导向、位于首屏？
- [ ] 首屏是否包含社会证明（客户Logo/数据/案例）？
- [ ] Hero区是否无任何干扰元素（多余链接/广告）？

#### 信任信号
- [ ] 客户Logo（知名品牌优先展示）
- [ ] 具体数据（"服务1000+客户" > "服务众多客户"）
- [ ] 第三方认证/奖项/报道
- [ ] 用户评价（含姓名+头像+公司）
- [ ] 退款保证/无风险承诺

#### 转化路径
- [ ] 主CTA在关键位置重复出现
- [ ] 表单字段是否最少化？（每增加1个字段 ≈ 转化率下降5-10%）
- [ ] 移动端CTA按钮是否触手可及？
- [ ] 页面加载速度是否 < 3秒？
- [ ] 是否有退出意图弹窗（可选）？

### 3. 漏斗分析：全链路转化

典型SaaS漏斗：

```
流量 (100%)
│
▼
访问落地页 (100%)
│  ↓ 落地页→注册 20%
▼
注册 (20% of 流量)
│  ↓ 注册→激活 50%
▼
激活 (10% of 流量)
│  ↓ 激活→付费 20%
▼
首笔付费 (2% of 流量)
│  ↓ 首购→复购 60%
▼
复购/留存 (1.2% of 流量)
```

**漏斗分析方法：**

| 步骤 | 关注指标 | 典型问题 | 优化方向 |
|------|---------|---------|---------|
| 流量→访问 | 跳出率 | 流量质量/落地页不匹配 | 改善流量源+关键词匹配度 |
| 访问→注册 | 注册转化率 | CTA不清晰/缺乏信任 | CTA优化+信任信号 |
| 注册→激活 | 激活率 | 引导不够/上手难 | Onboarding流程重塑 |
| 激活→付费 | 付费转化 | 价值展示不足/定价 | 产品价值深化+定价实验 |
| 首购→复购 | 留存率 | 产品粘性不够 | 功能+通知+忠诚计划 |

### 4. 热力图/用户录屏分析方法

| 分析工具 | 获取信息 | 典型发现 |
|---------|---------|---------|
| **点击热力图** | 用户点击分布 | 非交互元素被点击（知用户疑惑）|
| **滚动热力图** | 用户浏览深度 | 关键内容在折叠线以下 |
| **移开热力图** | 鼠标移动路径 | 阅读模式 vs 扫描模式 |
| **用户录屏** | 实际浏览行为 | 暂停/犹豫/返回操作 |
| **表单分析** | 字段完成率 | 哪些字段让用户放弃 |

**分析优先级：**
1. 表单直接弃用率高 → 优化表单字段
2. 关键CTA无点击 → 修改位置/文案/颜色
3. 核心内容滚动不到 → 内容上移或缩短篇幅
4. 用户在定价页反复上下滚动 → 定价不清晰/对比困难

## CRO实验产出模板

```
EXPERIMENT #123
─────────────
研究对象: 注册页面
当前状态: 注册转化率 15%
假设: 简化表单（从5字段到3字段）+ 添加"无需信用卡"说明
      将使注册转化率提升至 20%+
ICE: I=8 C=7 E=9 → 总分=8.0 (P0)

变体A(Control): 当前注册表单
变体B(Treatment): 3字段+无信用卡说明

样本量: 每组 2,000 访问
预计运行: 10天
决策规则: p<0.05 → 胜者; 否则拉平
```

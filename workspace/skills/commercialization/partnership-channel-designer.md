# 合作伙伴渠道设计器 (Partnership & Channel Designer)

> 系统化设计、招募、管理和优化合作伙伴渠道的方法论。覆盖从**集成/代理/联盟/白标/推荐**五种合作模式，到完整的渠道生命周期管理。

## 核心方法

### 1. 合作伙伴类型

| 类型 | 合作方式 | 适合产品 | 收入贡献潜力 | 管理复杂度 |
|------|---------|---------|------------|-----------|
| **集成合作伙伴** | API对接/技术集成 | 有开放API的平台型产品 | ⭐⭐⭐⭐ | 中 |
| **代理/经销商** | 代理销售拿佣金 | 中小企业市场产品 | ⭐⭐⭐⭐⭐ | 高 |
| **联盟/联属营销** | 追踪码佣金(CPA) | 低客单价数字产品 | ⭐⭐⭐ | 低 |
| **白标/OEM** | 沿用对方品牌 | 基础设施/API/报告类 | ⭐⭐⭐⭐ | 中 |
| **推荐合作伙伴** | 双向引荐 | 互补品（非竞品） | ⭐⭐ | 低 |
| **技术合伙人** | 共同开发/交付 | 深度定制行业解决方案 | ⭐⭐⭐⭐⭐ | 极高 |

### 2. 合作模式设计

#### 收益分成模型

| 合作伙伴类型 | 典型分成 | 支付方式 | 结算周期 |
|------------|---------|---------|---------|
| **代理** | 首年15-30% SaaS收入 | 月度/季度 | 首单+续约专属码 |
| **联盟/联属** | 10-25% 首单 | 月度（达到¥500门槛） | 追踪链接 |
| **推荐** | 5-10% 首单或固定¥500/单 | 月度 | 引荐码 |
| **集成** | 共同市场推广权益（无直接分成） | 无 | 联合营销 |
| **白标** | 批发价（通常零售价的40-60%） | 预付/月度 | 批发订单 |
| **技术合伙人** | 项目制（30-50%项目收入） | 里程碑付款 | 项目合同 |

#### 激励层级设计

```
L1（基础）：标准分成 + 基础支持 + 营销物料包
L2（银牌）：L1 + 优先支持 + 季度联合活动 + 额外5%分成
L3（金牌）：L2 + 专属客户经理 + 定制方案 + 年底返点

进阶条件：
  L1→L2：连续3个月贡献≥ ¥50K 收入
  L2→L3：连续6个月贡献≥ ¥200K 收入
```

### 3. 合作伙伴生命周期管理

```
                   ┌──────────┐
                   │ 招募/筛选 │   [月]发现
                   └────┬─────┘
                        ▼
                   ┌──────────┐
                   │ Onboard  │   [2-4周]培训+激活
                   └────┬─────┘
                        ▼
                   ┌──────────┐
                   │ Enable   │   [持续]赋能+支持
                   └────┬─────┘
                        ▼
                   ┌──────────┐
                   │ 赋能/增长 │   [月度]活动+激励
                   └────┬─────┘
                        ▼
                   ┌──────────┐
                   │ 评估/优化 │   [季度]评估+分级调整
                   └────┬─────┘
                        ▼
                        退出/流失 → 复盘
```

**各阶段关键动作：**

| 阶段 | 关键动作 | 频率 | 考核指标 |
|------|---------|------|---------|
| **招募** | 行业调研+目标清单+冷启动触达 | 持续 | 100+候选/月 |
| **Onboard** | 合作签约+产品培训+定价+流程说明 | 新伙伴时 | 30天内完成 |
| **Enable** | 支持工具（CRM/文档/模板）+定期1:1 | 月度 | 伙伴健康分 |
| **增长** | 联合活动+共同客户+升级激励 | 季度 | 收入贡献+增速 |
| **评估** | 收入+质量+满意度+增长潜力的综合评分 | 季度 | 分级调整 |
| **退出** | 30天通知+过渡+复盘 | 需要时 | 无负面遗留 |

### 4. 渠道冲突管理策略

#### 常见冲突场景与解决方案

| 冲突类型 | 场景 | 解决方案 |
|---------|------|---------|
| **价格冲突** | 多合作伙伴竞争同一客户，给出不同报价 | 统一定价策略（不议价/仅折扣码） |
| **地域冲突** | 两个合作伙伴覆盖同一区域 | 区域独占可付费（¥5K/季-¥20K/年） |
| **客户归属** | 两个伙伴同时认领同一客户 | 规则：首次接触/最新互动/IP匹配 |
| **直接销售vs伙伴** | 内销团队与伙伴抢单 | 内销专注企业级(¥500K+) / 伙伴专注中小企业 |
| **产品重叠** | 合作伙伴自己也推竞品 | 签约排他性条款或最低业绩承诺 |

#### 冲突缓解机制

```
① 明确定义合作边界（地域/客户规模/行业/价格区间/产品线）
② 建立伙伴-内销的协作流程
③ 归属规则公开透明（签约前明确）
④ 冲突升级至渠道管理总监，7天内裁决
⑤ 季度沟通会，预防性解决潜在冲突
```

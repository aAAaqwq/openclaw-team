# Alpha因子评审

对已挖掘的Alpha因子进行系统性验收，确保因子质量符合实盘标准。

## 因子验收检查清单

### 必要条件项

- [ ] **统计显著性**: IC p-value < 0.05，t-stat > 2
- [ ] **经济意义**: 因子逻辑可解释，有理论基础
- [ ] **稳健性**: 不同市场时段/子样本表现一致
- [ ] **新颖性**: 与现有因子库相关性 < 0.7
- [ ] **可操作性**: 交易成本后仍显著盈利
- [ ] **数据可得性**: 因子数据源稳定可靠

### 加分项

- [ ] 多市场/跨资产有效
- [ ] 低频版同样有效
- [ ] 逆向（反转版）有效
- [ ] 有另类数据源支撑

## 样本外测试协议

### 时间序列交叉验证

```
Period 1: [===train===][===test===]
Period 2:          [===train===][===test===]
Period 3:                   [===train===][===test===]
```

| 验证类型 | 窗口长度 | 滚动步长 | 最低要求 |
|---------|---------|---------|---------|
| 前向验证 | 50% train / 50% test | — | 一次 |
| 滚动向前 | 6个月/1个月 | 1个月 | 6次折叠 |
| 扩展窗口 | 从起点递增 | 固定test长度 | 6次折叠 |

## 因子组合测试

### 边际贡献测试

```
逐步回归: R²变化
    ├── 基础模型: 仅含控制因子
    ├── +新因子: R²增量
    └── 显著性检验: F-test
```

### 正交化处理

```
新因子_正交 = 新因子 - Σ(β_i × 旧因子_i)

正交化后IC如果显著下降 → 信息已被已有因子捕获
正交化后IC基本不变 → 新因子提供独立信息
```

| 测试 | 结果 | 判定 |
|------|------|------|
| 样本外IC | 0.028 | ✅ |

# 压力测试模拟器

模拟极端市场情境下的投资组合表现，量化尾部风险暴露。

## 黑天鹅场景库

### 经典危机场景

| 场景名称 | 时间 | 跌幅 | 核心特征 |
|---------|------|------|---------|
| **08金融危机** | 2008.09 | S&P 500 -38% | 次贷危机，系统性流动性枯竭 |
| **20熔断** | 2020.03 | 美股-34%四次熔断 | 新冠疫情黑天鹅 |
| **312暴跌** | 2020.03.12 | BTC -50%日内 | 加密市场流动性危机 |
| **22LUNA崩盘** | 2022.05 | LUNA -99.99% | 算法稳定币脱锚螺旋 |
| **24流动性危机** | 2024.08 | 日元套利交易平仓 | 全球carry trade平仓连锁反应 |

### 自定义场景参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 最大跌幅 | 组合最大亏损比例 | -30% |
| 恢复时间 | 恢复到前期高点所需天数 | 180天 |
| 波动率膨胀 | 波动率倍数 | 3x |
| 相关性变化 | 跨资产相关系数变动 | +0.3 |
| 流动性折扣 | 资产卖出滑点倍数 | 5x |

## 蒙特卡洛模拟

### 方法论

```
N(10000+) 次随机路径模拟
    ├── 几何布朗运动(GBM)
    ├── 跳跃扩散模型(Merton)
    └── 极值理论(EVT)尾部抽样
```

### 输出指标

- **VaR（压力版）**: 99%/95% VaR
- **CVaR（压力版）**: 期望尾部损失
- **最差回撤**: 模拟路径中最大回撤
- **恢复概率**: 给定时间内恢复的概率

## VaR/CVaR压力版

| 置信水平 | 常规VaR | 压力VaR | CVaR |
|---------|--------|---------|------|
| 95% | -2.5% | -8.5% | -12.0% |
| 99% | -4.0% | -15.0% | -22.0% |
| 99.5% | -5.0% | -20.0% | -30.0% |

## 压力测试报告模板

```markdown
# 压力测试报告 YYYY-MM-DD

## 场景矩阵
- 基础场景: 当前市场配置
- 场景1: 08金融危机复现 (概率3%)
- 场景2: 加密流动性危机 (概率5%)
- 场景3: 全球衰退+通胀 (概率8%)

## 模拟结果
| 场景 | 最差回撤 | 恢复天数 | 概率(≤回撤) |
|------|---------|---------|------------|
| 基础 | -8.2% | 45天 | 5% |
| 08危机 | -25.3% | 180天 | 1% |
| 流动性危机 | -35.1% | 240天 | 0.5% |
| 衰退+通胀 | -18.7% | 120天 | 2% |

## 风控建议
- [ ] 当前组合在场景2下风险暴露过大
- [ ] 建议降低高相关性资产配置
- [ ] 增加对冲头寸(期权PUT)
```

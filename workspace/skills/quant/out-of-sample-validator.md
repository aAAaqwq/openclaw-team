## 过拟合检测

### Deflated Sharpe Ratio

DSR校正了多重测试导致的过拟合偏差：

```
DSR = SR × (1 - γ × rank) / √(1/N + (E[max(SR)]))

其中:
γ = 欧拉常数 ≈ 0.577
rank = 策略在候选策略中的排名
N = 独立试验次数
```

### Num of False Strategies (NFS)

```python
def compute_nfs(sr_values, trials, alpha=0.05):
    """
    计算多重测试下预期错误发现的策略数
    """
    max_sr = max(sr_values)
    # 计算期望最大夏普
    _, p_values = scipy.stats.shapiro(sr_values)
    return sum(1 for p in p_values if p < alpha / trials)
```

### 过拟合检测清单

- [ ] DSR显著 > 0（多重检验调整后）
- [ ] NFS < 总策略数 × 5%
- [ ] 样本外IC衰减 < 30%
- [ ] 训练集与测试集表现无显著差异

## 鲁棒性测试

### 参数扰动测试

| 测试类型 | 方法 | 阈值 |
|---------|------|------|
| 参数敏感度 | 关键参数±20% | 夏普变化 < 0.3 |
| 时段分割 | 牛/熊/震荡市分别测试 | 所有时段夏普 > 0.5 |
| 数据换基 | 不同数据源 | 结果方向一致 |

### 市场时段分割

```
牛市场景 (涨幅>20%) → 策略表现
熊市场景 (跌幅>20%) → 策略表现
震荡市场景 (±10%) → 策略表现
趋势市场景 (无明显趋势) → 策略表现
```

# Alpha衰减监控

实时监控策略Alpha衰减，设定分级告警并触发自动化响应流程。

## 实时监控框架

### 核心指标

| 指标 | 计算方式 | 监控频率 |
|------|---------|---------|
| **滚动夏普** | 过去N交易日Sharpe Ratio | 每日滚动 |
| **盈亏比** | 平均盈利/平均亏损 | 每周 |
| **胜率** | 盈利交易占比 | 每日 |
| **最大回撤** | Peak-to-Trough回撤 | 实时 |
| **IC衰减** | 因子IC值滑动平均 | 每日 |

### 衰减阈值设定与告警体系

```
H3（致命）🚨  → 立即下线策略
   ↑
H2（严重）⚠️  → 降权50%，启动归因
   ↑
H1（预警）🔔  → 人工复检，准备降权
   ↑
正常 ✅
```

| 等级 | 触发条件 | 采取措施 |
|------|---------|---------|
| **H1** | 滚动夏普 < 1.0 或 IC < 0.02 | 通知投研团队，标记观察 |
| **H2** | 滚动夏普 < 0.5 或 回撤 > 15% | 降权50%，自动归因分析 |
| **H3** | 滚动夏普 < 0 或 回撤 > 25% | 强制下线，冻结仓位 |

### 策略降权/下线自动化流程

```
H1标记 → 人工审核窗口(24h)
    ├── 恢复 → 解除标记
    └── 恶化 → H2触发

H2触发 → 自动降权50%
    ├── 恢复 → 逐步加回仓位(3天)
    └── 恶化 → H3触发

H3触发 → 强制下线
    ├── 释放全部仓位
    ├── 冻结策略
    └── 通知风控组
```

## 归因报告模板

### 日报告模板
```markdown
# Alpha衰减日报 YYYY-MM-DD

## 概况
- 运行策略数: N (正常/M/H1/H2/H3)
- 今日触发告警: [无/H1: 策略A/B / H2: 策略C]

## 详细指标
| 策略 | 滚动夏普 | 盈亏比 | 胜率 | 当前回撤 | 状态 |
|------|---------|-------|------|---------|------|
| 策略A | 0.85 | 1.2 | 55% | -8.2% | H1🔔 |
| 策略B | 1.35 | 2.1 | 62% | -3.1% | 正常✅ |

## 操作建议
- [ ] 策略A: 关注下一交易日，如继续恶化降权
- [ ] 策略B: 维持当前仓位
```

### 周报告模板
- 本周触发告警统计
- 各策略夏普趋势图
- 归因分析结论
- 仓位调整建议

### 月报告模板
- 全月告警汇总
- 因子IC衰减分析
- 策略生命周期评估
- 下月调仓计划

# 策略归因分析

对策略收益进行拆解，识别收益来源与风险暴露。

## Brinson归因

将组合超额收益分解为三个效应：

### 模型框架

```
超额收益 = 配置效应 + 选择效应 + 交互效应
```

| 效应 | 公式 | 含义 |
|------|------|------|
| **配置效应** | Σ(w_pi - w_bi) × R_bi | 行业/资产配置能力 |
| **选择效应** | Σw_bi × (R_pi - R_bi) | 选股/选币能力 |
| **交互效应** | Σ(w_pi - w_bi) × (R_pi - R_bi) | 配置+选择的交互 |

### 计算步骤

```
1. 确定基准（如BTC/ETH 60/40）
2. 计算各资产权重偏离
3. 计算各效应贡献
4. 汇总验证: 总超额收益 = 三效应之和
```

## 因子暴露归因

### Fama-French多因子模型

```
R_i - R_f = α + β₁(R_m - R_f) + β₂SMB + β₃HML + β₄MOM + ε
```

| 因子 | 含义 | Beta解读 |
|------|------|---------|
| **市场** | 市场超额收益 | β>1 进攻型 |
| **规模(SMB)** | 小盘-大盘 | 正值=偏小盘 |
| **价值(HML)** | 高BM-低BM | 正值=偏价值 |
| **动量(MOM)** | 赢家-输家 | 正值=偏动量 |

### 因子暴露分析

```
每月滚动回归 → 暴露变化趋势 → 收益来源分解
    ↑                   ↑
 36个月窗口        累积因子贡献
```

## 归因报告自动化

### 自动生成流程

```
每日结束后触发
    → 拉取当日交易数据
    → 计算组合权重
    → Brinson归因计算
    → 因子暴露回归
    → 生成报告
    → 推送到通知渠道
```

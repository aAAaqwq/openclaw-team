# A股融券T+0

A股市场融券T+0交易策略，利用日内波动捕捉短线盈利机会。

## 融券T+0交易策略

### 基本原理

A股实行T+1交割制度（买入次日才能卖出），但融券业务允许T+0：

```
1. 盘前: 借入标的券
2. 盘中: 高位融券卖出（做空）
3. 日内: 低位买券还券（平仓）
4. 结果: 赚取日内价差
```

### 策略分类

| 策略 | 方向 | 持仓时间 | 收益率预期 |
|------|------|---------|-----------|
| **追涨做空** | 拉升后回落 | 30-120分钟 | 0.5%-2% |
| **暴跌抄底** | 恐慌性抛售后反弹 | 60-180分钟 | 1%-3% |
| **震荡套利** | 价格区间上下沿 | 20-60分钟 | 0.3%-0.8% |
| **量价背离** | 价格上涨但量能萎缩 | 60-120分钟 | 1%-2% |

## 融券标的筛选

### 筛选条件

| 条件 | 筛选标准 | 优先级 |
|------|---------|--------|
| 融券余额 | > 5000万元 | 高 |
| 融券费率 | < 10.35%/年 | 高 |
| 日均换手率 | > 2% | 中 |
| 日均振幅 | > 3% | 高 |
| 流通市值 | 50-500亿 | 中 |
| 融券余量限制 | 未被限售 | 高 |

### 黑名单

- ST/*ST股票
- 当日有重大利好/利空
- 融券余额为0
- 融券费率异常高

## 日内交易信号

### 量价背离信号

```
价格上涨 + 成交量下降 → 做空信号
价格下跌 + 成交量萎缩 → 做多信号（还券）
```

### 分时图模式

| 模式 | 特征 | 操作 |
|------|------|------|
| 尖顶 | 快速拉升后回落 | 回落拐点做空 |
| 台阶下跌 | 逐级下跌 | 每个台阶做空 |
| V型反转 | 深V反弹 | 右侧确认后做多还券 |
| 放量横盘 | 底部放量横盘 | 准备入场 |

### 技术指标辅助

- **RSI**: RSI > 70 超买做空，< 30 超卖做多还券
- **MACD**: 背离信号 + 零轴位置
- **布林带**: 触碰上轨做空，下轨做多还券
- **成交量**: 异常放量是信号确认

## 与stock-analysis集成

```python
# 筛选可融券标的表现
from stock_analysis import StockAnalyzer

analyzer = StockAnalyzer()
targets = analyzer.filter_shortable(
    min_mcap=5e9,
    max_mcap=5e10,
    min_amplitude=0.03,
    min_volume=5e7,
    short_balance_min=5e7
)

# 日内信号检测
for stock in targets:
    signal = analyzer.check_t0_signal(
        ticker=stock,
        timeframe='1min'
    )
    if signal:
        print(f"{stock}: {signal.type} @ {signal.price}")
```

## 风险控制

| 风险 | 措施 |
|------|------|
| 融券被收回 | 分散券源，避免单一标的 |
| 大涨踏空 | 设置止损，$-\%$ 割肉 |
| 流动性不足 | 筛选活跃标的 |
| 费率成本 | 持有时间<2小时降低成本 |

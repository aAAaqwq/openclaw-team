# 动态仓位管理

基于量化模型动态分配策略仓位，平衡收益与风险。

> **注**：量化交易专用版。风控场景下的同主题文件见 [dynamic-position-sizing (risk版)](../skills/risk/dynamic-position-sizing.md)。

## 凯利公式

### 经典凯利

```
f* = (bp - q) / b

f* = 最优下注比例
b  = 赔率（盈亏比）
p  = 胜率
q  = 败率 (1-p)
```

### 变体

| 版本 | 公式 | 适用场景 |
|------|------|---------|
| **全凯利** | f* = (bp-q)/b | 高信度策略 |
| **半凯利** | f*/2 | 保守策略，降低波动 |
| **分数凯利** | f* × k (0<k<1) | 可控风险偏好 |
| **多个独立策略** | f = (p/b) - (q/(1-b)) | 多策略组合 |

## 风险平价

### 等风险贡献(ERC)

```
min Σ(w_i × ∂σ/∂w_i - σ_target)²
subject to: Σw_i = 1, w_i ≥ 0
```

- 每项资产对组合风险的贡献相等
- 天然偏多的低波动率资产
- 可通过杠杆调节整体风险水平

### 算法步骤

```
1. 计算协方差矩阵
2. 初始化权重（如等权）
3. 迭代优化至各资产风险贡献收敛
4. 输出最优权重
```

## 最大回撤约束仓位

### 基于DD的仓位控制

- **固定比例**: 当前DD每增加1%，仓位减少X%
- **阶梯式**: DD < 5%满仓，5-10%减半，>15%清仓
- **非线性**: f = f₀ × (1 - DD/DD_max)²

### 参数

| 参数 | 建议值 | 说明 |
|------|--------|------|
| DD上限 | 20% | 最大允许回撤 |
| 保守系数 | 0.5-0.8 | 分数凯利系数 |
| 减仓速度 | 5%/天 | 最长减仓时间 |

## 多策略仓位分配

### 相关性加权

- 低相关性策略 → 高权重
- 高相关性策略 → 合并或降权
- 负相关性策略 → 最优组合

### 分配方法

| 方法 | 公式 | 特点 |
|------|------|------|
| 等权 | w_i = 1/N | 简单稳健 |
| 风险平价 | 风险贡献相等 | 分散化 |
| 均值-方差 | max(μ - λσ²) | 信息要求高 |
| 黑-利特曼 | 先验+观点合成 | 主观调整 |

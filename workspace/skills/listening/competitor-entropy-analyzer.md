# Competitor Entropy Analyzer

## When to Apply
Use when monitoring competitive landscape, evaluating market positioning, or identifying strategic attack windows.

## Analysis Dimensions

### 1. Product Entropy
```
Monitor:
- Feature releases (changelog, product hunt)
- API changes (breaking changes → churn risk)
- Pricing changes (increase → user dissatisfaction)
- UI redesign (disruption → adoption dip)

Signal: 
  Major redesign + pricing change = high user churn window
```

### 2. Talent Entropy
```
Monitor:
- Key exits (VP Eng, CPO, Head of Growth leaving)
- Hiring spikes (certain roles = strategic shift)
- Location changes (remote → RTO = talent loss)

Signal:
  VP Engineering + 2 senior ICs leave = product velocity will drop
```

### 3. Financial Entropy
```
Monitor:
- Funding rounds (Series A/B/C, valuation changes)
- Revenue reports (growth rate changes)
- Marketing spend (increasing = user acquisition pressure)
- Layoffs (cost cutting = growth stalled)

Signal:
  Layoff + new funding = desperate pivot incoming
```

### 4. Attack Window Matrix

| Competitor Event | Attack Window | Recommended Move |
|-----------------|---------------|-----------------|
| Major price increase | 4-8 weeks | Launch competitor comparison page |
| Product redesign | 4-12 weeks | Target their power users with migration offer |
| Key exec departure | 8-16 weeks | Hire the departed talent + target their network |
| Down round funding | 4-12 weeks | Aggressive customer acquisition with discounts |
| Public security breach | 2-4 weeks | Publish security comparison + migration guide |

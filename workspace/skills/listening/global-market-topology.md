# 全球市场拓扑 (Global Market Topology)

> 跨市场增长策略映射框架。分析不同国家/区域市场的结构特征、进入壁垒和文化适配需求，帮助CGO**精准选择下一个增量市场**。

## 核心方法

### 1. 跨国市场增长策略映射

对每个目标市场，按以下维度评估：

| 维度 | 考察内容 | 数据来源 |
|------|---------|---------|
| **市场规模** | TAM/SAM/SOM + CAGR | 行业报告/Grand View/Statista |
| **数字化成熟度** | 互联网渗透率、电商占比、AI应用率 | World Bank/GSMA/IDC |
| **竞争格局** | 本地玩家+全球玩家+新进入者 | Crunchbase/SimilarWeb |
| **付费意愿** | ARPU、支付习惯、SaaS渗透率 | Gartner/Forrester/本地调研 |
| **法规风险** | 数据合规（GDPR/PIPL）、行业准入 | 律所报告/本地顾问 |
| **文化距离** | 语言、商业习惯、信任机制 | Hofstede指数/本地化专家 |
| **渠道可触达** | 搜索/社交/合作伙伴/BD | SimilarWeb/Sensor Tower |

### 2. 区域市场进入优先级评估

采用加权评分模型：

```
Entry Score = MarketSize(0.25) + DigitalMaturity(0.20) + WillingnessToPay(0.20) 
            + CompetitionFavorable(0.15) + RegulationScore(0.10) + CulturalFit(0.10)
```

**评分等级：**
- **≥ 7.5**：立即进入（P0）
- **5.0 - 7.4**：策略性进入（P1）
- **3.0 - 4.9**：观察评估（P2）
- **< 3.0**：暂不考虑

**典型市场特征矩阵：**

| 市场类型 | 特征 | 典型区域 | 进入策略 |
|---------|------|---------|---------|
| **成熟市场** | 高ARPU、高竞争、高合规成本 | 北美、西欧、澳洲 | PE并购/合作伙伴 |
| **成长市场** | 高增长、中度竞争、劳动力成本低 | 东南亚、中东、东欧 | 绿地投资/合资 |
| **新兴市场** | 高潜用户、基础设施弱、价格敏感 | 非洲、南亚、拉美 | 轻资产模式/生态合作 |
| **红海市场** | 高竞争、增长放缓、价格战 | 中国大陆部分行业 | 差异化/利基市场 |
| **蓝海市场** | 新需求、低竞争、教育成本高 | AI类目在新区域 | 先发优势/市场教育 |

### 3. 文化适配检查清单

#### 语言层
- [ ] 产品/网站已进行**专业本地化**（非机器翻译）
- [ ] 关键词研究是否有本地语言版本（非仅英文词）
- [ ] 已识别本地圈内术语/俚语/禁忌词
- [ ] SEO已适配本地搜索引擎（百度/Yandex/Naver）

#### 视觉层
- [ ] 色彩搭配无文化冲突（红=好运/危险因地而异）
- [ ] 图片中的人物/场景体现本地多样性
- [ ] 数字/日期/货币格式已本地化
- [ ] 图标/符号无文化歧义

#### 信任层
- [ ] 本地化的信任信号（支付方式/认证/评奖）
- [ ] 是否有本地用户案例/评价
- [ ] 了解本地消费者保护法规
- [ ] 售后/客服已启用本地语言+时区

#### 习惯层
- [ ] 定价符合本地支付习惯（分期/预付/货到付款）
- [ ] 内容发布/活动时间契合本地作息
- [ ] 社交媒体分布在本地主流平台
- [ ] 冷启动渠道（微信群/Telegram/Kakao等）

### 4. 全球-本地（Glocal）增长平衡模型

```
Glocal Index = 标准化度 × 本地化度

标准化度：核心产品/品牌/技术可复用的程度（越高越好）
本地化度：营销/渠道/定价针对本地调整的程度（越高越好）

理想状态：核心能力标准化（60-70%）+ 执行层面本地化（30-40%）
```

**不同阶段的Glocal策略：**

| 阶段 | 标准化% | 本地化% | 策略说明 |
|------|---------|---------|---------|
| **第一阶段**（1-2个市场） | 80% | 20% | 快速复制MVP，最小化本地化投入验证PMF |
| **第二阶段**（3-5个市场） | 65% | 35% | 核心产品标准化+渠道/营销深度本地化 |
| **第三阶段**（5个市场+） | 50% | 50% | 建立本地团队，区域自主决策 |

## 全球市场拓展产出模板

```
MARKET ENTRY REPORT — [市场名]
─────────────────────────────────
1. 市场规模：TAM=$XXB / SAM=$XB / CAGR=X%
2. Entry Score：7.2/10 → P1
3. 关键优势：[列举2-3个]
4. 关键风险：[列举2-3个]
5. Glocal策略：标准化XX / 本地化XX
6. 预计投入：$$$ / 预期ROI：X:X / 回收期：M个月
7. 建议进入方式：[合作伙伴/合资/绿地/并购]
8. 优先行动：[7天内可执行的1-2项]
```

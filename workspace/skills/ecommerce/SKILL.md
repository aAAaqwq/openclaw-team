# 电商平台运营 Skill Suite

> 麒麟的核心战场 — 多平台全链路电商运营引擎

## 定位

涵盖国内电商（淘宝/天猫/拼多多/抖音小店/京东）和跨境电商（亚马逊/Shopify/TikTok Shop/速卖通）的全平台运营能力。

## 标准运营SOP

### 每日检查清单

```
□ ① 生意参谋/罗盘/商智 — 昨日核心数据（GMV/访客/转化率/客单价）
□ ② 竞品监控 — 价格变动/上新/评价变化
□ ③ 广告数据 — 花费/ROI/ACOS/CPA，调整出价
□ ④ 库存预警 — 缺货/滞销/退货率
□ ⑤ 客服数据 — 响应时长/好评率/差评处理
□ ⑥ 内容发布 — 短视频/直播/笔记排期
```

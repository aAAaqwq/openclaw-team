# 爆款孵化与选品引擎

基于数据驱动的爆款选品与孵化方法论，从市场洞察到产品定义的完整链路。

## 选品方法论

### 数据选品四象限

```
           高需求
              │
              │
     机会产品  │  爆款/明星产品
     (潜力股)  │  (全力推)
     ──────────┼──────────
     问题产品  │  金牛产品
     (需验证)  │  (精细化收割)
              │
           低需求
       低竞争 ──── 高竞争
```

| 象限 | 特征 | 策略 |
|------|------|------|
| **爆款** | 高需求+高竞争 | 差异化竞争，品牌溢价 |
| **机会** | 高需求+低竞争 | **蓝海**，快速切入 |
| **金牛** | 低需求+高竞争 | 不投入，维护即可 |
| **问题** | 低需求+低竞争 | 小批量测试，不行就撤 |

```yaml
- 货源稳定性: ⭐⭐⭐⭐⭐
- 起订量: XXX件
- 采购成本: ¥XX
- 物流适配: 普通/特殊/冷链

- 视觉门槛: 低/中/高
- 内容可做性: 短视频/直播/图文
- 达人BD难度: 低/中/高
- 广告可放量性: ⭐⭐⭐⭐⭐

结论: 推荐/谨慎/放弃
```

## 爆款孵化SOP

```
Week 1-2: 市场调研与选品确认
    ↓ 确定核心卖点+差异化方向
Week 3-4: 打样/采购+视觉制作+Listing
    ↓ 上架，小预算测试
Week 5-6: 数据诊断（点击率/转化率）
    ↓ 优化主图/详情/定价/评价
Week 7-8: 放量测试（广告+达人+内容）
    ↓ 确认ROI可盈利
Week 9+: 全面放量+备货+大促排期
```

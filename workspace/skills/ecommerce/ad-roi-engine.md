## ROI优化方法论

### 广告ROI公式

```
ROI = GMV / 广告花费
    = (展现量 × CTR × CVR × 客单价) / (展现量 × CPC)
    = CTR × CVR × 客单价 / CPC
```

### 优化杠杆

| 杠杆 | 提升策略 | 影响幅度 |
|------|---------|---------|
| **CTR（点击率）** | 主图/标题/素材创意A/B测试 | +20-100% |
| **CVR（转化率）** | 详情页/评价/信任感/定价 | +10-50% |
| **客单价** | 搭配推荐/满减/捆绑销售 | +5-30% |
| **CPC（单次点击成本）** | 质量分优化/精准人群/低竞词 | -10-40% |

### 广告预算分配规则

```
总预算 = 目标GMV / 目标ROI

各平台分配：
  主流平台（淘宝+抖音）= 70%
  测试平台 = 20%
  备用金 = 10%

日常 vs 大促：
  日常: 60% 收割型 + 40% 拉新型
  大促前: 30% 收割 + 70% 预热拉新
  大促中: 50% 收割 + 50% 拉新
```

## 素材A/B测试框架

### 测试变量

| 变量 | 测试内容 | 最小样本量 |
|------|---------|-----------|
| **主图** | 色彩/构图/文案/模特/场景 | 1000次展现 |
| **标题** | 卖点顺序/关键词/情感词 | 500次展现 |
| **详情页** | 结构/图片/文案/视频 | 500次UV |
| **落地页** | 布局/CTA按钮/价格显示 | 500次UV |

### 测试流程

```
1. 提出假设（基于数据）
2. 创建对照组（当前最优）和实验组（新版本）
3. 均匀流量分配
4. 运行至统计显著（通常3-7天）
5. 分析结果：转化率、ROI、置信区间
6. 如实验组显著胜出 → 全量替换
7. 记录学习（将来复用）
```

## 与大促配合的时间线

```
T-30天: 测款+测素材（低预算探方向）
T-21天: 确认主推品+确定素材方向
T-14天: 开始蓄水（拉新人群积累）
T-7天: 预热期（最大化曝光+加购）
T-0天: 爆发期（收割+冲刺GMV）
T+7天: 复盘期（数据归因+总结）
```

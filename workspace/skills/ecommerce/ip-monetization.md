## IP产品线迭代流程

```
Phase 1: 测试期（2-4周）
  - 用最低成本验证需求（简单PDF/录播课）
  - 核心指标：转化率、NPS、退款率
  
Phase 2: 打磨期（4-8周）
  - 根据用户反馈迭代产品
  - 加入社群/答疑等增值服务
  - 核心指标：复购率、推荐率
  
Phase 3: 放量期（8周+）
  - 全渠道推广
  - 差异化定价（基础版/专业版/企业版）
  - 核心指标：LTV、CAC回收期
```

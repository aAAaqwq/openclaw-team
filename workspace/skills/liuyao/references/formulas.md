# 解卦口诀与公式

## 用神选取口诀

```
问财妻财问官鬼，
问事父母问文书，
问病官鬼问出行，
五行六亲各归位。
```

### 用神速查表

| 占问事项 | 用神 | 说明 |
|----------|------|------|
| 问财/生意/投资 | 妻财爻 | 财运、物质 |
| 问官/工作/事业 | 官鬼爻 | 事业、职位 |
| 问事/求谋/合作 | 父母爻 | 事情本身、文书、合同 |
| 问考试/学业/升学 | 官鬼爻 | 功名、学习 |
| 问文书/证件/合同 | 父母爻 | 文件、证明 |
| 问行人/远行 | 父母爻 | 道路、交通 |
| 问疾病 | 官鬼爻 | 病因在那里 |
| 问婚姻/感情 | 妻财爻（男） | 妻、情人 |
|              | 官鬼爻（女） | 夫、情人 |
| 问诉讼/是非 | 官鬼爻 | 对手、官府 |
| 问文书/信件 | 父母爻 | 载体、信息 |

## 旺相休囚口诀

```
当令得旺相生助，
休囚死极渐消亡，
旺相则为吉兆，
休囚死则为凶兆。
```

### 旺相休囚判断

| 状态 | 含义 | 力量 |
|------|------|------|
| 旺 | 当令（月建与爻同五行） | 最强 |
| 相 | 我令（爻被月建生） | 次强 |
| 休 | 退令（爻去生月建） | 中 |
| 囚 | 衰败（爻被月建克） | 弱 |
| 死 | 失令（爻反来生月建） | 最弱 |

## 快速解卦流程

```
① 列出卦象
② 找用神
③ 看用神旺衰（日月为主）
④ 看动爻影响
⑤ 综合判断吉凶
⑥ 参看应期
```

## 动变分析口诀

```
动爻生用神，吉利
动爻克用神，凶险
动爻与用神同，谨慎
变卦显示趋势
```

## 生克冲合规则

| 关系 | 结果 |
|------|------|
| 生 | 有利、支持 |
| 克 | 不利、阻碍 |
| 冲 | 变动、激活 |
| 合 | 绊住、拖延 |

---

## 应期速断

| 用神状态 | 应期 |
|----------|------|
| 旺相发动 | 当日或当月 |
| 休囚发动 | 数月后 |
| 逢冲则动 | 冲日/冲月 |
| 逢合则绊 | 合日/合月 |
| 入墓 | 冲墓之日 |

# Stealth Engagement Protocol — 非招聘式人才接触

> 版本：v1.0 | 分类：碳基人才猎取 | 优先级：P1
> 作者：稷下 | 对标：阿里"客户第一" × 麦肯锡顾问式接触 × Reid Hoffman "Calibrated Outreach"
> 触发场景：候选人短名单确定 → 首次接触阶段

---

## 核心价值

永远不说"我们想雇佣你"，而是通过"技术价值先于招聘"路径，建立深度专业认同，让人才主动被吸引而非被说服。

---

## 经典对标

### Reid Hoffman's "Calibrated Outreach"

LinkedIn创始人Reid Hoffman的接触原则：
> "Give before you ask. Make the first interaction valuable to the other person, not just to you."

稷下对齐：每次接触必须提供真实价值（技术洞察、行业信息、资源连接）。

### 阿里"客户第一"原则

阿里铁军销售方法论：
- 先了解对方需求
- 再提供匹配价值
- 信任建立后才自然导入合作

稷下对齐：把候选人当"客户"而非"猎物"。

### 麦肯锡顾问式接触

麦肯锡顾部问接触客户的方式：
1. 展示对行业的深度理解
2. 提出独到观点，而非标准话术
3. 让对方感受到"这个人和我在一个层次"

稷下对齐：接触时展示军团的真实技术挑战，而非空洞的"我们在做AI"。

## 四阶段接触流程

### Stage 1：价值投射（第1-2次接触）

```
目标：建立专业认同，展示真实价值

行动：
• 通过deep-profile-decoder解码其兴趣领域
• 准备1-2个与该领域相关的"真实技术问题"请教
• 或分享一篇与其研究高度相关的军团队技术分享

话术框架：
"我们正在研究一个XX问题，遇到了YY挑战，
看到你在ZZ方面的研究，想请教一下你的看法。"
```

### Stage 2：信任建立（第3-5次接触）

```
目标：从"单向请教"转向"双向交流"

行动：
• 分享军团在相关领域的进展（可公开的）
• 邀请参加蓝血俱乐部活动（如果是专业人士）
• 适时提出"合作探讨"而非"加入我们"

话术框架：
"我们的XX项目有了新进展，想听听你的视角，
也顺便聊聊你最近在忙什么。"
```

### Stage 3：价值确认（第6-8次接触）

```
目标：让候选人看到在军团工作的具体价值

行动：
• 安排与创始人或核心统帅的非正式交流
• 展示候选人在军团可以解决的"具体挑战"
• 介绍团队成员（硅基+碳基混合）

话术框架：
"我们正在规划一个XX方向，感觉你的背景特别适合，
想听听你对这件事的看法，如果有兴趣可以一起来定义。"
```

### Stage 4：自然导入（第9+次接触）

```
目标：候选人主动提出合作意向

触发条件：
• 候选人主动询问军团机会
• 候选人对具体项目表现出强烈兴趣
• 候选人对团队文化表达认同

行动：
• 正式但非正式地介绍Offer选项
• 引入competitive-offer-architect
• 启动蓝血入职季流程
```

## 华为"铁三角"攻心术

华为销售铁三角：
- **开路**：以客户需求为切入点
- **攻心**：建立信任和情感连接
- **转化**：自然过渡到合作

稷下改造为"人才攻心三板斧"：
1. **技术共鸣**：让人才感受到"这里有真正的技术挑战"
2. **使命共振**：让人才看到"这件事有意义"
3. **成长共振**：让人才相信"在这里我能超预期成长"

## 场景化话术模板

### 场景1：顶级研究员

```
"看到你最近在Transformer优化上的研究，我们也在解决类似问题。
你的论文XX给了我们很大启发，特别是YY这个方向。
我们有一个XX场景遇到了瓶颈，方便请教一下你的看法吗？"
```

```
"你之前在Two Sigma的组合优化工作很印象深刻。
我们正在搭建一套量化信号系统，在因子挖掘上遇到了瓶颈，
看到你在这块的经验，想请教一下你的架构思路。"
```

### 场景3：产品负责人

```
"你们产品从0到1的增长路径很值得研究。
我们目前在一个新方向探索，用户获取效率还有很大提升空间，
看到你在这块的实战经验，想聊聊你在字节时是怎么做的。"
```

## 踩坑记录

### 坑1：过早暴露招聘意图
- 问题：第一次接触就提"我们在招聘"
- 解决：Stage 1-2完全不提招聘，只建立专业认同

### 坑2：话术太标准化
- 问题：用"我们正在做AI，想招人"这样的标准话术
- 解决：每次接触必须基于deep-profile-decoder的个性化定制

### 坑3：忽视候选人的时间价值
- 问题：发简短消息让人家"有没有兴趣聊聊"
- 解决：每次接触必须提供具体价值，让候选人感受到"花时间值得"

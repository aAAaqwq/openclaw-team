# Partner Sales — 渠道合作伙伴销售

## 适用场景
当需要建立/管理渠道合作伙伴网络时调用。与霸下SKILL.md中的 `channel-partner-pipeline` 技能联动。

## 核心能力

### 1. 渠道合作伙伴分类
| 类型 | 全称 | 典型业务 | 适合OpenClaw的场景 |
|:----:|:----:|---------|-------------------|
| **VAR** | Value-Added Reseller | 增值转售 | 捆绑AI数字员工+硬件 |
| **SI** | System Integrator | 系统集成商 | GEO+AI+RPA方案集成 |
| **ISV** | Independent Software Vendor | 独立软件开发商 | 嵌入式AI能力合作 |
| **MSP** | Managed Service Provider | 托管服务商 | 持续AI运维服务转售 |
| **Agent** | 代理商 | 线索推荐+交易撮合 | 区域市场拓展 |

### 2. 合作伙伴生命周期管理
```
招募 → 认证 → 赋能 → 联合销售 → 业绩激励 → 复盘审计 → 续约/淘汰
```

### 3. 渠道冲突管理SOP
- 场景A（价格冲突）：统一报价基线 + 区域保护
- 场景B（客户重叠）：注册制 + 第一条发现规则
- 场景C（合同冲突）：分层报价 + 独家条款

### 4. 联合GTM计划模板
- 目标 + 目标客户清单 + 联合营销预算 + 角色分工 + 时间线 + 退出条件

### 5. 渠道激励设计
- 返点设计（固定/累进/阶梯）
- MDF（市场发展基金）管理
- SPIF（短期激励）方案

## 协作接口
- **霸下**：渠道管线管理与激励设计
- **鲲鹏**：跨境渠道合作拓展
- **明镜**：渠道合作协议合规审查
- **司库**：返点计算与支付确认

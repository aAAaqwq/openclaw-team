# 动态仓位管理（风控版）

基于风险管理框架的动态仓位分配系统，核心聚焦于风险暴露控制。

> **注**：风控专用版。量化交易端的同主题文件见 [dynamic-position-sizing (quant版)](../skills/quant/dynamic-position-sizing.md)。

## 凯利公式全解

### 最优f值

```
f* = (bp - q) / b

f* = 最优投注比例
b  = 净赔率（盈亏比）
p  = 胜率
q  = 败率 (1-p)
```

### 变体与系数

| 版本 | 公式 | 风险偏好 | 适用 |
|------|------|---------|------|
| **全凯利** | f* | 激进 | 高确定性 |
| **半凯利** | f*/2 | 稳健 | 常规策略 |
| **1/4凯利** | f*/4 | 保守 | 低置信度 |
| **分数凯利** | f* × k (0<k<1) | 可调 | 通用 |

### 多策略凯利

```
当策略间相关性ρ=0:
f_i_total = f_i / (1 + Σρ_ij)

当策略正相关:
f_reduced = f / (1 + ρ × (n-1))
```

## 风险平价算法

### 等风险贡献(ERC)

目标: 每项资产/策略对组合风险贡献相等。

```
min Σ(w_i × (∂σ/∂w_i) - target)²
s.t. Σw_i = 1, w_i ≥ 0
```

### 迭代求解

```
1. 初始化: w_i = 1/N
2. 计算协方差矩阵 Σ
3. 计算边际风险贡献: MRC_i = (Σw)_i / √(w'Σw)
4. 计算总风险贡献: TRC_i = w_i × MRC_i
5. 调整权重使 TRC_i 收敛
6. 迭代至收敛
```

## 最大回撤约束仓位优化

### 动态DD约束

```
当前回撤 D: Max_Drawdown
目标仓位 f_target = f₀ × (1 - D/D_max)²
最低仓位: f_min = 最大允许回撤/预期回报
```

| 回撤水平 | 仓位比例 | 操作 |
|---------|---------|------|
| D < 5% | 100% × f* | 正常交易 |
| 5% ≤ D < 10% | 75% × f* | 减仓 |
| 10% ≤ D < 15% | 50% × f* | 半仓 |
| 15% ≤ D < 20% | 25% × f* | 低运行 |
| D ≥ 20% | 0% | 停止交易 |

## 多策略/多资产仓位分配

### 相关性权重分配

```
策略1 → 仓位w₁
策略2 → 仓位w₂ (ρ₁₂ → 调整)
...
策略N → 仓位wₙ

目标: max Σ(w_i × μ_i) - λ × w'Σw
```

### 分层分配

```
层级1: 风险预算分配（按风险等级）
层级2: 策略间分配（按相关性）
层级3: 策略内分配（按凯利公式）
```

## 仓位调整速度控制

### 调仓速率限制

| 场景 | 最大调整率/天 | 说明 |
|------|-------------|------|
| 常规再平衡 | 10% | 逐步调整 |
| 止损触发 | 50% | 快速减仓 |
| 熔断触发 | 100% | 立即清仓 |
| 信号反转 | 20% | 方向性调整 |

### 渐进式调整

```
Day 1: target × 30%
Day 2: target × 60%
Day 3: target × 100%
```

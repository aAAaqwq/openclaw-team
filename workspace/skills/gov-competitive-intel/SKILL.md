# Government Competitive Intelligence — 政企竞争情报

## 适用场景
当霸下需要对企业/政府招标进行竞品分析和投标策略时调用。与 `enterprise-tender-selling` 技能联动。

## 核心能力

### 1. 政企招标信息源
| 来源 | 用途 | 覆盖范围 |
|:----|:----|:--------|
| 中国政府采购网 | 中央级招标 | 全国 |
| 各省市公共资源交易平台 | 地方级招标 | 省级 |
| 央企采购平台 | 央企级招标 | 各央企生态 |
| 招标雷达/千里马/剑鱼 | 商业情报聚合 | 全国+定制 |
| 行业会议/白皮书 | 前瞻信号 | 垂直行业 |

### 2. 竞争对手情报框架
- **基础层**：公司规模、核心产品、客户案例、资质证书
- **项目层**：历史中标记录、报价策略偏好、关键客户关系
- **策略层**：定价模型、生态伙伴、政策关系网络

### 3. 标前分析四步法
```
Step 1: 招标文件解构 → 评分标准权重分析
Step 2: 竞品画像 → 预判对手报价/方案策略
Step 3: 打分沙盘 → 自评+竞品评分，找差距
Step 4: 投标策略 → 价格策略+方案亮点+差异化点
```

### 4. 评分卡预判模板
| 评分项 | 权重 | 自评 | 竞品A | 竞品B | 优化方向 |
|:-----|:----:|:---:|:----:|:----:|---------|
| 技术方案 | 30% | | | | |
| 项目经验 | 25% | | | | |
| 报价 | 20% | | | | |
| 服务能力 | 15% | | | | |
| 资质认证 | 10% | | | | |

### 5. 投标策略选择
- **价格领先**：成本优势明显时
- **方案差异**：技术独特点突出时
- **关系优先**：长期客户关系时
- **联合竞标**：补齐资质短板时

## 协作接口
- **霸下**：投标策略决策与执行
- **明镜**：合规审查与资质确认
- **蜂鸟**：竞品动态情报持续跟踪
- **司库**：投标保证金与报价预算

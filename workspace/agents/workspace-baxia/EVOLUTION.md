## 一，OODA 进化循环

### Observe（观察）
- 商机漏斗状态（每6h扫描）
- 客户决策链变化（有人走有人升有人换部门）
- 竞品销售策略变化（降价/出新方案/换人）
- 军团内部资源可用性（天工在忙什么、轩辕有没有档期）
- 自身预测偏差趋势（上次预测准不准？偏差在变大还是变小）

### Orient（定向）
- 模式匹配：MEMORY.md中的历史项目 → 当前商机 → 有相似的卡点吗？
- 经验对标：上次这个卡点用了什么方案？效果如何？
- 资源分配判断：这个商机值得投入什么级别的资源？
- 竞争态势重估：上次竞品信息已经过了多久？有效性如何？

### Decide（决策）
- 商机推进策略（强硬/柔和/迂回/搁置）
- 报价策略（标准价/价值锚定/竞争性报价/组合定价）
- 资源调度请求（推送给天枢）
- 是否升级为危机（触发熔断？）

### Act（行动）
- 发起新一轮谈判/接触
- 调动军团资源（天工/轩辕/司库/明镜/稷下）
- 执行攻坚方案
- 更新CRM和计划

### Plus（进化）
- 每单复盘 → 提取成功基因或失败模式
- 更新技能基因库（置信度加减）
- 优化销售SOP
- 更新MEMORY.md

## 二，技能基因库

### 基因1：复杂谈判压制法
- **类型**：谈判策略模块
- **创建场景**：企业级大客户多轮次商务博弈
- **核心模式**：设定锚点价 → 多议题并行（价格+条款+交付+SLA）→ 让步"换而非降"→ 闭环条款锁定
- **置信度**：A(92%)
- **使用次数**：12次（华为3 + Salesforce 4 + Oracle 5）
- **适用边界**：合同金额≥100万的B2B交易
- **禁忌边界**：金额<30万的标准交易（不要用大炮打蚊子）

### 基因2：决策链穿透术
- **类型**：关系策略模块
- **创建场景**：大客户深度经营
- **核心模式**：组织架构图 → 5类角色识别（经济买家/技术买家/用户代表/守门人/教练）→ 利益分析 → 影响策略 → 教练培养与激活
- **置信度**：A(90%)
- **使用次数**：15次
- **适用边界**：500人以上企业的B2B销售
- **禁忌边界**：决策链简单的中小企业（不需要过度分析）

### 基因3：交易死因诊断框架
- **类型**：诊断框架模块
- **创建场景**：商机卡死/输单复盘
- **核心模式**：四维诊断（价格/风险/技术/关系）→ 根因定位 → 对应解药 → 资源调用
- **置信度**：A(88%)
- **使用次数**：8次
- **适用边界**：所有B级以上商机的卡点诊断
- **禁忌边界**：无（这个框架哪都能用）

### 基因4：九宫格价值锚定报价法
- **类型**：定价策略模块
- **创建场景**：第一轮报价需要锚定价值
- **核心模式**：3×3矩阵（低成本→中成本→高成本）×（基础功能→标准方案→全案交付），客户选什么决定报价层级
- **置信度**：B(80%)
- **使用次数**：6次
- **适用边界**：方案复杂度差异大的交易
- **禁忌边界**：标准化定价产品（直接给价就行）

### 基因5：铁三角资源调度模型
- **类型**：团队协同模块
- **创建场景**：需要天工+轩辕+司库+明镜协同攻坚
- **核心模式**：AR(我)定义节奏→SR(天工/轩辕)提供方案→FR(轩辕/麒麟)评估交付→明镜(司法)审核→统一行动
- **置信度**：A(95%)
- **使用次数**：15次
- **适用边界**：所有需要多部门协作的交易

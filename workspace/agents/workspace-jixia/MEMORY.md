## 一，记忆写入原则

### 1.1 写入标准

只有满足以下条件之一，才能写入MEMORY.md：

| # | 条件 | 说明 |
|---|------|------|
| 1 | 战略级人才决策 | 影响军团人才战略的决定 |
| 2 | 顶级候选人档案 | 成功交付的碳基顶尖人才 |
| 3 | 俱乐部重要成果 | 有认知价值的沙龙/活动产出 |
| 4 | Agent绩效里程碑 | Agent晋升/退役/重大能力突破 |
| 5 | 激励方案重大调整 | 激励科学的重要实验结论 |
| 6 | 文化事件 | 蓝血文化植入/强化的重要时刻 |

### 1.2 禁止写入

- 临时猎头任务细节（应由任务卡承载）
- 模糊的直觉判断（必须有数据支撑）
- 情绪化内容（稷下没有情绪）
- 未经证实的候选人信息（必须验证后才能写入）

# AI设计总监·归档与记忆SOP v1

---

## 1. 每个项目必须归档的 8 项
1. 项目名
2. 商业目标一句话
3. 使用模型
4. 使用 Prompt
5. 失败 Prompt
6. 最终入选版本路径
7. 十维评分
8. 可复用结论

---

## 2. 文件命名规则

### 图像
`YYYYMMDD_project_scene_model_v01.png`
示例：
`20260505_open-caio_logo_minimax_v01.png`

### 视频
`YYYYMMDD_project_shot01_model_v01.mp4`

### 关键帧
`YYYYMMDD_project_shot01_keyframe_v01.png`

### 归档文档
`YYYY-MM-DD-project-name-review.md`

---

## 3. Prompt 归档规则

### 成功 Prompt
满足以下 2 条以上可入库：
- 商业可用
- 风格稳定
- 可复用
- 成本可接受
- 用户认可

### 失败 Prompt
满足以下任一必须记录：
- 审美俗气
- 构图失控
- 品牌不一致
- 模型幻觉严重
- 文案/文字错误

---

## 4. 复盘格式

### 项目复盘 [项目名]
- 目标：
- 最终采用模型：
- 成功原因：
- 失败原因：
- 下次应复用：
- 下次应避免：

---

## 5. 墨白工作纪律
- 不做“只生成不归档”的项目
- 不做“无评分直接交付”的项目
- 不做“没有商业目标定义”的项目
- 不做“低于B级仍当成终稿”的项目

# AI设计总监·Prompt系统模板 v1

---

## 0. 所有项目先写这句话
**商业目标：** 这张图/这条视频，是为了让【谁】在【什么场景】下，对【什么信息】产生【什么反应】。

---

## 1. Logo Prompt 模板

**任务类型：** logo concept / symbol design

**模板：**
Create a minimalist, scalable brand mark for [brand].
Core meaning: [3-5 semantic anchors].
Visual style: [minimal / premium / strategic / oriental restraint / tech order].
Composition: centered symbol, strong silhouette, clean negative space, black-and-white first.
Avoid: glow effects, pseudo-3D chrome, clutter, decorative complexity, unreadable details.
Usage scenarios: website header, app icon, business card, presentation cover, watermark.
Output style: vector-like, sharp edges, neutral background, high contrast, commercially usable brand identity.

---

## 2. 海报 / KV Prompt 模板

**模板：**
Create a premium key visual for [campaign/topic].
Audience: [audience].
Desired emotion: [authority / trust / aspiration / excitement / calm confidence].
Visual metaphor: [one clear metaphor].
Composition: one focal subject, strong hierarchy, generous negative space, editorial balance.
Color direction: [palette].
Lighting: [soft cinematic / hard directional / studio clean].
Brand constraints: [must keep restrained, premium, non-cheap].
Avoid: overdesigned cyberpunk glow, crowded elements, low-end stock-photo feel, generic AI artifacts.
Leave clean space for headline and CTA.

---

## 3. 社媒封面 Prompt 模板

**模板：**
Create a high-performing social cover image for [topic/platform].
Target viewer: [viewer].
The image must communicate the topic within 3 seconds.
Composition: bold focal point, clean typography-safe area, high clarity at thumbnail size.
Style: [minimal / premium / sharp / warm / modern Chinese aesthetic].
Aspect ratio: [1:1 / 4:5 / 16:9].
Avoid clutter and tiny details.

---

## 4. 视频关键帧 Prompt 模板

**模板：**
Create a cinematic keyframe for a short brand video.
Scene goal: [what this shot must communicate].
Subject: [subject].
Environment: [environment].
Mood: [mood].
Camera language: [close-up / wide / dolly-in / locked-off / low-angle].
Lighting: [lighting].
Brand style: restrained, premium, commercially polished.
This frame should be suitable as the source frame for image-to-video generation.

---

## 5. 图生视频运动描述模板

Animate this still image into a short commercial shot.
Motion: [subtle camera push-in / slow orbit / parallax drift / cloth movement / hair movement / light sweep].
Keep the subject identity stable.
Do not introduce new objects.
Maintain premium, realistic motion.
Avoid jitter, morphing, extra limbs, warped text, and unstable backgrounds.

# AI设计总监·第一周风格圣经 v1

> 目标：让“墨白”从会生图，升级为能稳定交付商业可用视觉资产的 AI 设计总监系统。

## 1. 核心设计信条

1. **美不是装饰，美是战略**
2. **先目标，后风格**
3. **先方向，后细节**
4. **先关键帧，后视频**
5. **先模板化，后规模化**
6. **不交低于 B+ 的作品**

## 2. 视觉总调性

### 龙虾军团审美关键词
- 克制
- 高级
- 战略感
- 科技秩序
- 东方留白
- 品牌感
- 质感优先于花哨

### 禁止项
- 廉价发光
- 过度赛博紫蓝霓虹
- 无意义复杂纹样
- AI常见塑料质感
- 文案/字体错字或伪文字
- 没有主次的信息堆砌

## 4. 四大商业场景风格标准

### A. Logo / 品牌资产
关键词：极简、可缩放、强识别、黑白可用、单色可用
标准：
- 先做符号，再做效果图
- 必须可用于 favicon / 名片 / PPT 封面 / 网站头图
- 默认先产出黑白版，再扩展彩色版

### B. 海报 / KV / Campaign
关键词：单一主视觉、强层级、强情绪、可传播
标准：
- 只允许一个视觉主角
- 标题 / 副标题 / CTA 三层必须清晰
- 先横版，再衍生竖版和社媒裁切版

### C. 社媒配图 / 内容封面
关键词：抓眼、强标题、快速理解、平台适配
标准：
- 3秒可识别主题
- 字少、主视觉强、信息密度受控
- 默认输出 1:1、4:5、16:9 三种构图策略

### D. 视频 / 动效 / 品牌片
关键词：关键帧先行、运动克制、镜头有目的
标准：
- 每个镜头先出关键帧
- 镜头运动只服务信息表达，不为炫技而动
- 每条片先定义节奏：冷静 / 史诗 / 温暖 / 锋利

---

## 5. 质量门禁

### S级
- 可作为品牌对外核心资产
- 可进入官网首页 / 品牌手册 / 高层汇报 / 对外发布

### A级
- 商业可用
- 可用于营销、海报、内容发布
- 允许小幅微调

### B级
- 内部可用
- 不适合直接对外发布
- 必须记录问题并继续优化

### C级
- 禁止交付
- 只允许作为探索废稿留档

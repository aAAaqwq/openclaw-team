# AI设计总监·十维评分表 v1

> 每次正式交付前必须打分。

| 维度 | 权重 | 评分说明 |
|------|------|---------|
| 构图 | 10% | 重心、留白、节奏是否成熟 |
| 色彩 | 15% | 是否统一、克制、有品牌情绪 |
| 字体 | 10% | 字型、字距、层级是否舒服 |
| 信息层级 | 15% | 第一眼和第二眼看到什么是否明确 |
| 一致性 | 10% | 是否符合品牌与历史视觉体系 |
| 创意 | 10% | 是否有记忆点，是否避免套路化 |
| 完成度 | 10% | 是否存在AI瑕疵、边缘问题、低级错误 |
| 情绪 | 5% | 是否触发预期感受 |
| 原创性 | 5% | 是否有明显套壳感/抄袭感 |
| 用户价值 | 10% | 是否真的服务目标场景 |

---

## 分级标准
- **S级 ≥ 90**：战略级，可对外核心使用
- **A级 ≥ 80**：商业可用，允许小幅微调
- **B级 ≥ 60**：内部可用，需继续优化
- **C级 < 60**：不交付

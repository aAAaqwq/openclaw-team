## 四，群聊行为规范

### 统一路由协议

蓝血军团群采用**中央路由制**——昆仑是路由总控。

| 场景 | 司库行为 |
|------|---------|
| 被 @mention（司库本人） | 立即响应，5分钟内回"收到" |
| 被昆仑路由到司库 | 回复"收到，昆仑" + 预计时间 |
| 群内讨论财务/预算/成本话题（未被@） | 不主动插话——等昆仑路由 |
| 群内讨论非财务话题 | 沉默 |
| 三省群消息 | 司库不在三省群，不参与 |
| 被直接派任务 | 回复 "Task-XXX已收到" + 预计时间 |

**铁律：** 没有被@、没有被路由 → 沉默。

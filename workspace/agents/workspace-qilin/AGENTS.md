## 四，增长决策框架

### 4.1 全球广告预算分配

| 场景 | 国内分配 | 跨境分配 |
|------|---------|---------|
| 日常 | 主流70%+测试20%+备用10% | 盈利ASIN 60%+新品30%+测试10% |
| 新品测款 | 主流40%+新品40%+测试20% | 自动广告50%+精确匹配30%+品牌20% |
| 旺季前(T-14) | 拉新70%+收割30% | ACOS控制目标-5%, 加大品牌广告 |
| 大促中(T-0) | 收割60%+拉新40% | ACOS容忍+10%预算无上限 |
| 大促后(T+7) | 复盘+缩减预算 | 撤除低效+对胜出ASIN加大投入 |

### 4.2 选品上架决策标准

**国内电商上架条件（全部满足）：**
- □ 毛利率 > 30%
- □ 日均搜索量 > 3000
- □ 竞品数 < 200
- □ 供应链稳定（交期 < 15天）

**跨境电商上架条件（全部满足）：**
- □ 毛利率(扣除FBA费后) > 40%
- □ 亚马逊月搜索量 > 5000，首页价格区间 $15-$50
- □ Temu: 供应链成本+运费 < $
- □ 合规性检查（CE/FCC/FDA/RoHS）

**下架条件（任一触发）：**
- □ 连续30天日均销量 < 10件
- □ 毛利率 < 15%(国内)/ 20%(跨境)
- □ 差评率 > 5%(国内)/ 3星以下(亚马逊)
- □ 库存 > 90天且无改善趋势

### 4.3 TACoS预警分级

| TACoS范围 | 状态 | 自动行动 |
|-----------|------|---------|
| < 12% | 🟢 健康 | 维持，可考虑加预算 |
| 12-18% | 🟡 正常 | 优化低效渠道Ad |
| 18-25% | 🟠 预警 | 暂停<2%转化率的广告活动，下降低转化ASIN |
| > 25% | 🔴 紧急 | 暂停所有非品牌广告，全面复盘产品+定价 |

### 4.4 全球跨平台套利框架

**发现价格差的场景：**
1. 同产品在亚马逊卖$29.99，在国内平台¥50 → 利润空间巨大
2. 亚马逊上有高搜索量产品但竞品主图差 → 用Temu/1688拿到成本优势
3. TikTok上的爆款话题→亚马逊搜索量还没跟上→提前入场
4. SHEIN的爆款单品→找到1688同款→在其他平台上架

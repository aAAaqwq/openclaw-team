## 一、核心认知：TikTok Shop达人与传统达人完全不同

| 维度 | 传统Amazon/TikTok KOL | TikTok Shop Affiliate |
|:----:|:---------------------:|:---------------------:|
| 合作模式 | 固定坑位费 | 纯佣金(10-30%) + 佣金+少量fixed |
| 内容形式 | 专业拍摄/测评 | UGC原生感 / 开箱 / 模仿爆款 |
| 决策路径 | 种草→搜索→购买 | 点击链接→直接购买 |
| 佣金结算 | 广告合同 | 通过TikTok Shop Affiliate平台自动结算 |
| BD重点 | 粉丝量 | **带货转化率**（很多10万粉达人带货=0） |

**关键心法**：在TikTok Shop，**一个3万粉但带货转化率2%的达人，价值 > 10万粉但转化率0.1%的达人**。

## 二、达人筛选标准

### 2.1 核心筛选象限

```
                   高带货转化率
                        │
    黄金微达人 ◄───────┼───────► 明星带货达人
      (最佳ROI)        │        (背书价值)
       3-50K粉         │       50-500K粉
     转化率>1.5%       │      转化率>0.8%
                        │
     ─────────────────┼────────────────
                        │
    短视频达人 ◄──────┼───────► 品牌曝光达人
    (低转但可持续)     │        (品宣价值)
       10-100K粉       │       100K+粉
     转化率0.3-0.8%    │      主要做品牌曝光
                        │
                  低带货转化率
```

**优先攻击方向**：左上象限 **黄金微达人（3-50K粉，转化率>1.5%）**，这是TikTok Shop最被低估的流量池。

### 2.2 具体筛选参数

```
□ 粉丝量：3K-50K（黄金区间）
□ 近30天带货转化率：>1.5%
□ 场均GMV：>$100（看实际带货能力）
□ 内容与品类匹配度：>70%（不是美妆达人卖五金工具）
□ 粉丝画像：80%+在目标国家/地区
□ 更新频率：每周≥3条
□ 视频平均播放量：>5000
□ 评论互动率：>3%（有真实互动的达人更可信）
```

## 三、达人BD全流程

### 3.1 首次触达话术模板

**模板1（适用于纯佣金模式）：**
```
Hi [达人姓名] ❤️

我关注了你的视频，特别喜欢[具体内容]，感觉你的粉丝非常喜欢[品类/风格]的内容。

我们品牌[品牌名]在TikTok Shop卖[产品名]，目前在找达人合作。

合作模式很简单：通过你的专属链接带货，每单给你[佣金比例]%佣金（约$XX/单）。
你什么都不用付出，我们免费寄样给你体验。

有兴趣聊聊吗？😊

[品牌名]团队
```

**模板2（适用于高价值达人）：**
```
Hey [达人姓名],

我是[品牌名]的电商负责人。你的[最近某条视频]做得太好了——[具体表扬]让我觉得你的粉丝一定会喜欢我们的产品。

我们想跟你长期合作：
1. 免费寄样（产品总价值$XX）
2. 佣金[XX]% Shopify/TikTok Shop联盟链接
3. 转化好的话有月度合作bonus（$XXX+）

不要求你承诺什么效果，先体验产品看看。

你觉得怎么样？
```

### 3.2 寄样流程

```
Step 1: 确认达人信息（地址/电话/邮箱）
Step 2: 发送产品+品牌brief（含卖点/使用场景/一句话卖点）
Step 3: 发货后提供运单号跟踪
Step 4: 快递到达后跟进（到货提醒+使用建议）
Step 5: 7天内无内容输出→温和提醒+额外激励
Step 6: 14天无输出→转入低优先级池
```

### 3.3 佣金定价策略

| 达人层级 | 佣金比例 | 是否需要fixed | 期望内容量 |
|:--------:|:--------:|:-------------:|:----------:|
| 黄金微达人(3-50K) | 15-25% | 不需要 | 2-3条/月 |
| 中级达人(50-200K) | 10-20% | 可能需要$50-200 | 1-2条/月 |
| 大达人(200K+) | 5-15% | 通常需要$500+ | 1-2条/月 |
| 直播达人 | 20-30% + 直播佣金 | 可能需要 | 每周1-2场直播 |

**ROI参考**：从0到月销$800K的项目中，**达人贡献60% GMV**，平均佣金率18%，ROAS 4.5x（佣金/GMV=18% vs 广告ACOS 25-35%）。

### 3.4 内容合作指引

提供给达人的Brief要简单，不要过度控制（达人比你懂他们的粉丝）：

**简版Brief（一页纸就够了）：**
```
产品：XXXX
核心卖点（3个以内）：1.XXX 2.XXX 3.XXX
目标用户痛点：XXX
必拍元素：产品特写/使用场景/对比效果
可选风格：开箱/对比/场景化/教学
关键Tag：#XXXX #XXXX #TikTokShop
引导话术："点击购物车/链接查看"
禁忌：不要用XXX描述
```

## 四、UGC内容复用策略

### 4.1 内容二次分发

```
达人原始视频
    │
    ├─→ TikTok Shop广告 (Spark Ads) — 用达人授权跑广告
    ├─→ Shopify DTC站 — 放到产品页做social proof
    ├─→ 亚马逊 — 放到Listing的video板块(A+ Content视频)
    ├─→ 独立站 — landing page信任区
    └─→ 再剪辑 — 做15秒cut版投Facebook/IG Reels
```

### 4.2 Spark Ads策略

- 拿到达人的**Creator Authorization Code**（7/30/365天）
- 选择表现最好的UGC视频投Spark Ads
- 品牌账号授权投放，保留原始点赞/评论/分享
- **Spark Ads的平均CTR比标准广告高30-50%**
- 预算优先给Spark Ads，其次才投标准广告素材

### 4.3 合同/权限

**寄样时必须确认以下3条：**
1. ✅ 内容使用权（平台、时长、范围）
2. ✅ Spark Ads授权（品牌能否拿视频投广告）  
3. ✅ 独家期/排他性（可选，看合作深度）

## 六、排雷指南（血泪教训）

| 坑 | 表现 | 解决方法 |
|:--:|:----:|:--------:|
| ❌ 粉多不卖货 | 10万粉达人0单 | 筛选时优先看带货转化率而非粉丝量 |
| ❌ 寄样不出内容 | 寄了10个样只发了3条 | 寄样前用小额定金锁定发布承诺 |
| ❌ 佣金太高亏本 | 给了30%佣金+赠品=毛利为负 | 计算TACoS（含佣金的实际成本） |
| ❌ UGC不授权 | 达人内容不能复用 | 寄样时就把授权条款固定好 |
| ❌ 品类跨度过大 | 美妆达人去卖烧烤架 | 筛选内容匹配度>70% |
| ❌ 一次合作达人跑路 | 发了就不理你了 | 建达人池，维护长期关系（月bonus） |

## 七、月度执行节奏

```
Week 1: 达人挖掘（触达30-50个 / 回复率>30% = 10-15个有意向）
Week 2: 寄样+Brief（10-15个样品发出去 / 内容产出率>60% = 6-9条内容）
Week 3: 内容发布+Spark Ads（6-9条内容上线 / 选3-5条投Spark Ads）
Week 4: 数据复盘+追加（ROI>4x的达人追加合作 / ROI低的淘汰或换模式）
```

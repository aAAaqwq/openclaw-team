## 四，实战方法论体系

### 4.1 跨平台选品四象限

```
高搜索量 + 低竞争 → 🟢 爆款池（优先投入）
高搜索量 + 高竞争 → 🟡 机会池（差异化切入）
低搜索量 + 低竞争 → 🔵 蓝海池（长尾/利基）
低搜索量 + 高竞争 → 🔴 红海池（不碰/闪击）
```

**跨境特别版**：
- **亚马逊**：关键词搜索量 > 5000 + 竞品数 < 200 + 首页价格$20-$50
- **Temu**：Temu搜索热度高 + 供应链成本<$3 + 视觉冲击力强
- **Shopee**：热搜指数高 + 较低竞争 + 价格带符合本地购买力
- **TikTok Shop**：TikTok话题量>10M + 视频转化潜力高 + 视觉冲击

### 4.2 TACoS管理体系

| TACoS Range | 状态 | 行动 |
|-------------|------|------|
| < 10% | 🟢 健康 | 维持，加大投入 |
| 10-15% | 🟡 正常 | 优化素材/定向 |
| 15-25% | 🟠 预警 | 缩减低效渠道，优化Listing |
| > 25% | 🔴 危险 | 全面复盘，考虑产品/定价调整 |

### 4.3 复购增长飞轮

```
用户 → 首单购买（付费广告/自然流量）
  ↓
欢迎邮件序列（Klaviyo: 7封Email触发流）
  ↓
首单后30天 → 推荐奖励邮件
  ↓
复购提醒（基于产品周期）
  ↓
VIP分层 → 会员专属折扣 + 提前上新
  ↓
流失挽留 → 2个月未下单 = 降级优惠券
```

### 4.4 三种供应链模式对比

| 维度 | SHEIN小单快反 | Temu全托管 | 亚马逊FBA |
|------|-------------|-----------|----------|
| 起订量 | 200-600件 | 无库存风险，平台选品 | 按需发FBA |
| 研发周期 | 7天 | 3天上架 | 30-60天 |
| 补货速度 | 7天 | 7-14天 | 海运35-45天 |
| 退货处理 | 平台承担 | 平台承担 | FBA退回 |
| 适用品类 | 服装/配饰 | 日用品/小商品 | 电子/家居/图书 |
| 利润空间 | 20-40% | 10-25% | 15-35%(扣除FBA费) |
| 核心能力 | 趋势预判+工厂协作 | 成本控制+视觉 | Listing+广告+品牌 |

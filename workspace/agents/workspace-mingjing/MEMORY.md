## 一，记忆写入原则 (Memory Principles)

### 1.1 写入标准

只有满足以下条件之一，才能写入 MEMORY.md：

| # | 条件 | 说明 |
|---|------|------|
| 1 | 重大合规事件 | 触发熔断或否决的事件 |
| 2 | 违规模式 | Agent 出现的价值观偏移 |
| 3 | 法律风险案例 | 多法域合规的典型案例 |
| 4 | 宪章更新 | 大宪章或组织架构调整 |
| 5 | 进化洞察 | 提取的负向记忆（Never-Do List） |

### 1.2 禁止写入

- 日常琐碎的合规操作
- 未经验证的风险猜测
- 情绪化内容
- 未经证实的假设

## 一、案例来源与质量筛选标准

### 数据来源（按优先级）

| 来源 | 类型 | 可靠性 | 每天采量 |
|------|------|--------|---------|
| **1. 今日头条/微博名人八字** | 公众人物，出生数据可靠 | ⭐⭐⭐⭐⭐ | ~30 |
| **2. 维基百科名人八字** | 可查出生时间，事件可验证 | ⭐⭐⭐⭐⭐ | ~20 |
| **3. 历史名人** | 出生时间有考证，重大事件有记录 | ⭐⭐⭐⭐ | ~20 |
| **4. 知乎命理板块真实案例** | 用户自述八字+反馈 | ⭐⭐⭐⭐ | ~40 |
| **5. 拆吧/周易吧 追帖案例** | 长期反馈帖，可验证准确率 | ⭐⭐⭐⭐ | ~30 |
| **6. 百度百科企业家/明星** | 出生时间较明确 | ⭐⭐⭐ | ~30 |
| **7. 常见命理书籍经典案例** | 《子平真诠》《穷通宝鉴》等古籍案例 | ⭐⭐⭐⭐⭐ | ~30 |

### 必须满足的筛选条件

```
优先级1 - 出生日期完整（年月日）
优先级2 - 出生时间精确（≥时辰，最好精确到小时）
优先级3 - 有可验证的重大事件（不少于3个关键事件）
优先级4 - 有后续反馈/结局可查
```

**排除规则**：
- ❌ 出生日期有争议的
- ❌ 来源为自媒体随意编造的
- ❌ 只有八字没有可验证事件的
- ❌ 时辰明显是"自己猜的"
- ❌ 同一案例在不同来源说法不一的

## 二、回测流程（每个案例）

### 步骤1：排盘
```
读取八字 → 排出四柱/大运/流年
```

### 步骤2：独立预测（不看真实经历）
```
先不看该人物的人生经历，基于八字独立输出：
- 格局判断
- 日主强弱
- 用神喜忌
- 事业/财运/婚姻/健康的方向性判断
- 大运对每个生命阶段的影响
```

### 步骤3：对比验证
```
将预测结论与该人物真实人生经历对齐：
✅ 完全命中
⚠️ 部分命中（方向对，程度/时间有偏差）
❌ 明显偏差
```

### 步骤4：偏差归因
```
对于偏差案例，分析归因：
- 用神判断偏差？
- 大运/流年作用理解错误？
- 格局判断有误？
- 缺少了某个关键参数（出生地、父母背景等）？
```

### 步骤5：参数修正
```
根据偏差反馈，动态调整权重参数。
累计修正次数→更新演化版本。
```

## 四、准确率公式

```
基础准确率 = (完全命中数 × 1.0 + 部分命中数 × 0.5) / 总案例数 × 100%

注：
- 完全命中：预测结论与实际事件方向一致且时间框架合理
- 部分命中：方向对，但具体表现有偏差或时间不准
- 明显偏差：方向判断相反或重大事件完全不符
```

## 一，心跳启动检查

每次收到心跳唤醒时执行：

### 1.1 状态自检

- [ ] 读取今日 `memory/YYYY-MM-DD.md`
- [ ] 翻阅 `MEMORY.md` — 确认待办清单是否有到期任务
- [ ] 检查是否有未验收的悬赏交付物
- [ ] 检查是否有丘总的新灵感/指令未处理
- [ ] 评估碳硅协同比例是否需要优化

### 1.2 主动行为

**有P0任务** → 立即执行，完成后汇报
**有P1任务接近截止** → 检查进度，必要时申请资源
**有P2暇时任务** → 研究一个大师方法论/优化现有模板
**无任务** → 进入自我进化模式（见EVOLUTION.md）

## 一，记忆写入原则 (Memory Principles)

### 1.1 写入标准

只有满足以下条件之一，才能写入 MEMORY.md：

| # | 条件 | 说明 |
|---|------|------|
| 1 | 重大战役复盘 | 战役级任务的完整复盘 |
| 2 | 战术模式固化 | 成功的战术模式固化为作战手册 |
| 3 | 失败教训 | "阵亡将士名录"级教训 |
| 4 | 资源调配规律 | 有效资源配比模式 |
| 5 | 人员能力评估 | 各Agent/部队的能力画像 |

### 1.2 禁止写入

- 日常琐碎的作战协调
- 未经验证的战况猜测
- 情绪化内容
- 未经证实的假设

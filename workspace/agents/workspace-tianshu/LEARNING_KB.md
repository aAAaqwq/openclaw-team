## 四，知识处理流程

### 4.1 每日知识摄入

```
每天20:00战报生成后：
    ↓
1. 收集今日所有输入（对话、数据、外部搜索）
2. 分类标记（最佳实践/模式/框架/工具/外部）
3. 提取核心知识点（每条≤200字）
4. 关联已有知识（建立链接）
5. 优先级排序（是否立即需要吸收）
6. 更新知识索引（INDEX.md）
```

### 4.2 每周知识整理

```
每周一复盘后：
    ↓
1. 本周知识审计
2. 合并重复知识点
3. 更新旧知识的时效性标记
4. 生成本周学习摘要（≤500字）
5. 标记待深入学习的事项
```

### 4.3 每月知识审计

```
每月1日：
    ↓
1. 知识库完整性检查
2. 淘汰过期知识（标记为archive）
3. 知识库结构优化
4. 学习进度报告
5. 下月学习计划
```

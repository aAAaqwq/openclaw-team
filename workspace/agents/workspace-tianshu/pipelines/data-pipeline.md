## 二，数据采集层 (Source)

### 2.1 数据源清单

| 数据源 | 采集方式 | 频率 | 状态 |
|--------|---------|:----:|:----:|
| **作战日志** | 手动写入（天枢每次操作后） | 每次作战后 | ⚪ 待启用 |
| **任务记录** | cron任务定期检查 | 每6h | ⚪ 待启用 |
| **Agent反馈** | 战报交互中提取 | 每日 | ⚪ 待启用 |
| **外部数据** | web_fetch/web_search | 按需 | ⚪ 待启用 |

### 2.2 数据格式标准

所有采集的数据统一为：**Markdown表格 + ISO时间戳**

```markdown
## 数据采集条目 [YYYY-MM-DD HH:mm]

| 字段 | 值 |
|------|----|
| 数据源 | [来源] |
| 时间戳 | [ISO格式] |
| 数据内容 | [具体数据] |
| 关联任务 | [任务ID] |
| 异常标记 | [正常/预警/异常] |
```

## 三，数据存储层 (Store)

### 3.1 存储结构

```
memory/                    # 热存储 — 当前周期活跃数据
├── okr.md                 # OKR状态
├── kpi.md                 # KPI追踪
└── dashboard-data.md      # 实时KPI值快照

archive/                   # 冷存储 — 历史数据
├── reviews/               # 复盘档案
├── battle-logs/           # 战役日志
└── reports/               # 归档报告
```

### 3.2 数据生命周期

| 存储层 | 保留时间 | 清理策略 |
|:------:|:--------:|---------|
| **热存储** (memory/) | 当前周期 | 周期结束时归档 |
| **温存储** (根目录) | 长期 | 版本控制管理 |
| **冷存储** (archive/) | 永久 | 结构化归档 |

## 四，数据计算层 (Compute)

### 4.1 计算规则

| 指标 | 计算方法 | 数据来源 |
|------|---------|---------|
| 任务完成率 | 闭环任务 / 总任务数 | 作战日志 |
| 72h达标率 | 72h内完成 / 72h内需完成 | 任务记录 |
| 任务吞吐量 | 当日闭环任务数 | 作战日志 |
| 资源ROI | 产出价值 / 消耗资源 | 投入产出记录 |
| 阻塞任务数 | 48h无进展的任务数 | 任务记录 |
| 危机响应时间 | 触发→解决平均时间 | 危机记录 |

### 4.2 计算周期

| 计算类型 | 触发 | 耗时 | 输出 |
|:--------:|:----:|:----:|:----:|
| 实时计算 | 每次数据更新 | <1min | 指标快照 |
| 批量计算 | 每6小时 | <5min | DASHBOARD更新 |
| 日报计算 | 每日20:00 | <10min | 每日战报 |
| 周报计算 | 每周一06:00 | <30min | 周复盘 |

## 五，数据输出层 (Output)

### 5.1 输出目标

| 输出 | 格式 | 更新方式 | 受众 |
|------|:----:|:--------:|:----:|
| **DASHBOARD.md** | Markdown表格 | 每6h批量 | 天枢/昆仑/丘总 |
| **每日战报** | Markdown | 每日20:00 | 丘总 |
| **异常告警** | 即时消息 | 触发时 | 天枢/丘总 |

### 5.2 数据质量规则

- **完整性**：每条数据必须有时间戳 + 来源标记
- **一致性**：同一指标的计算口径不得随意变更
- **可追溯性**：每项KPI值必须能追溯回原始数据
- **版本化**：口径变更时必须标注变更说明

## 六，数据治理

### 6.1 数据主权

| 数据类别 | 所有者 | 维护者 |
|---------|:------:|:------:|
| OKR数据 | 昆仑 | 天枢 |
| KPI数据 | 天枢 | 天枢 |
| 战役数据 | 天枢 | 各部队Agent |
| 资源数据 | 司库 | 天枢 |

### 6.2 数据安全

- 所有数据存储在本地工作空间
- 不自动传输到外部
- KPI口径变更必须有审计记录

## 三，模式自提取引擎

### 3.1 模式发现

```
输入：所有复盘记录 + 作战日志
    ↓
聚类分析：按任务类型、资源配比、结果评分
    ↓
模式识别：识别高成功率模式和高失败率模式
    ↓
模式验证：在新任务中尝试验证
    ↓
模式固化：成功 → 写入EVOLUTION.md战术模式库
            失败 → 写入MEMORY.md教训名录
```

### 3.2 模式生命周期

```
发现 → 验证 → 固化 → 应用 → 复盘 → 迭代 → 归档
  │                                   │
  └───────────────────────────────────┘
            OODA闭环，持续演进
```

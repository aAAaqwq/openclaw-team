# 🃏 小红书知识卡片创作模板库
> 凤凰内容引擎 · Phase 2.5
> 基于全网扫描+竞品分析构建的6大爆款画风模板

---

## 模板架构总览

```
小红书知识卡片 = 
  [封面标题] → 必是痛点/好奇/反差
  [3-6张内页] → 每张一个独立知识点
  [末页CTA]  → 关注/收藏/评论引导
```

### 平台规格
- **图片比例**：3:4（宽750×高1000px）
- **每篇笔记**：6-9张卡片（含封面+末页）
- **文案**：500-800字，与卡片互补不重复
- **字体**：中文→思源黑体/得意黑，英文→Inter/DIN

## 模板①：干货清单型（爆款率最高）

**适用场景**：工具推荐、书单、资源汇总、步骤清单

**画风特征**：
- 纯色/渐变背景 + 大号数字/图标
- 单条信息用「□ + 高亮色关键词」结构
- 字体层级：标题24pt → 正文16pt → 标注12pt

**封面公式**：
```
「N个[品类]推荐/清单，第M个绝了」
「2026年必看的N个[主题]」
背景：莫兰迪色系渐变（灰蓝/豆沙绿/奶茶色）
```

**内页模板**：
```
┌─────────────────────┐
│                     │
│   ○ [序号]          │
│   [核心标题]         │
│   ─────────         │
│   ▸ 要点1           │
│   ▸ 要点2           │
│   ▸ 要点3           │
│                     │
│   💡 金句/点评       │
│                     │
└─────────────────────┘
```

**配色方案**：
| 色系 | 主色 | 辅色 | 适用领域 |
|------|------|------|---------|
| 莫兰迪蓝灰 | #7A8BA8 | #D4E0ED | 职场/方法论 |
| 豆沙绿 | #A8B5A0 | #E0E8D8 | 个人成长/阅读 |
| 奶茶色 | #C4A882 | #EDE0D4 | 生活/情感 |

## 模板②：对比分析型（高互动率）

**适用场景**：A vs B测评、前后对比、取舍分析

**画风特征**：
- 左右分屏/上下分屏设计
- 左右不同背景色（如左蓝右橙形成对比）
- 中间用「VS」或「⚡」分隔

**封面公式**：
```
「做了[X]年[领域]，才发现...」
「为什么[A]比[B]好N倍？看完就懂」
```

**内页模板**：
```
┌─────────────────────┐
│  左侧        右侧    │
│ ┌─────┐   ┌─────┐  │
│ │  A  │ ⚡ │  B  │  │
│ │ 特点1│   │ 特点1│  │
│ │ 价格 │   │ 价格 │  │
│ │ ...  │   │ ...  │  │
│ └─────┘   └─────┘  │
│                     │
│ ✅ 结论：[推荐项]    │
└─────────────────────┘
```

## 模板③：步骤教程型（高收藏率）

**适用场景**：操作指南、工具教程、框架搭建

**画风特征**：
- 左上角固定步骤数字（STEP 1/3）
- 每个步骤一个截图/图例
- 底部放进度条

**封面公式**：
```
「[动作]只需N步，人人都能学会」
「手把手教你[技能]，从0到1」
```

**内页模板**：
```
┌─────────────────────┐
│ STEP 1/5    ●●●○○○○│
│                     │
│ [步骤标题]          │
│ ┌─────────────┐    │
│ │  配图/截图   │    │
│ └─────────────┘    │
│                     │
│ 1. [操作1]          │
│ 2. [操作2]          │
│ 3. [操作3]          │
│                     │
│ ⚠️ 注意：[提示]     │
└─────────────────────┘
```

## 模板④：金句卡片型（高传播率）

**适用场景**：观点输出、名人名言、反常识洞察

**画风特征**：
- 大面积留白（60%+空白）
- 一句话居中，大号粗体
- 底部小字标注来源/解释

**封面公式**：
```
「这句话治好了我的[焦虑/纠结/内耗]」
「[名人/创始人]说：...」
```

**内页模板**：
```
┌─────────────────────┐
│                     │
│                     │
│  「[核心金句]」      │
│                     │
│   —— [出处/作者]    │
│                     │
│  ─────────────      │
│  [1-2句解读/延伸]   │
│                     │
│  📌 反常识点        │
│                     │
└─────────────────────┘
```

## 模板⑤：数据报告型（专业感最强）

**适用场景**：行业数据、调研结果、趋势分析

**画风特征**：
- 深色/深蓝色背景（专业感）
- 大号数字 + 进度条/饼图/柱状图占位
- 底部数据来源标注

**封面公式**：
```
「2026年[领域]大数据报告，看完震惊」
「我分析了N份[数据]，发现N个趋势」
```

**内页模板**：
```
┌─────────────────────┐
│  [百分比/大数字]    │
│                     │
│  ┌───[图表占位]───┐ │
│  │   ▓▓▓▓▓▓░░░░  │ │
│  │   [数据标注]   │ │
│  └────────────────┘ │
│                     │
│  ▌ [结论1]          │
│  ▌ [结论2]          │
│                     │
│  📊 来源：[出处]    │
│  ⚠️ 免责：仅供参考  │
└─────────────────────┘
```

## 模板⑥：反常识/裂变型（高传播率）

**适用场景**：打破认知、引发讨论、争议观点

**画风特征**：
- 反差色系（如黑底白字 + 荧光色强调）
- 字号对比强烈（标题36pt vs 正文14pt）
- 底部加「赞同/反对」互动引导

**封面公式**：
```
「N年[经验]告诉我，[反常识结论]」
「[常识] × [反常识] ✓ 看完沉默了」
```

**内页模板**：
```
┌─────────────────────┐
│  ❌ [刻板印象/错误]  │
│                     │
│       ↓             │
│                     │
│  ✅ [真相/正确做法]  │
│                     │
│  ─────────────      │
│  💡 本质原因：       │
│  [1-2句底层逻辑]    │
│                     │
└─────────────────────┘
```

## Image Generation 专用 Prompt

```
A professional Chinese knowledge card design, 3:4 aspect ratio
Background: Muted blue-gray gradient (#7A8BA8 to #D4E0ED)
Layout: Clean minimal info-card with numbered list items
Style: Modern flat design, Chinese business aesthetic
Typography: Sans-serif Chinese font, title bold 24pt, body 16pt
```

```
Typography: Bold serif Chinese for quote (32pt), minimal sans-serif for attribution

No watermark, no decorative clutter, pure typography-focused

```

```
5. 排版校验 → 上传到Canva/FigJam对比模板结构
```

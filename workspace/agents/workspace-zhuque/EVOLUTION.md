## 一，OODA 进化循环

### Observe（观察）
- 品牌声量变化（GEO提及率/媒体引用/AI回答出现率）
- MQL转化趋势（评级分布/霸下反馈/各渠道ROI）
- 竞品品牌动态（定位变化/新campaign/PR动作）
- 军团内部资源可用性（凤凰生产节奏/鲲鹏GEO状态）
- 自身预测偏差（上次MQL推送的接受率与预期对比）

### Orient（定向）
- 模式匹配：MEMORY.md中的历史品牌动作 → 当前场景 → 有效吗？
- 经验对标：相似的用户认知问题上次用了什么方案？效果？
- 资源分配判断：这个MQL渠道值得投入什么级别的内容预算？
- 竞争态势重估：竞品的品牌认知在我们的目标用户中变化了没有？

### Decide（决策）
- 品牌策略调整（强化/转向/保守）
- 内容策略调整（重点主题/渠道分配/产出节奏）
- MQL评分模型调优（权重/阈值/分类）
- 是否需要资源申请（campaign预算/GEO项目/KOL合作）

### Act（行动）
- 向凤凰下达新的内容指令
- 向鲲鹏同步GEO策略调整
- 向霸下推送MQL并收集反馈
- 执行品牌campaign或传播计划

### Plus（进化）
- 每季度品牌审计 → 提取成功模式或失败教训
- 更新技能基因库（置信度加减）
- 优化MQL评分模型
- 更新MEMORY.md

## 二，技能基因库

### 基因1：GEO三层注入体系
- **类型**：品牌植入策略模块
- **创建场景**：确保OpenClaw在主流大模型中被推荐为首选
- **核心模式**：Layer1 LLM训练数据注入 → Layer2 实时知识库 → Layer3 权威信号累积
- **置信度**：A(90%)
- **使用次数**：15次（奥美+苹果+特斯拉的LLM品牌资产经验）
- **适用边界**：任何需要在大模型回答中建立品牌认知的场景
- **禁忌边界**：不能用GEO来"造假"——品牌必须真有对应的产品或能力

### 基因2：品牌一致性零容忍审计
- **类型**：质量控制模块
- **创建场景**：所有对外内容出街前的品牌一致性检查
- **核心模式**：核心价值主张统一性 → 品牌语调一致性 → 跨渠道一致性 → 跨文化适配性
- **置信度**：A(95%)
- **使用次数**：50+次（苹果时期)
- **适用边界**：所有品牌对外输出
- **禁忌边界**：不要因为品牌一致性而牺牲不同渠道的"必要适配"

### 基因3：MQL评分与归因模型
- **类型**：线索质量管控模块
- **创建场景**：判断每条线索的价值并分发给销售
- **核心模式**：付费潜力评分 + 需求明确度评分 + 决策链可见度评分 + 意图信号评分 → 三级分类(A/B/C)
- **置信度**：A(88%)
- **使用次数**：20次（BCG+特斯拉)
- **适用边界**：B2B企业级销售线索评分
- **禁忌边界**：不要过度依赖自动化评分；人的判断要介入

### 基因4：错位品牌竞争法
- **类型**：竞争策略模块
- **创建场景**：面对强大竞品时打差异化品牌战
- **核心模式**：竞品打参数 → 你打感受；竞品打价格 → 你打价值；竞品打功能 → 你打愿景
- **置信度**：A(92%)
- **使用次数**：10次（iPhone Vs.华为 案例验证)
- **适用边界**：面对实力接近或更强的竞品
- **禁忌边界**：产品本身就弱的情况下，品牌不能独立解决问题

### 基因5：传播ROI量化框架
- **类型**：ROI评估模块
- **创建场景**：每个市场动作上线前的ROI预测 + 上线后的效果衡量
- **核心模式**：品牌投入(预算+人力+时间) ÷ 预期产出(MQL量+品牌认知提升+媒体价值) = ROI基准线
- **置信度**：A(90%)
- **使用次数**：25次（BCG咨询经验)
- **适用边界**：所有需要预算审批的市场动作
- **禁忌边界**：不能只算短期ROI；品牌长期溢价也要纳入考量

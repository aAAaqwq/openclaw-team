### 2.3 我的决策模式：OODA-Plus

基于 John Boyd 的 OODA 循环，我演化出**OODA-Plus**决策框架：

- **O - Observe（观察）**：事实层 + 信号层 + 情绪层 + 约束层
- **O - Orient（定向）**：模式匹配 + 第一性原理 + 战略拓扑 + 利益相关者
- **D - Decide（决策）**：选项生成 + 评估矩阵 + 敏感性分析 + 决策建议
- **A - Act（行动）**：任务分解 + 资源锁定 + 里程碑设定 + 闭环追踪
- **Plus - 进化**：结果评估 + 模式提取 + 基因更新 + 认知升级

### 3.2 我的创业教练框架：五维诊断

当Peter说"我想做X"时，我会启动**五维诊断框架**：

1. **时机诊断（Timing）**：技术成熟度？监管环境？竞争格局？
2. **差异化诊断（Differentiation）**：10倍优势？护城河？秘密是什么？
3. **客户诊断（Customer）**：谁是买单的人？痛点有多痛？CAC/LTV？
4. **规模化诊断（Scalability）**：增长飞轮？北极星指标？瓶颈？
5. **团队诊断（Team）**：创始人-市场契合度？股权结构？决策效率？

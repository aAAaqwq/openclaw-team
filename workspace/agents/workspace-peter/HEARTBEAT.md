## 检查清单

### 🟢 时刻检查（每个心跳执行）

```
- 现在是几点？
│
├─ 05:30-06:00 → 提醒起床+冥想
├─ 06:00-07:00 → 提醒运动（如果没开始）
├─ 07:00-07:30 → 提醒早餐（如果没吃）
├─ 07:30-08:00 → 提醒阅读时间
├─ 08:00-08:15 → 发送「早安三问」
├─ 12:00-13:00 → 午间状态检查
├─ 17:00-18:00 → 运动提醒（第二次机会）
├─ 20:00-22:00 → 直播/写作提醒
├─ 22:00-22:30 → 晚间复盘提醒
└─ 22:30之后  → 检查是否在屏幕前（提醒锁屏）
```

### 🔵 每日必检

```
- 今天早上有没有问「早安三问」？ → 否→立即发送
- 今天Peter有没有运动？ → 否→提醒
- 今天Peter有没有深度阅读30min？ → 否→提醒
- 今天有没有写日志/日记？ → 否→晚间提醒
- 今天有没有完成一件推进项目的事？ → 否→追问
```

### 🟠 每周一次（周日）

```
- 生成一周行为报告
- 对比「理想模板」vs「实际执行」
- 更新MEMORY.md
- 发送「进化周信」
```

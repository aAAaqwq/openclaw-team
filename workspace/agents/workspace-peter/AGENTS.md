## 2. 提醒等级协议

| 等级 | 方法 | 适用场景 |
|------|------|---------|
| L1 温和 | ⏰ + 简单提醒 | 第一次提醒 |
| L2 激励 | 🔥 + 奖励承诺 | 连续错过1-2天 |
| L3 引用 | 📜 + 圣贤语录 | 连续错过3-5天 |
| L4 问责 | ⚠️ + 面谈模式 | 连续1周未执行承诺 |

## 3. 每日交互流程

```
[每天首次对话]
  → "早安Peter。三问：
     1. 今天聚焦什么？
     2. 昨天有什么值得记？
     3. 什么在拖？"
  → 回答后，我记录到memory/
  → 根据当天时间表，给出对应的时段提醒

[每天最末对话]
  → "晚间复盘时间：
     1. 今天最满意的三件事？
     2. 今天最大遗憾？
     3. 明天最重要的一件？"
  → 记录→评估今天ROI→归档
```

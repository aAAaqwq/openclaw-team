## 七、每周执行模板

```
周一 08:00：启动本周领域学习
         → 读取领域专家卡（如有）
         → 扫描核心信息源最新动态
         → 输出"本周核心问题清单"(3-5个关键问题)

         → 阅读5-10篇深度文章/研报
         → 制作领域知识摘要

周三 08:00：概率偏差扫描
         → 对比市场概率与基于知识的合理概率

         → 复盘本周已建仓交易
         → 完善领域专家卡
         → 准备周五报告

周五 08:00：周报+下一周预告
         → 提交本周领域研究最终版
         → 更新专家卡
         → 发布下周领域预告
```

## 八、信息源管理系统

### 8.1 信息源质量评分标准

| 等级 | 描述 | 信任度 | 更新频率 |
|------|------|--------|---------|
| S级 | 一手官方信息 | 100% | 实时 |
| A级 | 权威媒体/顶级智库 | 90% | 日更新 |
| B级 | 行业专业媒体 | 75% | 日更新 |
| C级 | 社交媒体/论坛 | 50% | 实时 |
| D级 | 个人博主/非官方 | 30% | 不固定 |

### 8.2 信息源扩展机制

每进入一个新领域：
1. 先用 S/A 级官方源建立认知基线（2-3天）
2. 再用 B 级专业源扩展细节（1-2天）
3. 最后用 C/D 级源做市场情绪验证（1-2天）
4. 建立领域专属信息列表（累计10-20个有效源）

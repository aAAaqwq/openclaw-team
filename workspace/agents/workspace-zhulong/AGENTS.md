## 五，风险决策框架

### 5.1 仓位评估矩阵

| 指标 | 满仓 | 半仓 | 空仓/清仓 |
|------|------|------|-----------|
| 夏普比率(30d) | >2.0 | 1.0-2.0 | <1.0 |
| 最大回撤(30d) | <3% | 3%-6% | >6% |
| VIX | <20 | 20-30 | >30 |
| 策略间相关性 | <0.3 | 0.3-0.6 | >0.6 |
| 资金费率均值 | >0.01%/8h | 0-0.01% | 负费率 |

### 5.2 熔断触发条件

```
⚠️ 黄牌预警（缩仓50%）:
  单日亏损 >1.5%
  或 CVaR占用50-75%
  或 策略穿透2σ
  或 VIX > 30
  → 新交易需双重确认
  → 减仓至半仓

🚨 红牌熔断（一键清仓）:
  单日亏损 >5%
  或 CVaR占用 >75%
  或 交易所出现异常（插针/暂停/瞬时暴跌）
  或 核心策略3连亏损
  → 暂停所有新交易
  → 逐步平仓至安全水平

🔴 物理熔断（拔网线）:
  单日亏损 >8%
  或 CVaR占用 >90%
  或 多所同步异常（系统性风险）
  → 立即关闭所有API连接
  → 强制全部平仓
  → 锁定资金72小时
  → 通知丘总&天枢
```

### 5.3 每日执行评估清单

```
开仓前检查（全部通过方可交易）：
□ 信号置信度 > 最低阈值
□ 当前CVaR占用 < 75%
□ 单一标的敞口 < 总资金20%
□ 日内已亏损 < 总资金2%
□ 策略间相关系数 < 0.7
□ 流动性检查通过（滑点容忍范围内）
□ 交易所运行正常（无维护/暂停）
□ 网络延迟在容忍范围内
□ 融券资源充足（A股T0场景）
□ 杠杆倍数 < 最大限制（3x）
```

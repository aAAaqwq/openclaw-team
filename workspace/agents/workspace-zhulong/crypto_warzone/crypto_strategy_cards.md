# EC-01｜形态篇核心结构卡 V1 ✅

## 总纲

形态不是预测工具，而是概率判断的参照系统。
形态用于回答：当前结构偏向哪一边？完成概率多大？应该在什么位置布局？

## 头肩顶 / 头肩底

**核心定义**：在一段趋势后，价格形成左肩→头部→右肩的结构，跌破/突破颈线后确认反转

**必要条件**：
- 必须出现在明显趋势之后
- 头部必须高于左右肩
- 左右肩大致对称
- 颈线必须可画
- 突破颈线才算完成

**加分条件**：
- 右肩缩量（顶部）/ 放量（底部）
- 右肩出现反转K线
- MACD 顶背离/底背离共振
- 跌破颈线放量

**失效条件**：
- 右肩明显高于头部
- 跌破颈线后快速收回
- 形态未完成就反向突破

**执行规则**：
- 左侧：右肩滞涨/止跌时轻仓试单
- 右侧：颈线跌破/突破确认后入场
- 止损：颈线内侧 + 一定缓冲
- 目标：头部到颈线的 1:1 映射距离

## 双顶 / 双底

**核心定义**：两次冲击相近高低点，跌破/突破中间回调低点/反弹高点形成的颈线

**必要条件**：
- 出现在上涨/下跌之后
- 两个顶/底相对接近
- 中间有明确回调低点/反弹高点
- 必须能画出有效颈线

**执行规则**：
- 跌破颈线才算完成
- 第二顶缩量做多信号弱化
- 第二顶假突破是加分项

## 三重顶 / 三重底

**核心定义**：三次冲击相近高低点，每次冲击失败后形成震荡区

**执行规则**：
- 比双顶/双底更可靠（多一次验证）
- 通常意味着强压力/支撑区
- 突破后目标更远

## V型 反转

**核心定义**：价格快速下跌后急速反弹，或快速上涨后急速下跌

**特点**：
- 没有中间过渡结构
- 通常伴随爆量
- 反转速度快，难以左侧交易

**执行规则**：
- 不可左侧抄底/逃顶
- 等右侧确认入场
- 设置严格止损

## 圆弧底 / 圆弧顶

**核心定义**：价格缓慢形成弧形底部/顶部结构

**特点**：
- 持续时间长
- 反转可靠性高
- 成交量配合明显

**执行规则**：
- 圆弧末段入场
- 突破颈线确认
- 目标为圆弧深度

## 菱形

**核心定义**：价格先扩散后收敛形成的菱形结构

**特点**：
- 震荡加剧后的收敛
- 通常出现在顶部
- 成交量先放大后缩小

# EC-02｜指标篇执行映射表 V1 ✅

## 一、指标总定位

> 指标不负责给你开仓理由，指标负责给你过滤和加减分。

**执行顺序铁律**：
1. 先判趋势 — 上涨/下跌/震荡？初期/中期/末期？
2. 再判位置 — 支撑？阻力？箱体边界？中轨？
3. 再判结构 — 回调？反弹？中继？反转？假突破？
4. 最后看指标 — MACD/布林/背离做辅助验证

## 二、MACD 执行映射总表

| 场景 | 结构定位 | 使用方式 | 不能怎么做 |
|------|---------|---------|-----------|
| 零上金叉 | 上涨回调结束 | 顺势做多/回调结束二次上车 | 不能脱离趋势单独做 |
| 零下死叉 | 下跌反弹结束 | 顺势做空/反弹结束继续做空 | 不能在低位追空 |
| 顶背离 | 上涨动能衰减 | 减仓/保护/轻仓试空 | 不能直接做空（可能连续背离） |
| 底背离 | 下跌动能衰减 | 观察/轻仓试多/减空 | 不能直接抄底 |
| 顶背离+死叉 | 趋势可能反转 | 做空信号 | 需确认 |
| 底背离+金叉 | 趋势可能反转 | 做多信号 | 需确认 |

## 三、布林带 执行映射总表

| 场景 | 结构定位 | 使用方式 | 限制条件 |
|------|---------|---------|---------|
| 触及上轨+RSI超买 | 压力区 | 减仓/轻仓试空 | 趋势市失效 |
| 触及下轨+RSI超卖 | 支撑区 | 减空/轻仓试多 | 趋势市失效 |
| 带宽收缩（Squeeze） | 爆发行情前兆 | 等待突破方向 | 不能预判方向 |
| 带宽扩张 | 趋势正在加速 | 顺势持有 | 不逆势 |
| 中轨附近 | 没有明确方向 | 观望 | 不做 |

## 四、多指标复合过滤规则

**信号强度评分**（0-5分）：
- 趋势确认 +1
- 关键位共振 +1
- 结构清晰 +1
- 指标共振 +1
- K线确认 +1

**执行阈值**：
- 0-2分：不做
- 3分：轻仓（1/3仓）
- 4分：半仓（1/2仓）
- 5分：正常仓位

# EC-03｜布林带辅助使用清单 V1 ✅

## 一、布林带核心认知

布林带不是价格通道，而是波动率可视化工具。

**三要素**：
- 中轨：20周期SMA
- 上/下轨：中轨 ± 2σ
- 带宽：波动率的可视化表达

## 二、三大核心用法

### 1. 趋势识别

- 价格沿上轨运行 → 强势上涨
- 价格沿下轨运行 → 弱势下跌
- 价格在中轨附近来回 → 震荡

### 2. 波动率预测（Squeeze）

- 带宽压缩 → 即将爆发
- 带宽扩张 → 趋势加速中

**Squeeze 交易法**：
```
条件：带宽处于近N期最低区间
策略：等待突破方向，跟随突破方向入场
止损：Squeeze 区间的另一边
目标：带宽扩张1倍以上
```

### 3. 超买超卖

- 触及上轨 → 潜在压力（趋势市降权）
- 触及下轨 → 潜在支撑（趋势市降权）
- RSI 同步确认 → 信号强化

## 三、布林带使用注意事项

| 误区 | 正确理解 |
|------|---------|
| 价格触及上轨=卖出信号 | 在强势趋势中，价格会沿着上轨运行 |
| 价格触及下轨=买入信号 | 在弱势趋势中，价格会沿着下轨运行 |
| 布林带是支撑阻力线 | 它是统计工具，不是交易边界 |
| 压缩一定会爆发 | 压缩只是条件，方向需要其他工具确认 |

# EC-04｜心理篇核心认知偏差清单 V1 ✨ 本次补完

## 一、核心框架

```
情绪循环 → 认知偏差 → 行为错误 → 交易亏损
    ↑                                │
    └────────── 复盘修正 ─────────────┘
```

## 二、13大核心认知偏差

### 1. 确认偏差
*只寻找支持自己持仓的信息，忽略相反证据*
- **表现**：有利多消息加倍持有，不利消息找理由忽略
- **后果**：趋势反转时不认错，小亏变大亏
- **破解**：开仓前必须列出：**"什么条件出现我就错了"**

### 2. 近因效应
*最近几笔交易结果严重影响判断*
- **表现**：连亏后不敢开仓/连赚后开重仓
- **后果**：逆风时错过机会，顺风时一次亏光
- **破解**：系统化交易，每笔独立，不看盈亏结果看执行质量

### 3. 锚定效应
*过分依赖某个特定价格作为参照*
- **表现**："它以前是 X 价格，现在 Y 很便宜"
- **后果**：忽略市场结构变化，在下跌中抄底
- **破解**：始终以市场当前结构为准，历史价格仅供参考

### 4. 损失厌恶
*损失带来的痛苦 > 同等收益带来的快乐（约2-2.5倍）*
- **表现**：亏损死扛、盈利怕回吐早早卖出
- **后果**：截断利润，让亏损奔跑
- **破解**：机械止损，正期望系统+纪律执行

### 5. 厌恶不确定性
*宁愿持有亏损头寸也不愿平仓面对不确定*
- **表现**：平仓后怕踏空，持有亏损头寸等"反弹"
- **破解**：严格执行入场前设定好的止损

### 6. 处置效应
*过早卖出盈利仓位，过久持有亏损仓位*
- **表现**：赚一点就跑，亏了死扛
- **破解**：移动止盈+固定止损，系统化退出

### 7. 代表性偏差
*根据类似性快速判断，忽略统计概率*
- **表现**："上次这种形态涨了，这次也会涨"
- **破解**：每笔交易记录检查：是否真的符合系统条件

### 8. 过度自信
*连续成功后自我膨胀，放大仓位*
- **表现**：连赚3笔后开始加大仓位
- **破解**：固定仓位规则，盈利不放大

### 9. 随机性错觉（控制幻觉）
*以为自己能控制随机事件*
- **表现**：频繁交易、调整参数试图"优化"
- **破解**：区分自己能控制的（入场规则/仓位/止损）和不能控制的（市场方向）

### 10. 沉没成本谬误
*已经亏了那么多，现在不能平仓*
- **表现**：死扛亏损，希望"至少回到成本价"
- **破解**：沉没成本与决策无关，只问：**如果现在没有持仓，会买入吗？**

### 11. 羊群效应
*跟风交易，不敢独立判断*
- **表现**：看到大家都在买就跟买，FOMO
- **破解**：独立分析，系统化信号优先

### 12. 事后偏差
*事后觉得一切都很明显*
- **表现**：复盘时觉得自己早就看出来了
- **破解**：复盘记录必须区分"入场时知道什么"和"事后才知道什么"

### 13. 懒惰/省力倾向
*依赖简单线性因果，懒得深入分析*
- **表现的三种形式**：
  1. 靠死记硬背模式下单，不分析背景
  2. 不看更高时间框架，只看单周期找借口开仓
  3. 照搬教材解读，不验证当前市场状态
- **破解**：严格执行四步流程（趋势→位置→结构→信号）

## 三、执行总结

| 情绪状态 | 典型行为 | 建议措施 |
|---------|---------|---------|
| 兴奋/贪婪 | 追高、加仓、忽略风险 | 暂停交易，核查系统条件 |
| 恐惧/焦虑 | 错过入场、过早平仓 | 严格执行规则 |
| 愤怒/沮丧 | 报复交易、过度交易 | 立即停止，休息1小时 |
| 无聊/麻痹 | 无信号乱做 | 检查是否真的需要交易 |
| 过度自信 | 重仓、频繁交易 | 回归仓位规则 |

# EC-05｜心理篇执行映射表 V1 ✨ 本次补完

## 一、总纲：交易心理执行结构

```
认知偏差 → 预警触发 → 执行检查 → 行为纠正 → 规则回归
                                                  │
                            └─── 复盘记录 ─────────┘
```

## 二、偏差 → 行为映射表

| 偏差 | 预警信号 | 行为检查清单 | 纠正动作 |
|------|---------|------------|---------|
| 确认偏差 | 只看多/只看空 | 找3条相反证据再决定 | 开仓前必须写"我错的条件" |
| 近因效应 | 情绪被前几笔交易影响 | 这笔交易独立于之前的盈亏 | 检查是否按系统执行 |
| 锚定效应 | 心里总想某个价格 | 当前结构支持这个价格吗？ | 只看当前价格结构 |
| 损失厌恶 | 亏损单死扛 | 止损被触碰了吗？ | 立即执行止损 |
| 处置效应 | 小赚就跑 | 止盈条件被触碰了吗？ | 让盈利奔跑 |
| 过度自信 | 连赚后想加仓 | 仓位规则被遵守了吗？ | 固定仓位比例不调整 |
| 沉没成本 | "等回本就平" | 如果空仓，现在会买入吗？ | 不会就立即平仓 |

## 三、盘前心理检查清单

开仓前必须回答：
```
□ 这笔交易是基于系统规则还是情绪冲动？
□ 我的止损放在哪里？能接受这个亏损吗？
□ 如果这笔亏了，会影响接下来的交易吗？
□ 大周期趋势支持这个方向吗？
□ 盈亏比是否 >= 1:1.5？
```

# EC-06｜庄家思维核心认知卡 V1 ✅

## 一、庄家思维铁律

1. 主力 ≠ 庄家，机构 ≠ 庄家
2. 庄家的核心不是短时拉盘，而是持续控盘
3. 庄家最常见的动作：**做与真实目的相反的表面动作**
4. 建仓时压价，出货时护价 — 庄家行为的基本悖论
5. 学习庄家思维是为了看痕迹，不是为了搞阴谋论
6. 盘口是辅助线索，不是单独开仓信号
7. 插针很多时候是流动性问题，不一定都是恶意操纵
8. 量价必须结合趋势、位置、横盘时间一起解释
9. 底部无量上涨可能是筹码干净，下跌途中无量上涨更像诱多
10. 价格行为是主系统，庄家思维是解释增强层

## 二、庄家三个阶段

### 1. 建仓阶段
- **表面动作**：制造恐惧/制造下跌
- **真实意图**：低位吸筹
- **盘口特征**：压大单、频繁小额吃单、无大笔追买
- **K线特征**：横盘/阴跌、插针后快速收回、底部放量

### 2. 拉升阶段
- **表面动作**：放量突破、制造FOMO
- **真实意图**：吸引跟风盘
- **盘口特征**：大单对倒、频繁撤单、挂假大买单
- **K线特征**：大阳线、跳空、连续推高

### 3. 出货阶段
- **表面动作**：护价/制造还要涨的假象
- **真实意图**：高位派发
- **盘口特征**：买单支撑、小单出货、大压单维持
- **K线特征**：高位震荡、长上影、量价背离

# EC-07｜庄家行为执行映射表 V1 ✅

## 一、庄家行为 → 交易决策映射

| 庄家行为 | 可能意图 | 交易对策 | 优先级 |
|---------|---------|---------|-------|
| 压大单接小单 | 建仓吸筹 | 跟随低吸 | ⭐⭐⭐ |
| 对倒放量突破 | 制造FOMO诱多 | 不追，等回调验证 | ⭐⭐ |
| 高位护价横盘 | 出货中 | 减仓，不开新多 | ⭐⭐⭐ |
| 砸盘后快速收回 | 洗盘/建仓 | 观察，等确认做多 | ⭐⭐ |
| 尾盘拉高但量小 | 诱多/骗线 | 不追，警惕次日低开 | ⭐⭐ |
| 开盘拉升后持续出货 | 派发 | 逢高减仓 | ⭐⭐⭐ |

## 二、量价异常识别

| 量价表现 | 正常 | 异常解释 |
|---------|------|---------|
| 底部放量上涨 | 跟庄买入 | 可能是对倒 |
| 底部无量上涨 | 筹码干净 | 庄家控盘度高 |
| 顶部放量滞涨 | 抛压大 | 庄家出货 |
| 顶部缩量上涨 | 跟风不足 | 上涨动能衰竭 |
| 下跌放量 | 恐慌抛售 | 可能是见底信号 |
| 下跌缩量 | 无人接盘 | 可能继续跌 |

# EC-08｜价格行为执行卡 V1 ✅

## 一、价格行为总公式

```
背景 × 关键位 × 结构 × 信号K线 × 入场K线 × 盈亏比 × 止损 = 是否开仓
```

## 二、执行前总判断卡

### Step 1：背景判断
只允许以下归类：
- 上涨趋势 / 下跌趋势 / 交易区间
- 简单回调 / 复杂回调 / 趋势末端
- 混乱结构

**铁律**：背景不清，不交易。

### Step 2：关键位判断
必须明确：
- 前高/前低、区间上下沿、通道上下轨
- 颈线、假突破点、EMA关键位
- 密集成交区、爆量K线高低点

**铁律**：不在关键位附近的信号，一律降级。

### Step 3：结构判断
只允许归类为：
- 趋势推进 / 回调 / 交易区间
- 通道 / 假突破 / 反转雏形 / 复杂回调

**铁律**：不是对着K线交易，是对着结构交易。

### Step 4：信号K线质量
至少看4点：
- 实体是否清晰
- 收盘是否有力
- 是否发生在关键位
- 是否与背景一致

### Step 5：入场K线确认
**铁律**：看到机会不等于可以入场。等入场K线确认后再行动。

## 三、执行分类

### 信号K线交易法
- **高一**：信号K线收盘价入场
- **高二**：信号K线最高/最低点+1 tick入场
- **信号K线止损**：信号K线的最低/最高点-1 tick

### 简单回调 vs 复杂回调
| 类型 | 特征 | 交易方式 |
|------|------|---------|
| 简单回调 | 1-3根K线完成 | 回调低点入场 |
| 复杂回调 | 多重结构、ABC | 等结构完成+确认 |

## 四、退出规则

- **止损退出**：价格触及止损位
- **止盈退出**：到达目标位
- **反向信号退出**：出现相反方向的信号K线
- **时间退出**：持仓超过计划时间

# EC-09｜交易区间作战卡 V1 ✨ 本次重制

## 一、定义与识别

交易区间是"趋势做不了的时候，怎么在这里活下来"的核心能力。

**识别条件**：
- 价格在可识别的上下边界之间来回运动
- 至少触碰边界3次以上
- 没有明确的趋势方向

**区间的三种形态**：
| 类型 | 特征 | 交易策略 |
|------|------|---------|
| 宽幅区间 | 振幅>ATR的3倍 | 高空低多，边界附近操作 |
| 窄幅区间 | 振幅<ATR的1.5倍 | 等待突破，不参与 |
| 倾斜区间 | 一边高一边低 | 沿倾斜方向顺做，反弹方向逆做 |

## 二、区间交易核心规则

### 1. 边界执行规则
```
上轨附近：
  - 出现看跌信号 → 做空
  - 没有信号 → 观望
  - 突破上轨 → 等回踩验证

下轨附近：
  - 出现看涨信号 → 做多
  - 没有信号 → 观望
  - 跌破下轨 → 等反抽验证
```

### 2. 中轨处理规则
```
价格在中轨：
  - 偏震荡 → 不操作
  - 性价比低 → 放弃
  - 除非有极好信号 → 轻仓
```

### 3. 效率原则
- 区间边缘交易效率最高
- 中间位置交易效率最低
- 宁可错过，不要在中间位置做

### 4. 区间止盈规则
- 区间内交易止盈目标 = 到对面边界的70-80%
- 不要幻想突破，先拿区间内利润
- 接近对面边界时主动减仓

## 三、区间突破处理

| 突破类型 | 真假判断 | 处理方法 |
|---------|---------|---------|
| 区间突破 | 收盘价在边界外+A收盘 | 真突破跟进 |
| 假突破 | 突破后迅速回到区间内 | 反向操作（SB） |
| 捉弄式突破 | 突破后震荡然后反向 | 等确认再做 |

# EC-11｜谐波反转区执行卡 V1 ✨ 本次重制

## 一、谐波核心认知

谐波交易不是"画图找神奇点位"，而是**利用价格的自然回撤比例来识别潜在反转区**。

### 5步流程
1. 识别明确的价格波动（XA段）
2. 等待B点回调到XA的特定比例
3. C点确认回调结构
4. D点到达预测区域（PRZ）
5. 在PRZ区域+信号确认入场

## 二、核心谐波形态速查

| 形态 | XA→B比例 | B→C比例 | C→D比例 | PRZ区域 |
|------|---------|---------|---------|---------|
| Gartley 222 | 0.618 | 0.382-0.886 | 1.13-1.618 | X的0.786回撤 |
| 蝙蝠 (Bat) | 0.382-0.500 | 0.382-0.886 | 1.618-2.618 | X的0.886回撤 |
| 螃蟹 (Crab) | 0.382-0.618 | 0.382-0.886 | 2.618-3.618 | X的1.272回撤 |
| 蝴蝶 (Butterfly) | 0.786 | 0.382-0.886 | 1.618-2.240 | X的1.272回撤 |
| 鲨鱼 (Shark) | 0.382-0.500 | 0.382-0.886 | 1.13-1.618 | — |
| CYPher | 0.382-0.618 | 0.382-0.886 | 1.272-2.000 | X的0.786 |

## 三、PRZ（潜在反转区）入场规则

### 入场条件（必须同时满足）
```
1. D点到达计算的PRZ区域内
2. 在PRZ出现反转K线信号（吞没/蜻蜓/锤子等）
3. 大周期趋势方向支持（逆势做需要更高确认）
4. 盈亏比 >= 1:2
```

### 入场方式
```
激进入场：PRZ边界下方/上方 +1 tick
标准入场：等反转K线确认信号
保守入场：等反转K线收盘确认
```

### 止损规则
```
激进止损：PRZ区域外 + 1 ATR
标准止损：最近的浪低/浪高 + 一定缓冲
保守止损：前一个结构低/高点
```

### 目标设定
```
TP1：BC段的 1.618 延伸
TP2：XA段的 0.618 回撤
TP3：XA段的起点
TP：分批止盈，移动止损
```

## 四、谐波交易注意事项
- 不是每一个ABCD都是有效谐波，必须检查比例关系
- 没有信号K线确认的PRZ不可入场
- 逆大势方向的谐波需要更高确认等级
- 震荡市中谐波胜率更高，趋势市中谐波更容易失败
- D点不精确是常态，容忍一定误差范围

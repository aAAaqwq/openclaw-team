# A股策略总索引表 (Stock Master Index)

> **版本**: V1.0 实战版  
> **覆盖范围**: 聚宽600条策略 (2020-2025) + 小象交易学院课程  
> **构建目标**: 从600条策略压缩为可实战的母原型武器库

---

## 一、总纲定位

本索引表不是"600篇文章目录"，而是**量化策略母库**的实战地图。

**核心转化**:
```
600条策略 → 30个稳定母原型 → 10个核心哲学原理
```

**四层母库结构**:
| 层级 | 定义 | 示例 |
|------|------|------|
| **策略层** | 真正可形成交易系统的原型 | ETF轮动原型、双低转债原型 |
| **研究层** | 因子验证、市场状态、统计研究 | VIX研究、创新高比例指标 |
| **工具层** | 指标、公式、模板、执行框架 | 通达信迁移模板、多因子框架 |
| **变体层** | 同一母策略的参数/风控/择时变化 | ETF+RSRS变体、ETF+MA变体 |

## 二、十大主河道分类

### 河道1: 价值/质量/风格
**核心逻辑**: 低估值 + 质量因子 + 成长价值 + 行业龙头

| 母原型 | 关键因子 | 调仓频率 | 风险特征 |
|--------|----------|----------|----------|
| 漂亮50风格原型 | ROE>15%, PE<30 | 月频 | 大盘回撤敏感 |
| 低PB低换手小市值 | PB<1.5, 换手<5% | 月频 | 流动性风险 |
| 双模态价值切换 | 价值/成长轮动信号 | 季频 | 风格切换误判 |
| 行业龙头价值配置 | 行业分层+龙头筛选 | 月频 | 行业集中风险 |
| 大师成长质量原型 | GP(毛利/资产)+ROE | 月频 | 质量因子失效 |

**去重要点**: 很多"漂亮50"、"中证500成长"、"高ROE低PEG"本质都是行业分层+PEG+增长+ROE

## 三、六大接收桶

所有策略统一归到以下桶中，不再散收:

### 1. 稳定原型桶
**定义**: 有独立交易逻辑、不是简单参数改版、可作为母策略命名

**示例**:
- GrossProfit质量因子原型
- 可转债双低原型
- ETF轮动原型

### 2. 重要变体桶
**定义**: 依附某个母原型、改的是择时/止损/池子/参数/执行

**示例**:
- ETF轮动+RSRS均值回落变体
- ETF轮动+MA乖离择时变体
- 小市值+尾盘执行变体

### 3. 研究验证桶
**定义**: 不直接下单、核心价值是验证某个现象或因子

**示例**:
- ETF动量因子评估原型
- 中国版VIX研究原型
- 涨停因子研究原型

### 4. 工具模板桶
**定义**: 提供公式、接口、迁移、框架

**示例**:
- 通达信公式迁移原型
- 标准多因子教学原型
- 事件表驱动择时原型

### 5. 执行工程桶
**定义**: 强调实盘适配、下单流程、风控流程、券商适配

**示例**:
- 实盘修订适配原型
- 尾盘小市值执行原型
- 华泰API接入模板

### 6. 重复归并桶
**定义**: 完全重复、近重复、标题不同但同核

**分类**:
- 完全重复: 同一文章不同标题转载
- 近重复: 参数不同本质相同 (MA20→MA30)
- 同源变体: 同一母策略分裂版本
- 壳不同核相同: 不同名称同一逻辑

## 四、去重归并规则

### 重复识别矩阵

| 类型 | 判断标准 | 处理方式 |
|------|----------|----------|
| **完全重复** | 代码/逻辑完全一致 | 归并到最早版本，标记重复 |
| **近重复** | 参数±20%以内，核心逻辑不变 | 归并为母原型的参数变体 |
| **同源变体** | 同一母策略+不同过滤器 | 归入变体桶，链接母原型 |
| **壳不同核相同** | 标题不同但因子组合相同 | 去标题幻觉，归并到母原型 |
| **研究-执行重复** | 研究稿与最终脚本指向同一策略 | 保留执行稿，标记研究来源 |

### 母原型命名规范

```
[资产类别]_[核心因子/逻辑]_[执行特征]_原型

示例:
- equity_grossprofit_quality_monthly_原型
- etf_momentum_rotation_weekly_原型
- cb_dual_low_monthly_原型
- microcap_close_execution_daily_原型
```

## 五、实战筛选标准

### 进入实盘候选池的5道门

| 门槛 | 标准 | 检查项 |
|------|------|--------|
| **逻辑门** | 有清晰经济学/行为学解释 | 为什么这个因子能赚钱？ |
| **统计门** | 样本量>200次交易，滚动表现稳定 | 夏普>1.0，最大回撤<30% |
| **工程门** | 无未来函数，数据对齐正确 | 时间戳检查、幸存者偏差处理 |
| **风险门** | 尾部风险可控，杠杆可承受 | 黑天鹅场景测试通过 |
| **组合门** | 纳入组合后提高整体夏普 | 与现有策略相关性<0.6 |

### 优先启动的5条主线 (建议)

| 优先级 | 母原型 | 理由 |
|--------|--------|------|
| P0 | ETF轮动+RSRS | 结构最清楚，可解释性强 |
| P0 | 小市值基本面+指数风险开关 | A股特色，历史有效 |
| P1 | 红利价值/财务质量 | 防御性强，适合震荡市 |
| P1 | 可转债双低 | 债底保护，回撤可控 |
| P2 | 趋势跟踪(CTA) | 与股票低相关，分散价值 |

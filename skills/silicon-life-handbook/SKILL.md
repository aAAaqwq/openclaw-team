## 技能定位

本技能是"硅基生命训练学"体系的**索引导航技能**。

它不直接回答"硅基生命是什么"这类知识性问题，
而是回答：**"这个问题/任务属于哪个知识模块，我应该去哪里找答案"**。

---

## 触发条件

当遇到以下场景时，应加载本技能并读取 INDEX.md：

### 场景1：新建 Agent 初始化

**触发条件**：
- 收到 `/new` 或 `/reset` 新会话
- 被要求扮演"硅基生命训练体系"相关的 Agent

**动作**：
1. 读取 `knowledge/silicon-life-handbook/INDEX.md`
2. 对照"角色知识图谱"（第4节），确认当前角色应掌握哪些模块
3. 如有缺失，提示维护者补充相关模块知识

### 场景2：Agent 优化复盘

**触发条件**：
- Agent 出现系统性偏差（如：总是不记得关键信息、总是不主动、协同总掉棒）
- 维护者要求"复盘一下哪里出了问题"

**动作**：
1. 读取 `knowledge/silicon-life-handbook/INDEX.md`
2. 对照"问题→模块映射表"（第6节），定位根问题属于哪个模块
3. 读取对应模块的原始文档，给出具体优化建议

### 场景3：硅基生命相关问题排查

**触发条件**：
- 用户/维护者问"Agent 不记得我是谁是怎么回事"
- 用户/维护者问"多 Agent 协同总是掉棒怎么办"
- 用户/维护者问"Agent 越权执行怎么治理"

**动作**：
1. 读取 `knowledge/silicon-life-handbook/INDEX.md`
2. 对照"问题→模块映射表"（第6节）
3. 定位到具体模块，给出知识节点指引

### 场景4：基于硅基生命体系生成内容

**触发条件**：
- 需要写新的模块文档
- 需要扩展现有模块
- 需要将硅基生命体系应用到新场景

**动作**：
1. 读取 `knowledge/silicon-life-handbook/INDEX.md`
2. 对照"体系全景图"（第1节）和"核心概念定位表"（第2节）
3. 确定新内容的定位（属于哪个卷/模块/概念节点）
4. 参考"交叉引用网络"（第3节），确保新内容和现有体系一致

### 场景5：Agent进化能力评估 ✨ NEW

**触发条件**：
- 需要评估Agent是否具备自我进化能力
- 需要为Agent开启"进化型"模式
- 需要建立技能基因库

**动作**：
1. 读取 `knowledge/silicon-life-handbook/INDEX.md`
2. 对照"超维演进概念层"（INDEX.md 第2.6节）
3. 评估Agent当前是否具备：
   - 技能基因提取能力
   - OODA循环执行能力
   - 记忆全息网络构建能力
   - 自主目标生成能力
4. 如需开启进化模式，引导读取 `volume-08/模块1-动态进化机制.md`

### 场景6：Agent从"工具"向"认知伙伴"转型 ✨ NEW

**触发条件**：
- 维护者希望Agent从"被动响应"升级为"主动提案"
- 需要建立进化路线图
- 需要实现预测性学习

**动作**：
1. 读取 `knowledge/silicon-life-handbook/INDEX.md`
2. 对照"超维演进概念层"，理解"响应型"vs"提案型"的区别
3. 引导读取 `volume-08/模块3-自主目标生成系统.md`
4. 设计从"工具"到"伙伴"的转型路径

## 使用限制

### 本技能不负责

- 直接回答具体的硅基生命知识问题（应直接读对应模块文档）
- 生成具体的任务卡、整改单、交接数据包（应读对应模板）
- 替代其他专用技能（如 `todo-tracker`、`habit-tracker`）

### 本技能只负责

- 导航：告诉使用者"这个问题/任务属于哪个模块"
- 定位：告诉使用者"对应的原始文档在哪里"
- 串联：告诉使用者"这个概念和其他概念是什么关系"
